原子核基本性质、放射性、核反应与核模型

原子核与核物理的基本图景

为什么研究原子核更复杂

在原子物理中，常把原子核近似为点电荷来研究电子能级与光谱；但一旦关心核的内部结构、核能级以及核的转变过程，就进入核物理的范围。复杂性主要来自两个方面：

多体量子体系

原子核由许多核子（质子与中子）构成，是典型多体量子系统。核子间并不存在一个简单的“中心势”能完全主导，很多现象需要用集体运动与独立粒子运动共同描述。

多种相互作用并存

核现象不仅受电磁作用影响，还强烈涉及：

  • 强相互作用：提供核子束缚、决定核结构与稳定性；
  • 弱相互作用：主导 β 衰变等过程；
  • 电磁作用：导致库仑斥力、γ 跃迁、超精细结构等。

原子核的组成与基本性质

质子与中子：原子核的基本组成

中子发现的关键思路

早期人们已通过放射性认识到 α、β、γ 射线，并通过 Rutherford 散射实验确立“原子有核”。质子逐步被确认是核内带正电的成分，但“把电子塞进核里”会在自旋、统计性质与能量尺度等方面引发严重矛盾。

1932 年 Chadwick 研究 α 粒子轰击铍产生的中性辐射，结合能量与动量守恒分析，证明该辐射是质量接近质子的中性粒子——中子。典型反应示例：

  • α+9Be→n+12C\alpha + {}^{9}\mathrm{Be} \rightarrow n + {}^{12}\mathrm{C}

由此确立现代图景：原子核由质子与中子组成。

核的尺寸、密度与“近似不可压缩”

核半径经验关系

大量实验（电子散射、中子散射等）表明核的特征半径与质量数 AA 满足经验公式：

  • R=r0A1/3R = r_0 A^{1/3}

因为体积 V∝R3∝AV \propto R^3 \propto A，核内平均密度近似为常数。这与核力的短程性与饱和性密切相关：每个核子主要与有限数目的近邻强烈相互作用，从宏观看，核物质像“不可压缩液滴”。

细化理解：核表面与形变（补充）

更精细的密度分布会显示“表面弥散层”（边界并非陡峭截止）。某些核还会发生形变（椭球形甚至更复杂），这与壳层结构与集体激发有关。

质量亏损、结合能与比结合能

结合能与质量亏损

把 ZZ 个质子与 A−ZA-Z 个中子从无穷远聚合成一个原子核，会释放能量，称为结合能 BB。用质量差表示：

  • B=ΔMc2=[Zmp+(A−Z)mn−MN]c2B = \Delta M c^2 = \left[Z m_p + (A-Z)m_n - M_N\right]c^2

其中 MNM_N 为核质量。

比结合能与裂变/聚变的能量学方向

定义比结合能：

  • ε=B/A\varepsilon = B/A

经验上，ε\varepsilon 随 AA 增大先上升，在铁族附近达到较高值，随后缓慢下降。由此得到核能的基本判断：

  • 轻核聚变：向更高的比结合能“爬坡”，放能；
  • 重核裂变：裂变产物更接近铁族区域，放能。

原子质量与电子质量的注意点（补充）

实际计算中常用“原子质量”表。许多反应/衰变中电子数在两边相同（或能被等效处理），原子质量的电子质量会相互抵消，从而仍可得到正确的 QQ 值。但对 β+\beta^+ 衰变与电子俘获，电子数变化会影响能量条件，需要格外小心。

核自旋、核磁矩与超精细结构

核自旋 II

核的总角动量称为核自旋 II。实验上，不同核素的 II 反映了核子在能级中的填充方式，尤其与壳层结构密切相关。

核磁矩与核子内部结构

质子与中子的磁矩都显著偏离“点粒子自旋磁矩”的简单预言，表现为反常磁矩，提示核子并非无结构点粒子，而有内部自由度（现代观点为夸克-胶子结构）。

超精细结构：电子与核磁矩的耦合

原子谱线的超精细分裂来自电子总角动量 JJ 的磁矩与核磁矩的相互作用。角动量耦合后，总角动量为：

  • F=J+I\mathbf{F}=\mathbf{J}+\mathbf{I}

不同 FF 对应不同的微小能级位移，从而产生超精细分裂。

放射性与衰变

放射性的基本概念

放射性是指不稳定原子核自发转变为其他核种或较低能级状态，并发射粒子或辐射。常见过程包括：

  • α\alpha 衰变、β\beta 衰变、γ\gamma 跃迁（以及内转换等竞争过程）

衰变定律、半衰期、平均寿命与活度

指数衰变定律

设核素数目为 N(t)N(t)，衰变常数为 λ\lambda，则：

  • N(t)=N0e−λtN(t)=N_0 e^{-\lambda t}

半衰期与平均寿命分别为：

  • T1/2=ln2/λT_{1/2}=\ln2/\lambda
  • τ=1/λ\tau=1/\lambda

活度

活度表示单位时间内衰变数：

  • A=−dNdt=λNA=-\frac{dN}{dt}=\lambda N

工程和实验中常用 Bq 等单位刻画放射源强度。

衰变链与平衡（补充）

许多核素并非只衰变一次，而是形成衰变链。链中某一子体的活度会随时间增长到与母体活度相当（在一定条件下形成暂时/长期平衡），这在天然放射系与核测量中很常见。

衰变能（QQ 值）与动能分配

对衰变：

  • A→B+C(+⋯)A \rightarrow B + C (+\cdots)

衰变能由质量差给出：

  • Q=(mA−mB−mC−⋯)c2Q=(m_A-m_B-m_C-\cdots)c^2

当初态近似静止且为两体衰变时，动量守恒会使轻粒子通常获得更高速度与显著动能；多体衰变则会出现连续能谱与多种分配。

α\alpha 衰变：量子隧穿与寿命-能量关联

基本形式

α\alpha 衰变常见于重核：

  • ZAX→Z−2A−4Y+α{}^A_Z X \rightarrow {}^{A-4}_{Z-2}Y+\alpha

势垒贯穿的量子图像

核内强相互作用束缚与核外库仑斥力共同形成势垒。α\alpha 粒子通过量子隧穿穿出势垒。该机制解释了：

  • α\alpha 能量略有差别会导致半衰期跨越巨大数量级的变化（能量越高，隧穿概率越大，寿命越短）。

β\beta 衰变：连续谱、中微子与弱相互作用

早期矛盾与关键事实

β\beta 衰变的电子（或正电子）能谱连续，引发三类核心困难：

  • 核能级离散，为何出射电子能量连续？
  • 若电子能量小于最大值，能量似乎不守恒
  • 核内并没有“预先存在的电子”可直接发射

中微子的引入与弱相互作用观点

Pauli 提出中微子（或反中微子）带走部分能量与动量以恢复守恒定律。随后实验验证中微子存在。Fermi 将 β\beta 衰变描述为弱相互作用过程：核内核子发生类型转变并伴随轻子发射。

三种常见类型

  • β−\beta^-：中子 →\rightarrow 质子，同时放出电子与反中微子
  • β+\beta^+：质子 →\rightarrow 中子，同时放出正电子与中微子
  • 电子俘获（EC）：核俘获轨道电子使质子 →\rightarrow 中子，放出中微子

β+\beta^+ 与 EC 的能量条件（补充）

β+\beta^+ 衰变需要额外能量来“制造”正电子，并且还要满足动量守恒对应的运动学要求；当能量条件不满足时，同一核素可能通过电子俘获发生转变。

壳层空穴与 Auger 电子

EC 发生后原子壳层出现空穴，外层电子跃迁填补会发射特征 X 射线；也可能把跃迁能量传给另一电子使其逸出，形成 Auger 电子。

γ\gamma 跃迁、内转换与 Mössbauer 效应

γ\gamma 跃迁与核能级

γ\gamma 辐射通常对应同一核素从激发态向低能级跃迁：

  • $X^* \rightarrow X + \gamma$

内转换（IC）

核的跃迁能也可能不以光子形式发射，而是直接传给原子电子使其逸出，形成内转换电子。对很多核跃迁来说，γ\gamma 与 IC 之间存在竞争，其比例与跃迁能量、多极性与核结构有关。

同质异能态与“受阻跃迁”（补充）

某些激发态寿命很长，可形成同质异能态。一个重要原因是跃迁受到选择定则或多极性限制，导致跃迁几率显著降低（“受阻跃迁”）。

Mössbauer 效应

若发射/吸收核处在晶格中，反冲动量可由整个晶体共同承担，使反冲能量近乎为零，从而实现极窄线宽的共振吸收与极高分辨率测量。

放射性的典型应用

示踪与诊断

利用放射性核素易被探测的特性追踪物质迁移与反应路径，在化学、生物与医学中都有重要应用。

辐射探测与计量

放射性可通过电离、闪烁与半导体效应等方式被探测。能谱、计数率与衰变链信息常用于核素识别与定量分析。

放射性年代学：以 14C^{14}\mathrm{C} 为例

宇宙线在大气中产生 14C^{14}\mathrm{C}，生物体在生前与环境保持近似平衡；死亡后 14C^{14}\mathrm{C} 仅按指数规律衰减。测得剩余比例即可估算年代，核心仍是衰变定律在自然体系中的应用。

核反应：能量学、运动学与几率描述

核反应的定义与记号

核反应指外来粒子作用使靶核转变为其他核或其他状态。一般写作：

  • A+a→b+B(+⋯)A + a \rightarrow b + B (+\cdots)

常用紧凑记号：

  • A(a,b)BA(a,b)B

其中 AA 为靶核，aa 为入射粒子，bb 为出射粒子，BB 为反冲核或产物核。

反应能 QQ：放能与吸能

对 A(a,b)BA(a,b)B：

  • Q=(mA+ma−mb−mB)c2Q=(m_A+m_a-m_b-m_B)c^2

物理含义：

  • Q>0Q>0：放能反应，产物总动能增加；
  • Q<0Q<0：吸能反应，需要入射动能提供能量。

阈能的概念（补充）

当 Q<0Q<0 时，在实验室系往往存在阈能：即使入射动能刚好等于 ∣Q∣|Q| 也未必能发生反应，因为还必须满足动量守恒，体系需要保留一定的整体动能。

反应运动学与 Q 方程：用测量量反推核能级

在核质量或能级信息不够精确时，可测量出射粒子的能量与角度，并结合能量与动量守恒建立关系式（常称 Q 方程），从而反推出 QQ 值或产物核的激发能。

一个常见现象是：在固定角度测得出射粒子能量出现双分支，往往意味着产物核落在不同能级（基态/激发态）。两条分支的能量差直接对应不同激发能的差值，这类方法是核谱学的重要手段。

反应截面：把“发生几率”量化

薄靶近似下的基本关系

设靶中可反应核数为 NN，入射粒子流量为 II（单位面积单位时间粒子数），则在时间 dtdt 内发生反应的核数满足：

  • dN=−σINdtdN=-\sigma I N\,dt

σ\sigma 具有面积量纲，直观可理解为靶核对该反应的“有效面积”。

单位

  • 1 barn=10−24 cm21\ \mathrm{barn}=10^{-24}\ \mathrm{cm}^2

能量依赖与共振（补充）

截面通常随入射能量显著变化。若反应经过复合核中间态，截面可能出现尖锐共振峰，对应复合核的离散能级结构；而直接反应机制则常呈现不同的角分布与能量依赖特征。

裂变与聚变：核能的两条路径

核裂变：链式反应与慢化思想

裂变产物与能量规模

以 235U^{235}\mathrm{U} 为代表的重核可在吸收中子后裂变，通常产生：

  • 两个中等质量裂变产物
  • 平均约若干个中子（典型值约 2.5）
  • 放能约 200 MeV200\ \mathrm{MeV} 量级（常见给出约 210 MeV210\ \mathrm{MeV} 的代表数）

链式反应与中子慢化

裂变放出的中子若能继续诱发更多裂变，就形成链式反应。很多可裂变核对热中子截面更大，因此常通过慢化剂降低中子能量，提高裂变概率，实现可控功率输出。

可控与不可控释放的概念差异

反应堆依靠控制棒、冷却与反馈机制把链式反应稳定在可控状态；而核武器追求极短时间的超临界与快速能量释放。这里的差异关键不在“是否能裂变”，而在“是否能稳定控制反应速率与中子经济”。

钍基路线：增殖得到可裂变核素

钍资源丰度高，可通过吸收中子并经历 β−\beta^- 衰变链增殖得到 233U^{233}\mathrm{U}：

  • 232Th+n→233Th+γ{}^{232}\mathrm{Th}+n \rightarrow {}^{233}\mathrm{Th}+\gamma
  • 233Th→233Pa+β−{}^{233}\mathrm{Th}\rightarrow {}^{233}\mathrm{Pa}+\beta^-
  • 233Pa→233U+β−{}^{233}\mathrm{Pa}\rightarrow {}^{233}\mathrm{U}+\beta^-

这种燃料循环思想涉及中子经济性、材料与安全等多方面工程约束，但在核能体系设计中占有重要位置。

核聚变：克服库仑势垒与约束问题

为什么聚变难

轻核带正电，靠近必须克服库仑斥力。提高温度能提升粒子平均动能，使它们更可能接近到强相互作用起效的距离。但工程上更难的常常是：

  • 如何把高温等离子体稳定约束足够久

磁约束与 Tokamak

磁场对带电粒子产生洛伦兹力，从而实现对等离子体的磁约束。Tokamak 是典型方案之一，核心在于同时满足温度、密度与约束时间的要求。

D–T 反应与 ITER

较“容易点火”的聚变反应之一是氘-氚反应：

  • D+T→4He+n+17.6 MeVD+T\rightarrow {}^{4}\mathrm{He}+n+17.6\ \mathrm{MeV}

ITER 旨在验证高功率聚变等离子体在工程上的可行性。

氚增殖：用锂“养氚”

氚在自然界极稀少，因此常讨论在包层利用锂与中子反应增殖氚：

  • 6Li+n→4He+T{}^{6}\mathrm{Li}+n\rightarrow {}^{4}\mathrm{He}+T
  • 7Li+n→4He+T+n{}^{7}\mathrm{Li}+n\rightarrow {}^{4}\mathrm{He}+T+n（条件更复杂）

为什么不直接主打 D–D（补充）

D–D 不需要氚，但反应截面更小、点火条件更苛刻；此外反应链会受到核结构与通道限制（某些质量数组合对应的不稳定核使路径不“顺滑”），因此工程上常优先考虑 D–T。

核力与核模型：从相互作用到结构规律

核力的主要性质

短程性

核力作用程在几个 fm 量级，远短于电磁相互作用的有效范围，因此核相互作用主要发生在近邻核子之间。

饱和性

由于短程性，每个核子只能与有限数目的近邻强烈相互作用，使总结合能随 AA 近似线性增长，这是核物质呈“近似常密度液滴”的关键原因之一。

排斥芯

在极短距离上核力呈强烈排斥，防止核子无限靠近导致系统塌缩。

电荷无关性与同位旋

除去库仑效应后，p–p、n–n、p–n 的核力相近，可用同位旋统一描述核子两种状态：

  • 同位旋 I=1/2I=1/2，第三分量 I3=±1/2I_3=\pm 1/2 对应质子/中子。

核力的来源：介子交换与 QCD 剩余力

汤川介子交换思想

汤川提出核力可视为核子交换介子产生，作用程与介子质量相关。用作用程估算得到介子质量能标在百 MeV 附近，符合核力短程性的经验。

现代观点：强相互作用的有效表现

更根本层面上，核子间核力可视为 QCD 色力在强束缚体系上的有效“剩余力”，在层级结构上与范德华力的出现方式具有可类比之处。

费米气体模型：核子的量子简并体系

基本思想

核子是费米子，满足泡利不相容原理。在平均场近似下，核子填充从低到高的动量态形成费米海。

对称能的直观来源

若 NN 与 ZZ 相差很大，数量更多的一类核子必须占据更高动量态，导致能量上升，因此稳定核总体倾向 N≈ZN\approx Z。对重核还要叠加库仑斥力的影响，常见稳定核呈现 N>ZN>Z。

液滴模型与半经验质量公式

液滴图像

核密度近似常数、核力短程且饱和，使核在宏观上像带电液滴。液滴模型用若干项分别描述不同效应，构成半经验质量公式，用于拟合与解释结合能的总体趋势。

各项的物理含义（概念性）

  • 体积项：内部核子与近邻结合的主要贡献
  • 表面项：表面核子近邻更少，结合减弱
  • 库仑项：质子间斥力降低结合能，随 ZZ 增大更显著
  • 对称项：NN 与 ZZ 不平衡的能量代价
  • 配对项：偶偶核更稳定、奇奇核相对不稳定的经验规律

液滴模型擅长解释宏观趋势（如裂变倾向、总体稳定性），但对幻数与精细结构解释不足。

壳层模型：平均场与强自旋-轨道耦合

引入壳层结构的动机

实验发现某些 ZZ 或 NN 的核特别稳定，称为幻数。为解释这一现象，引入核子在平均势场中运动的量子能级结构，并按泡利原理填充。

强自旋-轨道耦合与幻数

仅靠简单平均势场可得到部分幻数，但要完整解释经验幻数，需要引入强自旋-轨道耦合导致能级重新分裂与重排。由此得到常见幻数序列：

  • 质子幻数：2,8,20,28,50,822, 8, 20, 28, 50, 82
  • 中子幻数：2,8,20,28,50,82,1262, 8, 20, 28, 50, 82, 126

与核自旋规律的联系（补充）

壳层填充直接影响基态核自旋：闭壳层附近核常呈现更“简单”的自旋与磁矩特征，而开壳层核则更复杂。

与宇宙元素丰度的联系（补充）

在天体核合成的 s/r 过程路径上，闭壳层会造成俘获与衰变过程的“堆积”，对应元素丰度峰，这反映了核结构对宇宙化学演化的深刻影响。

多模型互补的总体认识

原子核既呈现集体行为（液滴/振动/转动等），也呈现单粒子行为（壳层占据、魔数效应）。因此：

  • 费米气体模型强调统计与对称能来源
  • 液滴模型擅长描述宏观趋势与能量学
  • 壳层模型解释精细结构、幻数与核自旋等

现实研究中通常综合多种模型视角，才能得到更完整的核图景。
