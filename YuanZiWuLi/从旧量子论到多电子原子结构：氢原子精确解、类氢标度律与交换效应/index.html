从旧量子论到多电子原子结构：氢原子精确解、类氢标度律与交换效应

全局主线与核心问题

实验“逼问”理论的三个问题

原子为何稳定？

按经典电磁学，电子若绕核加速运动应连续辐射并损失能量，最终坠入原子核；但真实原子长期稳定。

光谱为何离散？

经典理论下辐射频率应连续分布；但氢等原子的发射/吸收光谱是离散谱线（而且具有非常规律的线系结构）。

多电子原子为何呈现周期性与精细分裂？

一旦不止一个电子，除了库仑排斥，还必须考虑“全同粒子不可分辨”带来的交换对称与 Pauli 原理，于是出现单态/三重态、Hund 定则、周期律等结构性规律。

“原子假说”的信息密度（思想导引）

如果只能用一句话保留全部科学知识的核心信息，一个常见的选择是：物质由原子构成。这句话的威力在于：它把宏观世界的复杂规律压缩为微观结构与相互作用的后果。

原子观念的形成：从化学原子论到带电结构

从“不可无限分”到化学可检验的原子论

  • 早期关于物质结构的讨论提出“最小单元”的思想。
  • 近代化学定比定律、倍比定律等规律，使“原子/分子”成为可检验模型：化学反应体现为离散粒子的重排。

电子的发现：原子并非不可分

  • 阴极射线等实验揭示了带负电、质量远小于原子质量的粒子（电子）。
  • 结果：原子必须具有内部结构，原子模型进入“带电粒子如何排列”的阶段。

Thomson 模型与其关键问题

  • Thomson 提出“正电荷弥散分布 + 电子嵌入其中”的模型（常被形象比喻为“葡萄干布丁”）。
  • 重要局限：无法解释某些散射与辐射的定量特征（例如对原子散射与辐射尺度的预测与实验不符），也无法自然导出离散光谱规律。

Rutherford 散射实验：有核模型确立

  • α 粒子散射显示：大角散射事件虽少但确实存在，说明正电荷集中在极小体积区域。
  • 得出有核模型：原子大部分是空的，正电荷与大部分质量集中在原子核，电子在核外分布。
  • 随之出现经典悖论：
    • 电子加速辐射 → 轨道不稳定；
    • 辐射频率连续 → 与离散谱线矛盾。

氢原子光谱：离散谱线与线系结构

氢谱的经验规律：Balmer 与线系

  • 可见光区 Balmer 线系可由经验公式拟合，并给出特征波长（例如常见几条可见谱线波长作为例子）。
  • 进一步在紫外/红外区发现其他线系：Lyman（紫外）、Paschen/Brackett/Pfund（红外）等。

里德伯形式：把所有线系统一成“能级差”

  • 统一表达（以波数/频率形式）可写为两个整数之差的函数：

    ν~=R(1nf2−1ni2),ni>nf\tilde{\nu} = R\left(\frac{1}{n_f^2}-\frac{1}{n_i^2}\right),\quad n_i > n_f

  • 物理含义：谱线来自两个离散能级的能量差，实验上等价于“原子具有离散能级”。

折合质量修正：为什么同位素会有微小谱线差异

  • 更精确的能级计算中，需把电子质量替换为折合质量：

    μ=mMm+M\mu=\frac{mM}{m+M}

    这会导致里德伯常数与能级出现微小修正，从而解释同位素（不同核质量）导致的谱线差别。

旧量子论的突破：Bohr 模型（定态 + 跃迁）

两条核心假设

定态假设

电子只能处于某些允许的稳定状态（定态）且在定态不辐射。

频率条件（光子能量来自能级差）

hν=∣Ef−Ei∣h\nu = |E_f-E_i|
这使辐射天然离散：只要能级离散，谱线就离散。

Bohr 模型的主要结果（氢原子）

  • 角动量量子化（经典轨道上附加量子条件）：

    L=mvr=nℏL=mvr=n\hbar

  • 能级标度：

    En=−(μ/me)Z2Ryn2E_n=-\frac{(\mu/m_e) Z^2 \mathrm{Ry}}{n^2}

    对氢 Z=1Z=1，得到熟知的 En∝−1/n2E_n\propto -1/n^2 结构，解释里德伯公式。
  • 基态半径给出典型原子尺度（玻尔半径量级），说明原子大小为何是 Å 量级。

用 de Broglie 物质波理解“定态条件”

  • de Broglie 提出粒子具有波动性，波长：$ \lambda=\frac{h}{p} $
  • Bohr 定态可理解为：轨道周长恰容纳整数个物质波波长

    2πr=nλ2\pi r = n\lambda

    等价于 mvr=nℏmvr=n\hbar。
  • 这一解释把“量子化条件”从神秘假设变成“驻波条件”的直觉结果。

Bohr–Sommerfeld 改进与精细结构常数

  • 允许椭圆轨道并引入相对论修正，可解释部分精细结构来源。
  • 精细结构常数：$ \alpha\approx\frac{1}{137} $
    是电磁相互作用强度的核心无量纲参数，在原子精细效应中反复出现。

氢原子与类氢离子：量子力学的严格解与标度律

类氢体系的普适结论：能量与尺度如何随 Z 变化

对单电子库仑势（氢/类氢离子）：

  • 能级随核电荷数按 Z2Z^2 缩放：

    En∝−μZ2n2E_n\propto -\frac{\mu Z^2}{n^2}

  • 典型长度尺度随 1/Z1/Z 缩放（轨道“变紧”）：

    rn∼n2Za0r_n \sim \frac{n^2}{Z}a_0

    （更精确还会带 me/μm_e/\mu 的修正）
    这解释了：同样的主量子数下，He+^+ 等离子比氢更“紧凑”，跃迁能量更大。

高激发态跃迁与核尺寸测量（动机与图景）

  • 在高主量子数（Rydberg 态）下，能级间隔与波函数在小 rr 区域的敏感性可用于提取核相关信息。
  • 一个重要应用动机：通过精密测量氢原子高激发态能级/跃迁频率，反推质子电荷半径等核结构参数（属于“原子谱学—核结构”交叉）。

薛定谔方程与分离变量：波函数结构

库仑势具有球对称性，定态波函数可分离为

ψnlm(r,θ,ϕ)=Rnl(r)Ylm(θ,ϕ)\psi_{nlm}(r,\theta,\phi)=R_{nl}(r)Y_l^m(\theta,\phi)

  • 角向部分：球谐函数 YlmY_l^m
  • 径向部分：Rnl(r)R_{nl}(r)（常涉及 Laguerre 多项式/关联 Laguerre）

四个量子数与取值规则（含自旋）

  • 主量子数 n=1,2,3,⋯n=1,2,3,\cdots
  • 角量子数 l=0,1,⋯,n−1l=0,1,\cdots,n-1（对应 s,p,d,f,⋯s,p,d,f,\cdots）
  • 磁量子数 m=−l,−l+1,⋯,lm=-l,-l+1,\cdots,l
  • 电子自旋：s=12s=\tfrac12，自旋磁量子数 ms=±12m_s=\pm\tfrac12

简并度：同一能量对应多少状态

对理想氢原子（忽略微小精细效应）：

  • 能量只依赖 nn，因此给定 nn 的轨道简并度为

    ∑l=0n−1(2l+1)=n2\sum_{l=0}^{n-1}(2l+1)=n^2

  • 计入自旋后总态数为 2n22n^2。

概率解释与“轨道形状”

  • 概率密度：∣ψ∣2|\psi|^2
  • 径向分布常写为：

    P(r)=r2∣Rnl(r)∣2P(r)=r^2|R_{nl}(r)|^2

  • 角节点由 ll 决定（决定“瓣”的方向性），径向节点由 (n,l)(n,l) 决定（决定径向峰结构）。
  • 典型形状：
    • s(l=0)s(l=0)：球对称
    • p(l=1)p(l=1)：双叶形
    • d(l=2)d(l=2)：四叶/环状等更复杂结构

宇称（Parity）

氢原子态的宇称由 ll 决定：

π=(−1)l\pi = (-1)^l

宇称与角动量选择定则共同影响哪些跃迁强、哪些跃迁弱或禁戒。

周期表与多电子原子：在“忽略相互作用”的近似上理解排布

壳层/次壳层：K、L、M… 与 s、p、d、f…

  • 壳层由 nn 划分（K 对应 n=1n=1，L 对应 n=2n=2，依此类推）。
  • 每个壳层包含多个次壳层（由 ll 标记：s、p、d、f…）。
  • 每个轨道（给定 n,l,mn,l,m）最多容纳 2 个电子（自旋上下）。

Pauli 不相容原理：电子排布的底层约束

任何两个电子不能拥有完全相同的一组量子数 (n,l,m,ms)(n,l,m,m_s)。
这直接奠定：

  • 壳层容量（例如每个 nn 层最多 2n22n^2 个电子）
  • 周期表的分区结构与化学周期性

屏蔽效应与 ll 简并破缺：为什么多电子原子能级顺序变了

  • 多电子原子中，电子-电子排斥导致核电荷被部分“屏蔽”，引入有效核电荷 ZeffZ_\text{eff}。
  • s 轨道更“穿透”到核附近，受屏蔽较少，能量往往更低；p、d、f 轨道穿透性依次减弱，能量相对更高。
  • 结果：同一 nn 下不同 ll 的简并被打破，形成经验填充顺序（与能量最低原则一致）。

电离能的周期性变化：定性规律

  • 同一周期中 ZeffZ_\text{eff} 整体增强 → 电离能一般上升。
  • 但在次壳层开始填充、电子成对、或达到半满/全满等“较稳定结构”处会出现局部起伏。
  • 稀有气体满壳层电离能高；碱金属外层单电子易失去，电离能低。

典型排布“例外”的物理原因（如 Cr）

  • 某些元素出现偏离最简单填充顺序的组态（例如 Cr）。
  • 其背后通常与开壳层中交换效应增强、半满次壳层稳定性等有关（与 Hund 定则密切相关）。

氦原子与交换效应：从两电子体系看“全同粒子”的威力

描述原子态的项符号 2S+1LJ^{2S+1}L_J

原子态常用项符号描述（尤其在 LS 耦合框架下）：

2S+1LJ{}^{2S+1}L_J

  • SS：总自旋量子数，多重数为 2S+12S+1
  • LL：总轨道角动量量子数（用字母表示：S,P,D,F,G,… 对应 L=0,1,2,3,4,⋯L=0,1,2,3,4,\cdots）
  • JJ：总角动量量子数（由 J=L+S\mathbf{J}=\mathbf{L}+\mathbf{S} 合成）

角动量合成法则：经典直觉与量子规则

经典图像

角动量矢量可按平行四边形法则合成，合成量大小随夹角连续变化。

量子规则

两个角动量 j1,j2j_1, j_2 合成的总角动量 JJ 取离散值：

J=∣j1−j2∣,∣j1−j2∣+1,⋯,j1+j2J = |j_1-j_2|, |j_1-j_2|+1, \cdots, j_1+j_2

这套规则用于从单电子的 l,sl,s 得到原子总的 L,S,JL,S,J。

氦原子电离能：先“拆开”再“修正”的近似思路

忽略核运动时，两电子氦原子的 Hamilton 结构包含：

  • 两个电子各自的动能与核吸引势
  • 电子-电子库仑斥力项 e2/r12e^2/r_{12}

若暂时忽略电子-电子斥力

把两电子当成各自处在 Z=2Z=2 的类氢势中：

  • 单电子能级（类氢标度）：

    En=−13.6eV⋅Z2n2=−13.6eV⋅4n2E_n=-13.6\,\text{eV}\cdot\frac{Z^2}{n^2}=-13.6\,\text{eV}\cdot\frac{4}{n^2}

    基态单电子能量为 −54.4-54.4 eV。
  • 两个电子都在 n=1n=1 时，总能量估计为 −108.8-108.8 eV，
    对应“第一电离能”粗略给出 54.454.4 eV（这是忽略斥力时的过大估计）。

加上电子-电子库仑斥力修正

  • 若用两电子基态波函数的乘积去估算库仑斥力期望值，可得到一个数量级修正（材料中给出约 ΔE∼34\Delta E\sim 34 eV 的结果，并指出实际因为电子相互排斥使平均距离更大，因此修正会小一些）。
  • 真实的氦第一电离能约 24.524.5 eV，说明电子-电子相互作用对能量结构影响巨大。

全同粒子与交换对称：为什么“交换”是物理而不是数学游戏

  • 对全同粒子，交换标签 1 与 2 不应产生可观测差别，因此波函数在交换下必须具有确定对称性。
  • 对比：
    • 光子（γ\gamma）：玻色子 → 交换对称
    • 电子（ee）：费米子 → 交换反对称

Pauli 原理：总波函数必须交换反对称

两电子总波函数可分为“空间 × 自旋”：

Ψtotal=ΨspaceΨspin\Psi_\text{total}=\Psi_\text{space}\,\Psi_\text{spin}

对电子（费米子）必须满足：

Ψtotal(1,2)=−Ψtotal(2,1)\Psi_\text{total}(1,2)=-\Psi_\text{total}(2,1)

因此：

  • 若自旋部分对称（S=1S=1，三重态），则空间部分必须反对称；
  • 若自旋部分反对称（S=0S=0，单态），则空间部分必须对称。
    由此立刻得到 Pauli 不相容原理：两电子不能处于完全相同的量子态。

单态/三重态分裂：为什么三重态通常更低？

概率图景（直觉）

  • 三重态：自旋对称 → 空间反对称 → 两电子更倾向于避开彼此 → 平均库仑斥力更小 → 能量更低。
  • 单态：自旋反对称 → 空间对称 → 两电子更可能靠近 → 库仑斥力更大 → 能量更高。

“交换项”的语言（定性）

在更系统的近似计算（例如变分/微扰/自洽场）中，会出现“直接库仑项”与“交换项”。交换项来自对称化/反对称化，是单态/三重态能量差的重要来源。

氦光谱的跃迁特征：自旋选择定则

  • 电偶极跃迁中常见近似选择定则：

    ΔS=0\Delta S = 0

  • 因此单态系与三重态系之间跃迁被强烈抑制，表现为两套相对独立的谱系结构（也影响某些态的寿命与跃迁强度）。
  • 常用术语：
    • 正氦（parahelium）：单态系
    • 仲氦（orthohelium）：三重态系

推广到多电子：Hund 定则的来源（与交换效应相连）

在同一开壳层内，经验上：

  • 总自旋 SS 越大能量往往越低（电子自旋倾向平行）；
  • 其背后与交换效应导致的“电子彼此避让”有关，从而降低平均库仑斥力；
  • 这也能解释某些组态“例外”（如 Cr）与半满/全满次壳层的稳定性。

结语：三句话串起整套知识

离散谱线 → 能级离散

光谱线系的规律性迫使我们接受：原子能量不是连续的。

旧量子论 → 量子力学严格解

Bohr 用定态与跃迁解释了氢谱主结构；量子力学用薛定谔方程给出波函数、量子数与简并度，形成统一框架。

多电子复杂性来自两件事：屏蔽 + 交换

屏蔽效应改变能级顺序与周期性趋势；全同粒子交换对称与 Pauli 原理引入单/三重态分裂、Hund 定则等结构性规律。
