两体问题与中心势

两体问题概述

本节讨论在无外力作用下两质点通过中心力相互作用时的运动, 即经典的两体问题。
典型实例包括行星在太阳引力场中的运动、电子在库仑场中的运动、粒子与原子核的散射等。

主要内容:

  1. 两体问题向单体中心势问题的约化
  2. 有效势、径向运动与轨道方程
  3. 开普勒问题与圆锥曲线轨道
  4. 拉普拉斯–龙格–楞次矢量
  5. 粒子在中心力场中的弹性散射

两体问题的约化: 质心与相对坐标

拉格朗日量与相互作用势

考虑质量分别为 m1m_{1}, m2m_{2} 的两个质点, 在无外力下仅通过中心势 VV 相互作用:

L=12m1r⃗˙12+12m2r⃗˙22−V(r⃗1−r⃗2)\displaystyle L = \frac{1}{2} m_{1} \dot{\vec{r}}_{1}^{\,2} + \frac{1}{2} m_{2} \dot{\vec{r}}_{2}^{\,2} - V\!\left( \vec{r}_{1} - \vec{r}_{2} \right)

势能只依赖于相对位矢 r⃗1−r⃗2\vec{r}_{1} - \vec{r}_{2}, 体系具有平移对称性, 但目前两粒子的运动是耦合在一起的。

质心坐标与相对坐标

引入总质量与约化质量:

M=m1+m2,1μ=1m1+1m2\displaystyle M = m_{1} + m_{2}, \qquad \frac{1}{\mu} = \frac{1}{m_{1}} + \frac{1}{m_{2}}

定义质心位矢 R⃗\vec{R} 与相对位矢 r⃗\vec{r}:

{R⃗=m1r⃗1+m2r⃗2Mr⃗=r⃗1−r⃗2\begin{cases} \displaystyle \vec{R} = \dfrac{m_{1} \vec{r}_{1} + m_{2} \vec{r}_{2}}{M} \\ \displaystyle \vec{r} = \vec{r}_{1} - \vec{r}_{2} \end{cases}

可以解出各粒子的位置:

{r⃗1=R⃗+m2Mr⃗=R⃗+μm1r⃗r⃗2=R⃗−m1Mr⃗=R⃗−μm2r⃗\begin{cases} \displaystyle \vec{r}_{1} = \vec{R} + \frac{m_{2}}{M} \vec{r} = \vec{R} + \frac{\mu}{m_{1}} \vec{r} \\ \displaystyle \vec{r}_{2} = \vec{R} - \frac{m_{1}}{M} \vec{r} = \vec{R} - \frac{\mu}{m_{2}} \vec{r} \end{cases}

代入拉格朗日量并整理, 得到

L=12MR⃗˙2+12μr⃗˙2−V(r)\displaystyle L = \frac{1}{2} M \dot{\vec{R}}^{\,2} + \frac{1}{2} \mu \dot{\vec{r}}^{\,2} - V(r)

记

Lc=12MR⃗˙2,L′=12μr⃗˙2−V(r)\displaystyle L_{c} = \frac{1}{2} M \dot{\vec{R}}^{\,2}, \qquad L' = \frac{1}{2} \mu \dot{\vec{r}}^{\,2} - V(r)

于是

L=Lc+L′\displaystyle L = L_{c} + L'

物理意义: 两体问题被等效为

  1. 质心作质量为 MM 的自由质点运动;
  2. 约化质量为 μ\mu 的虚拟质点, 在中心势 V(r)V(r) 中的相对运动。

总动量与质心运动

由平移对称性可知, 质心坐标 R⃗\vec{R} 为可遗坐标, 对应的广义动量守恒:

$ \displaystyle \vec{P} = \frac{\partial L}{\partial \dot{\vec{R}}} = M \dot{\vec{R}} = \text{常矢量} $

因此质心做匀速直线运动:

R⃗¨=0\displaystyle \ddot{\vec{R}} = 0

在质心系中 R⃗=0\vec{R} = 0, 整个问题就只剩下相对运动 L′L'.

两体中心力场中的平面运动

动能与角动量的分解

总动能:

T=12m1r⃗˙12+12m2r⃗˙22=12MR⃗˙2+12μr⃗˙2=Tc+T′\displaystyle T = \frac{1}{2} m_{1} \dot{\vec{r}}_{1}^{\,2} + \frac{1}{2} m_{2} \dot{\vec{r}}_{2}^{\,2} = \frac{1}{2} M \dot{\vec{R}}^{\,2} + \frac{1}{2} \mu \dot{\vec{r}}^{\,2} = T_{c} + T'

其中

Tc=12MR⃗˙2,T′=12μr⃗˙2\displaystyle T_{c} = \frac{1}{2} M \dot{\vec{R}}^{\,2}, \qquad T' = \frac{1}{2} \mu \dot{\vec{r}}^{\,2}

总角动量:

J⃗=r⃗1×m1r⃗˙1+r⃗2×m2r⃗˙2=R⃗×MR⃗˙+r⃗×μr⃗˙=J⃗c+l⃗\displaystyle \vec{J} = \vec{r}_{1} \times m_{1} \dot{\vec{r}}_{1} + \vec{r}_{2} \times m_{2} \dot{\vec{r}}_{2} = \vec{R} \times M \dot{\vec{R}} + \vec{r} \times \mu \dot{\vec{r}} = \vec{J}_{c} + \vec{l}

其中

l⃗=r⃗×μr⃗˙\displaystyle \vec{l} = \vec{r} \times \mu \dot{\vec{r}}

称为内禀角动量(或相对角动量)。

内禀角动量守恒与轨道平面

对相对运动拉格朗日量

L′=12μr⃗˙2−V(r)\displaystyle L' = \frac{1}{2} \mu \dot{\vec{r}}^{\,2} - V(r)

相互作用力为中心力

f⃗(r)=−dVdrr⃗^\displaystyle \vec{f}(r) = - \frac{\mathrm{d}V}{\mathrm{d}r} \,\hat{\vec{r}}

其力矩为零:

τ⃗=r⃗×f⃗(r)=0\displaystyle \vec{\tau} = \vec{r} \times \vec{f}(r) = 0

于是内禀角动量守恒:

dl⃗dt=ddt(r⃗×μr⃗˙)=0\displaystyle \frac{\mathrm{d}\vec{l}}{\mathrm{d}t} = \frac{\mathrm{d}}{\mathrm{d}t} \left( \vec{r} \times \mu \dot{\vec{r}} \right) = 0

所以 l⃗\vec{l} 是一个定向不变的常矢量, 相对运动始终在垂直于 l⃗\vec{l} 的平面内进行, 即轨道是一个平面曲线。

这允许我们用平面极坐标 (r,θ)(r,\theta) 来描述相对运动。

极坐标下的拉格朗日量与运动方程

在轨道平面内采用极坐标:

L′=12μ(r˙2+r2θ˙2)−V(r)\displaystyle L' = \frac{1}{2} \mu \left( \dot{r}^{2} + r^{2} \dot{\theta}^{2} \right) - V(r)

由欧拉–拉格朗日方程得到:

径向方程:

μ(r¨−rθ˙2)=−dVdr=f(r)\displaystyle \mu \left( \ddot{r} - r \dot{\theta}^{2} \right) = - \frac{\mathrm{d}V}{\mathrm{d}r} = f(r)

角向方程:

μ(rθ¨+2r˙θ˙)=0\displaystyle \mu \left( r \ddot{\theta} + 2 \dot{r} \dot{\theta} \right) = 0

后者可以写成角动量守恒形式。由

pθ=∂L′∂θ˙=μr2θ˙=l\displaystyle p_{\theta} = \frac{\partial L'}{\partial \dot{\theta}} = \mu r^{2} \dot{\theta} = l

可知 ll 为常数。于是

θ˙=lμr2\displaystyle \dot{\theta} = \frac{l}{\mu r^{2}}

有效势与径向运动

能量守恒与有效势

相对运动的哈密顿量为

H=12μ(r˙2+r2θ˙2)+V(r)=E\displaystyle H = \frac{1}{2} \mu \left( \dot{r}^{2} + r^{2} \dot{\theta}^{2} \right) + V(r) = E

利用角动量守恒 μr2θ˙=l\mu r^{2} \dot{\theta} = l, 得到

12μr˙2+l22μr2+V(r)=E\displaystyle \frac{1}{2} \mu \dot{r}^{2} + \frac{l^{2}}{2 \mu r^{2}} + V(r) = E

定义有效势能

U(r)=l22μr2+V(r)\displaystyle U(r) = \frac{l^{2}}{2 \mu r^{2}} + V(r)

其中第一项

l22μr2\displaystyle \frac{l^{2}}{2 \mu r^{2}}

称为有效离心势。于是径向运动化为一维问题:

12μr˙2+U(r)=E\displaystyle \frac{1}{2} \mu \dot{r}^{2} + U(r) = E

相当于质量为 μ\mu 的质点在一维有效势 U(r)U(r) 中的运动。

径向运动的定性分析

从

12μr˙2=E−U(r)≥0\displaystyle \frac{1}{2} \mu \dot{r}^{2} = E - U(r) \ge 0

可知允许的径向运动范围由

E≥U(r)\displaystyle E \ge U(r)

决定。图像上, 取给定能量 EE, 允许的 rr 值是水平直线 EE 与势能曲线 U(r)U(r) 的交点之间的区域:

  1. 若 E<minU(r)E < \min U(r), 无实数解, 不可能发生。
  2. 若 EE 介于局部极小值与极大值之间, 径向运动在两个转折点之间往复, 是束缚态。
  3. 若 EE 大于势能极值, 粒子可以由无穷远射入再散射回无穷远, 是散射态。

轨道方程与 Binet 方程

由时间参数消去得到轨道方程

从能量方程与角动量守恒出发:

r˙=drdt,θ˙=dθdt=lμr2\displaystyle \dot{r} = \frac{\mathrm{d}r}{\mathrm{d}t}, \qquad \dot{\theta} = \frac{\mathrm{d}\theta}{\mathrm{d}t} = \frac{l}{\mu r^{2}}

消去时间:

drdθ=r˙θ˙=±μr2l2μ[E−U(r)]\displaystyle \frac{\mathrm{d}r}{\mathrm{d}\theta} = \frac{\dot{r}}{\dot{\theta}} = \pm \frac{\mu r^{2}}{l} \sqrt{\frac{2}{\mu} \left[ E - U(r) \right]}

积分得到

θ−θ0=±∫r0rldrμr22μ[E−U(r)]\displaystyle \theta - \theta_{0} = \pm \int_{r_{0}}^{r} \frac{l \,\mathrm{d}r}{\mu r^{2} \sqrt{\dfrac{2}{\mu} \left[ E - U(r) \right]}}

从而得到轨道形式 r=r(θ)r = r(\theta)。

Binet 方程

引入新变量

u(θ)=1r(θ)\displaystyle u(\theta) = \frac{1}{r(\theta)}

有

r˙=drdθθ˙=−lμdudθ\displaystyle \dot{r} = \frac{\mathrm{d}r}{\mathrm{d}\theta} \dot{\theta} = - \frac{l}{\mu} \frac{\mathrm{d}u}{\mathrm{d}\theta}

进一步

r¨=dr˙dt=−l2u2μ2d2udθ2\displaystyle \ddot{r} = \frac{\mathrm{d}\dot{r}}{\mathrm{d}t} = - \frac{l^{2} u^{2}}{\mu^{2}} \frac{\mathrm{d}^{2}u}{\mathrm{d}\theta^{2}}

而

rθ˙2=r(lμr2)2=l2u3μ2\displaystyle r \dot{\theta}^{2} = r \left( \frac{l}{\mu r^{2}} \right)^{2} = \frac{l^{2} u^{3}}{\mu^{2}}

代入径向方程

μ(r¨−rθ˙2)=f(r)\displaystyle \mu \left( \ddot{r} - r \dot{\theta}^{2} \right) = f(r)

得到Binet 方程

d2udθ2+u=−μl2u2f(1u)\displaystyle \frac{\mathrm{d}^{2} u}{\mathrm{d}\theta^{2}} + u = - \frac{\mu}{l^{2} u^{2}} f\!\left( \frac{1}{u} \right)

一旦给定力函数 f(r)f(r), 就可以解出 u(θ)u(\theta), 从而得到轨道 r(θ)r(\theta)。

轨道闭合与 Bertrand 定理(定性)

一般中心势下, 轨道是绕中心点的玫瑰线(会发生进动)。
Bertrand 定理指出: 所有有界轨道都闭合的中心势只有两类:

  1. 引力型势 V(r)∝−1rV(r) \propto -\dfrac{1}{r}
  2. 简谐振子势 V(r)∝r2V(r) \propto r^{2}
例题 1: 利用 Binet 方程求简谐势轨道的形状

考虑平面内的简谐势

V(r)=12kr2\displaystyle V(r) = \frac{1}{2} k r^{2}

其力为

f(r)=−dVdr=−kr\displaystyle f(r) = - \frac{\mathrm{d}V}{\mathrm{d}r} = - k r

代入 Binet 方程:

d2udθ2+u=−μl2u2[−k(1u)]=μkl2u\displaystyle \frac{\mathrm{d}^{2}u}{\mathrm{d}\theta^{2}} + u = - \frac{\mu}{l^{2} u^{2}} \left[ -k \left( \frac{1}{u} \right) \right] = \frac{\mu k}{l^{2} u}

乘以 uu 得

ud2udθ2+u2=μkl2\displaystyle u \frac{\mathrm{d}^{2}u}{\mathrm{d}\theta^{2}} + u^{2} = \frac{\mu k}{l^{2}}

令

u2=y\displaystyle u^{2} = y

则

d2ydθ2+4y=4μkl2\displaystyle \frac{\mathrm{d}^{2}y}{\mathrm{d}\theta^{2}} + 4 y = \frac{4 \mu k}{l^{2}}

这是关于 θ\theta 的简谐振动方程, 解为

y(θ)=μkl2+Acos(2θ+δ)\displaystyle y(\theta) = \frac{\mu k}{l^{2}} + A \cos\left( 2\theta + \delta \right)

从而

r(θ)=1y(θ)\displaystyle r(\theta) = \frac{1}{\sqrt{y(\theta)}}

在适当参数下, 轨道是一个中心在原点的椭圆(频率为径向与角向频率之比的缘故), 从而说明简谐势下有界轨道闭合。

幂律势 V(r)=−αr−nV(r) = -\alpha r^{-n} 的有效势分析

有效势与极值

考虑吸引势

V(r)=−αrn,α>0\displaystyle V(r) = - \frac{\alpha}{r^{n}}, \qquad \alpha > 0

有效势为

U(r)=l22μr2−αrn\displaystyle U(r) = \frac{l^{2}}{2 \mu r^{2}} - \frac{\alpha}{r^{n}}

其导数为

U′(r)=−l2μr3+nαrn+1\displaystyle U'(r) = - \frac{l^{2}}{\mu r^{3}} + \frac{n \alpha}{r^{n + 1}}

令 U′(rm)=0U'(r_{m}) = 0 得到可能的圆轨道半径 rmr_{m}:

l2μrm3=nαrmn+1⇒rmn−2=nαμl2\displaystyle \frac{l^{2}}{\mu r_{m}^{3}} = \frac{n \alpha}{r_{m}^{n + 1}} \quad \Rightarrow \quad r_{m}^{n - 2} = \frac{n \alpha \mu}{l^{2}}

因此若 n≠2n \neq 2, 则存在有限的 rmr_{m} 使得 U′(rm)=0U'(r_{m}) = 0, 即存在圆轨道。

圆轨道的稳定性条件

一般圆轨道半径 rmr_{m} 处, 稳定性由有效势二阶导决定:

存在性:

U′(rm)=0\displaystyle U'(r_{m}) = 0

稳定性:

U′′(rm)>0\displaystyle U''(r_{m}) > 0

对幂律势, 有

U′′(r)=3l2μr4−n(n+1)αrn+2\displaystyle U''(r) = \frac{3 l^{2}}{\mu r^{4}} - \frac{n (n + 1) \alpha}{r^{n + 2}}

在 r=rmr = r_{m} 处代入上式并利用前述关系化简, 可得

U′′(rm)∝2−n\displaystyle U''(r_{m}) \propto 2 - n

因此

  • 当 n<2n < 2 时, 有 U′′(rm)>0U''(r_{m}) > 0, 圆轨道稳定;
  • 当 n>2n > 2 时, 有 U′′(rm)<0U''(r_{m}) < 0, 圆轨道不稳定;
  • 当 n=2n = 2 时, 为临界情况, 稳定性需更精细分析。

不同 nn 值下径向运动的类型

令总能量满足

12μr˙2+U(r)=E\displaystyle \frac{1}{2} \mu \dot{r}^{2} + U(r) = E

结合有效势曲线, 可以得到以下定性结论:

  1. 0<n<20 < n < 2:
    有效势在某个 rmr_{m} 处有极小值 UminU_{\min}。

    • 若 E=Umin<0E = U_{\min} < 0, 发生稳定圆轨道。
    • 若 Umin<E<0U_{\min} < E < 0, 粒子在两个转折点 r1r_{1}, r2r_{2} 之间振荡, 为束缚态(椭圆轨道)。
    • 若 E>0E > 0, 为散射态, 粒子由无穷远射入, 在势场中偏转后离去。

  2. n>2n > 2:
    有效势在某个 rmr_{m} 处有极大值 UmaxU_{\max}。

    • E=UmaxE = U_{\max} 时, 存在不稳定的圆轨道。
    • 0<E<Umax0 < E < U_{\max} 时, 径向运动在 r≤r1r \le r_{1} 或 r≥r2r \ge r_{2} 的区域中, 可能出现落心轨道。
    • E>UmaxE > U_{\max} 时, 粒子可在全空间运动。

  3. n=2n = 2:
    特殊情形 V(r)=−αr2V(r) = - \dfrac{\alpha}{r^{2}}, 有效势为

    U(r)=l22μr2−αr2=12r2(l2μ−2α)\displaystyle U(r) = \frac{l^{2}}{2 \mu r^{2}} - \frac{\alpha}{r^{2}} = \frac{1}{2 r^{2}} \left( \frac{l^{2}}{\mu} - 2 \alpha \right)

    这时有效势 ∝1/r2\propto 1/r^{2}, 其符号取决于 α\alpha 与 l22μ\dfrac{l^{2}}{2 \mu} 的大小比较。

    • 若 α>l22μ\alpha > \dfrac{l^{2}}{2 \mu}, 有效势整体为负, 粒子可能落向原点。
    • 若 α<l22μ\alpha < \dfrac{l^{2}}{2 \mu}, 有效势整体为正, 粒子表现为散射态。
例题 2: 幂律势下存在稳定圆轨道的条件

对于势

V(r)=−αrn,α>0\displaystyle V(r) = - \frac{\alpha}{r^{n}}, \quad \alpha > 0

有效势为

U(r)=l22μr2−αrn\displaystyle U(r) = \frac{l^{2}}{2 \mu r^{2}} - \frac{\alpha}{r^{n}}

  1. 由 U′(rm)=0U'(r_{m}) = 0 求圆轨道半径:

    −l2μrm3+nαrmn+1=0⇒rmn−2=nαμl2\displaystyle - \frac{l^{2}}{\mu r_{m}^{3}} + \frac{n \alpha}{r_{m}^{n + 1}} = 0 \quad \Rightarrow \quad r_{m}^{n - 2} = \frac{n \alpha \mu}{l^{2}}

  2. 再计算二阶导:

    U′′(r)=3l2μr4−n(n+1)αrn+2\displaystyle U''(r) = \frac{3 l^{2}}{\mu r^{4}} - \frac{n (n + 1) \alpha}{r^{n + 2}}

    在 r=rmr = r_{m} 处代入上式并利用前述关系, 得到

    U′′(rm)=(2−n)l2μrm4\displaystyle U''(r_{m}) = \frac{(2 - n) l^{2}}{\mu r_{m}^{4}}

    因此当且仅当 n<2n < 2 时, U′′(rm)>0U''(r_{m}) > 0, 圆轨道是稳定的。

结论: 幂律势 V(r)∝−r−nV(r) \propto - r^{-n} 中存在稳定圆轨道的必要且充分条件是 n<2n < 2。

开普勒问题: V(r)=−αrV(r) = - \dfrac{\alpha}{r}

有效势与稳定圆轨道

对引力型势

V(r)=−αr,α>0\displaystyle V(r) = - \frac{\alpha}{r}, \qquad \alpha > 0

有效势:

U(r)=l22μr2−αr\displaystyle U(r) = \frac{l^{2}}{2 \mu r^{2}} - \frac{\alpha}{r}

对 U(r)U(r) 求导:

U′(r)=−l2μr3+αr2\displaystyle U'(r) = - \frac{l^{2}}{\mu r^{3}} + \frac{\alpha}{r^{2}}

令 U′(rm)=0U'(r_{m}) = 0:

−l2μrm3+αrm2=0⇒rm=l2μα\displaystyle - \frac{l^{2}}{\mu r_{m}^{3}} + \frac{\alpha}{r_{m}^{2}} = 0 \quad \Rightarrow \quad r_{m} = \frac{l^{2}}{\mu \alpha}

二阶导

U′′(r)=3l2μr4−2αr3\displaystyle U''(r) = \frac{3 l^{2}}{\mu r^{4}} - \frac{2 \alpha}{r^{3}}

在 rmr_{m} 处, 可得

U′′(rm)=μ3α4l6>0\displaystyle U''(r_{m}) = \frac{\mu^{3} \alpha^{4}}{l^{6}} > 0

因此存在稳定的圆轨道, 最小有效势为

Umin=−μα22l2<0\displaystyle U_{\min} = - \frac{\mu \alpha^{2}}{2 l^{2}} < 0

由 Binet 方程得到轨道方程

对开普勒力

f(r)=−dVdr=−αr2\displaystyle f(r) = - \frac{\mathrm{d}V}{\mathrm{d}r} = - \frac{\alpha}{r^{2}}

代入 Binet 方程:

d2udθ2+u=−μl2u2(−α(1u)2)=μαl2=1rm\displaystyle \frac{\mathrm{d}^{2}u}{\mathrm{d}\theta^{2}} + u = - \frac{\mu}{l^{2} u^{2}} \left( - \frac{\alpha}{\left( \dfrac{1}{u} \right)^{2}} \right) = \frac{\mu \alpha}{l^{2}} = \frac{1}{r_{m}}

其中

rm=l2μα\displaystyle r_{m} = \frac{l^{2}}{\mu \alpha}

该常微分方程的通解为

u(θ)=1rm+Bcos(θ−θ0)\displaystyle u(\theta) = \frac{1}{r_{m}} + B \cos\left( \theta - \theta_{0} \right)

即

r(θ)=1u(θ)=rm1+Brmcos(θ−θ0)\displaystyle r(\theta) = \frac{1}{u(\theta)} = \frac{r_{m}}{1 + B r_{m} \cos\left( \theta - \theta_{0} \right)}

适当选择极轴方向, 令 θ0=0\theta_{0} = 0, 并定义

p=rm=l2μα,e=Bp\displaystyle p = r_{m} = \frac{l^{2}}{\mu \alpha}, \qquad e = B p

其中 pp 为半通径, ee 为离心率。于是轨道方程为著名的圆锥曲线方程

r(θ)=p1+ecosθ\displaystyle r(\theta) = \frac{p}{1 + e \cos\theta}

  • 0≤e<10 \le e < 1: 椭圆轨道(束缚态, E<0E < 0);
  • e=1e = 1: 抛物线轨道(E=0E = 0);
  • e>1e > 1: 双曲线轨道(散射态, E>0E > 0)。

近日点与远日点

对椭圆轨道 0≤e<10 \le e < 1, 近日点与远日点分别对应 θ=0\theta = 0, θ=π\theta = \pi:

r1=p1+e,r2=p1−e\displaystyle r_{1} = \frac{p}{1 + e}, \qquad r_{2} = \frac{p}{1 - e}

并有

r1+r2=2p1−e2\displaystyle r_{1} + r_{2} = \frac{2 p}{1 - e^{2}}

另一方面, 在转折点 r˙=0\dot{r} = 0 处, 能量方程

l22μr2−αr=E\displaystyle \frac{l^{2}}{2 \mu r^{2}} - \frac{\alpha}{r} = E

对 r1,r2r_{1}, r_{2} 都成立。联立可得

r1+r2=αE,E<0\displaystyle r_{1} + r_{2} = \frac{\alpha}{E}, \qquad E < 0

比较两式可以得到能量与轨道参数的关系

E=α2p(1−e2)=−μα22l2(1−e2)\displaystyle E = \frac{\alpha}{2 p} (1 - e^{2}) = - \frac{\mu \alpha^{2}}{2 l^{2}} (1 - e^{2})

由此解出离心率

e=1+2El2μα2\displaystyle e = \sqrt{1 + \frac{2 E l^{2}}{\mu \alpha^{2}}}

这给出了能量 EE、角动量 ll 与轨道形状之间的联系。

例题 3: 已知能量和角动量判断轨道类型

假设有两体系统满足开普勒势 V(r)=−αrV(r) = - \dfrac{\alpha}{r}, 约化质量为 μ\mu, 相对运动的总能量为 EE, 角动量为 ll。

  1. 计算离心率:

    e=1+2El2μα2\displaystyle e = \sqrt{1 + \frac{2 E l^{2}}{\mu \alpha^{2}}}

  2. 根据 ee 判定轨道类型:

    • 若 E<0E < 0, 则 e<1e < 1, 为椭圆轨道, 半通径

      p=l2μα\displaystyle p = \frac{l^{2}}{\mu \alpha}

    • 若 E=0E = 0, 则 e=1e = 1, 为抛物线轨道。
    • 若 E>0E > 0, 则 e>1e > 1, 为双曲线散射轨道。

  3. 若给定 E<0E < 0, 例如

    E=−μα24l2\displaystyle E = - \frac{\mu \alpha^{2}}{4 l^{2}}

    则

    e=1−12=12\displaystyle e = \sqrt{1 - \frac{1}{2}} = \frac{1}{\sqrt{2}}

    说明轨道为离心率 e=1/2e = 1/\sqrt{2} 的椭圆。

拉普拉斯–龙格–楞次矢量

定义与守恒

在开普勒问题中, 除能量与角动量外, 还存在一个额外守恒量, 反映了体系的动力学对称性, 即拉普拉斯–龙格–楞次矢量(LRL 矢量):

A⃗=p⃗×l⃗−mαr⃗r\displaystyle \vec{A} = \vec{p} \times \vec{l} - m \alpha \frac{\vec{r}}{r}

其中 p⃗=mr⃗˙\vec{p} = m \dot{\vec{r}}, 势为 V(r)=−αrV(r) = - \dfrac{\alpha}{r}。

对 A⃗\vec{A} 求时间导数:

dA⃗dt=dp⃗dt×l⃗+p⃗×dl⃗dt−mαddt(r⃗r)\displaystyle \frac{\mathrm{d}\vec{A}}{\mathrm{d}t} = \frac{\mathrm{d}\vec{p}}{\mathrm{d}t} \times \vec{l} + \vec{p} \times \frac{\mathrm{d}\vec{l}}{\mathrm{d}t} - m \alpha \frac{\mathrm{d}}{\mathrm{d}t} \left( \frac{\vec{r}}{r} \right)

由于 l⃗\vec{l} 守恒, 有

dl⃗dt=0\displaystyle \frac{\mathrm{d}\vec{l}}{\mathrm{d}t} = 0

而牛顿方程给出

dp⃗dt=mr⃗¨=−αr3r⃗\displaystyle \frac{\mathrm{d}\vec{p}}{\mathrm{d}t} = m \ddot{\vec{r}} = - \frac{\alpha}{r^{3}} \vec{r}

经向量恒等式整理可证

dA⃗dt=0\displaystyle \frac{\mathrm{d}\vec{A}}{\mathrm{d}t} = 0

因此 A⃗\vec{A} 为常矢量。

LRL 矢量与轨道几何

注意到

l⃗=r⃗×p⃗\displaystyle \vec{l} = \vec{r} \times \vec{p}

有

A⃗⋅l⃗=(p⃗×l⃗)⋅l⃗−mαr⃗r⋅l⃗=0\displaystyle \vec{A} \cdot \vec{l} = \left( \vec{p} \times \vec{l} \right) \cdot \vec{l} - m \alpha \frac{\vec{r}}{r} \cdot \vec{l} = 0

说明 A⃗\vec{A} 与角动量 l⃗\vec{l} 垂直, 即 A⃗\vec{A} 位于轨道平面内, 并指向轨道的近日点方向。

通过计算 A⃗2\vec{A}^{2}, 可以得到其大小与离心率的关系。利用

p2=2m(E+αr)\displaystyle p^{2} = 2 m \left( E + \frac{\alpha}{r} \right)

以及向量恒等式, 最终可得

A2=m2α2(1+2El2mα2)\displaystyle A^{2} = m^{2} \alpha^{2} \left( 1 + \frac{2 E l^{2}}{m \alpha^{2}} \right)

比较前面给出的离心率表达式

e=1+2El2mα2\displaystyle e = \sqrt{1 + \frac{2 E l^{2}}{m \alpha^{2}}}

得到

A=mαe\displaystyle A = m \alpha e

因此 LRL 矢量的大小正比于离心率, 方向指向近日点。

此外, 由标量积

A⃗⋅r⃗=Arcosθ\displaystyle \vec{A} \cdot \vec{r} = A r \cos\theta

等关系, 可以重新导出轨道方程

r(θ)=l2mα+Acosθ=p1+ecosθ\displaystyle r(\theta) = \frac{l^{2}}{m \alpha + A \cos\theta} = \frac{p}{1 + e \cos\theta}

与之前由 Binet 方程得到的形式完全一致。

动量空间中的圆轨道

在动量空间中, 可以证明粒子的轨迹是一条圆。
由

p⃗×l⃗=A⃗+mαr⃗r\displaystyle \vec{p} \times \vec{l} = \vec{A} + m \alpha \frac{\vec{r}}{r}

两边平方并整理, 可以得到一个关于 p⃗\vec{p} 分量的圆方程。适当选择坐标轴, 例如令 l⃗\vec{l} 沿 zz 方向, A⃗\vec{A} 沿 xx 方向, 则 (px,py)(p_{x}, p_{y}) 在动量平面内的轨迹是一条圆, 其圆心与半径与 α\alpha, AA, ll 有关。这体现了开普勒问题中隐藏的对称性。

例题 4: 由 LRL 矢量求开普勒轨道方程

已知 LRL 矢量定义

A⃗=p⃗×l⃗−mαr⃗r\displaystyle \vec{A} = \vec{p} \times \vec{l} - m \alpha \frac{\vec{r}}{r}

取轨道平面为 xx-yy 平面, 令 l⃗\vec{l} 沿 zz 轴, A⃗\vec{A} 沿 xx 轴。设极角 θ\theta 从 A⃗\vec{A} 方向量起, 则

r⃗=r(cosθ,sinθ,0)\displaystyle \vec{r} = r (\cos\theta, \sin\theta, 0)

两边点乘 r⃗\vec{r}:

A⃗⋅r⃗=(p⃗×l⃗)⋅r⃗−mαr⃗r⋅r⃗=l2−mαr\displaystyle \vec{A} \cdot \vec{r} = (\vec{p} \times \vec{l}) \cdot \vec{r} - m \alpha \frac{\vec{r}}{r} \cdot \vec{r} = l^{2} - m \alpha r

另一方面

A⃗⋅r⃗=Arcosθ\displaystyle \vec{A} \cdot \vec{r} = A r \cos\theta

于是

Arcosθ=l2−mαr\displaystyle A r \cos\theta = l^{2} - m \alpha r

化为

r=l2mα+Acosθ\displaystyle r = \frac{l^{2}}{m \alpha + A \cos\theta}

令

p=l2mα,e=Amα\displaystyle p = \frac{l^{2}}{m \alpha}, \qquad e = \frac{A}{m \alpha}

得到

r(θ)=p1+ecosθ\displaystyle r(\theta) = \frac{p}{1 + e \cos\theta}

这正是开普勒轨道的圆锥曲线方程。

粒子在中心势中的弹性散射

弹性散射与两体约化

考虑两粒子在中心势 V(r)V(r) 下的弹性散射:

  1. 散射过程始终只有相互作用力, 无外力。
  2. 散射前后粒子内部状态不变, 因而静能不变。
  3. 总动量与总能量守恒。

利用两体约化, 在质心系中只需考虑约化质量 μ\mu 的等效粒子在势 V(r)V(r) 中的运动, 初始相对速度为 vv, 总能量

E=12μv2\displaystyle E = \frac{1}{2} \mu v^{2}

瞄准距离与最近距离

设入射方向为水平, 定义:

  • 瞄准距离(碰撞参数) bb: 入射直线与势中心之间的最小距离(如果没有相互作用)。
  • 散射角 θ\theta: 粒子最终速度偏转的角度。
  • 轨道的最近距离 rminr_{\min}: 实际轨道上离势中心的最小距离。

角动量由入射条件给出:

l=μvb\displaystyle l = \mu v b

最接近点处径向速度为零, 即

12μr˙2+U(r)=E⇒E=U(rmin)\displaystyle \frac{1}{2} \mu \dot{r}^{2} + U(r) = E \quad \Rightarrow \quad E = U(r_{\min})

从而

12μv2=l22μrmin2+V(rmin)=μ2v2b22μrmin2+V(rmin)\displaystyle \frac{1}{2} \mu v^{2} = \frac{l^{2}}{2 \mu r_{\min}^{2}} + V(r_{\min}) = \frac{\mu^{2} v^{2} b^{2}}{2 \mu r_{\min}^{2}} + V(r_{\min})

可以用来求 rmin(b,E)r_{\min}(b,E)。

由轨道方程得到散射角

轨道方程(见前文)可写为

drdθ=±μr2l2μ[E−U(r)]\displaystyle \frac{\mathrm{d}r}{\mathrm{d}\theta} = \pm \frac{\mu r^{2}}{l} \sqrt{\frac{2}{\mu} \left[ E - U(r) \right]}

对散射轨道, 从 rminr_{\min} 积分到无穷远, 极角变化记为 φ\varphi:

φ(b,E)=∫rmin(b,E)∞ldrμr22μ[E−U(r)]\displaystyle \varphi(b,E) = \int_{r_{\min}(b,E)}^{\infty} \frac{l \,\mathrm{d}r}{\mu r^{2} \sqrt{\dfrac{2}{\mu} \left[ E - U(r) \right]}}

几何上, 粒子由 −∞-\infty 射入, 经最接近点后转向 +∞+\infty, 整个偏转角为

θ(b,E)=π−2φ(b,E)\displaystyle \theta(b,E) = \pi - 2 \varphi(b,E)

给定势 V(r)V(r), 可通过积分算出 θ(b,E)\theta(b,E), 这就是散射角与瞄准距离的函数关系。

散射截面与微分散射截面

设入射粒子束在单位时间穿过单位面积的粒子数为 II (粒子流密度)。
瞄准距离在 [b,b+db][b, b + \mathrm{d}b] 内的入射粒子数为

dNin=I⋅2πbdb\displaystyle \mathrm{d}N_{\text{in}} = I \cdot 2 \pi b \,\mathrm{d}b

散射后, 粒子在空间中分布。由于中心势具有轴对称性, 散射概率只依赖于极角 θ\theta, 与方位角无关。定义微分散射截面

dσdΩ(θ)\displaystyle \frac{\mathrm{d}\sigma}{\mathrm{d}\Omega}(\theta)

使得落入立体角 dΩ\mathrm{d}\Omega 内的散射粒子数为

dNsc=IdσdΩdΩ\displaystyle \mathrm{d}N_{\text{sc}} = I \frac{\mathrm{d}\sigma}{\mathrm{d}\Omega} \,\mathrm{d}\Omega

在距离散射中心足够远处, 面元面积为

dS=r2dΩ\displaystyle \mathrm{d}S = r^{2} \,\mathrm{d}\Omega

由粒子数守恒 dNin=dNsc\mathrm{d}N_{\text{in}} = \mathrm{d}N_{\text{sc}}, 以及几何关系

dΩ=2πsinθdθ\displaystyle \mathrm{d}\Omega = 2 \pi \sin\theta \,\mathrm{d}\theta

可得到著名关系式:

dσdΩ=bsinθ∣dbdθ∣\displaystyle \frac{\mathrm{d}\sigma}{\mathrm{d}\Omega} = \frac{b}{\sin\theta} \left| \frac{\mathrm{d}b}{\mathrm{d}\theta} \right|

给定 θ(b,E)\theta(b,E) 的反函数 b(θ,E)b(\theta,E), 就可以求出微分散射截面。

例题 5: 利用 "b 对 θ" 关系计算微分散射截面的一般步骤

设已知某种中心势 V(r)V(r) 下, 通过轨道积分得到散射角函数

θ=θ(b,E)\displaystyle \theta = \theta(b,E)

  1. 求反函数 b=b(θ,E)b = b(\theta,E)。

  2. 对 θ\theta 求导:

    dbdθ\displaystyle \frac{\mathrm{d}b}{\mathrm{d}\theta}

  3. 代入

    dσdΩ=bsinθ∣dbdθ∣\displaystyle \frac{\mathrm{d}\sigma}{\mathrm{d}\Omega} = \frac{b}{\sin\theta} \left| \frac{\mathrm{d}b}{\mathrm{d}\theta} \right|

    得到微分散射截面。

  4. 若需要总散射截面, 则进一步积分:

    σtot=∫dσdΩdΩ\displaystyle \sigma_{\text{tot}} = \int \frac{\mathrm{d}\sigma}{\mathrm{d}\Omega} \,\mathrm{d}\Omega

刚球势散射

考虑“硬球”势

V(r)={∞,r<a0,r≥a\displaystyle V(r) = \begin{cases} \infty, & r < a \\ 0, & r \ge a \end{cases}

粒子不能进入 r<ar < a 区域, 相当于在半径 aa 的球面上弹性反射。几何关系给出

b=acosθ2\displaystyle b = a \cos\frac{\theta}{2}

于是

dbdθ=−a2sinθ2\displaystyle \frac{\mathrm{d}b}{\mathrm{d}\theta} = - \frac{a}{2} \sin\frac{\theta}{2}

代入微分截面公式:

dσdΩ=bsinθ∣dbdθ∣=acosθ2sinθ⋅a2sinθ2\displaystyle \frac{\mathrm{d}\sigma}{\mathrm{d}\Omega} = \frac{b}{\sin\theta} \left| \frac{\mathrm{d}b}{\mathrm{d}\theta} \right| = \frac{a \cos\frac{\theta}{2}}{\sin\theta} \cdot \frac{a}{2} \sin\frac{\theta}{2}

利用

sinθ=2sinθ2cosθ2\displaystyle \sin\theta = 2 \sin\frac{\theta}{2} \cos\frac{\theta}{2}

得到

dσdΩ=a24\displaystyle \frac{\mathrm{d}\sigma}{\mathrm{d}\Omega} = \frac{a^{2}}{4}

即刚球势的微分散射截面为常数。积分得总散射截面:

σtot=∫dσdΩdΩ=a24⋅4π=πa2\displaystyle \sigma_{\text{tot}} = \int \frac{\mathrm{d}\sigma}{\mathrm{d}\Omega} \,\mathrm{d}\Omega = \frac{a^{2}}{4} \cdot 4 \pi = \pi a^{2}

这与几何截面完全一致。

Coulomb 势散射与 Rutherford 公式

考虑库仑型势

V(r)=αr\displaystyle V(r) = \frac{\alpha}{r}

取 α>0\alpha > 0 表示排斥势(如带同号电荷粒子散射)。
可算出最近距离 rminr_{\min} 与 bb, EE 的关系, 进而由轨道积分得到

cotθ2=2Ebα\displaystyle \cot\frac{\theta}{2} = \frac{2 E b}{\alpha}

解出

b(θ)=α2Ecotθ2\displaystyle b(\theta) = \frac{\alpha}{2 E} \cot\frac{\theta}{2}

求导:

dbdθ=−α4E1sin2θ2\displaystyle \frac{\mathrm{d}b}{\mathrm{d}\theta} = - \frac{\alpha}{4 E} \frac{1}{\sin^{2}\dfrac{\theta}{2}}

代入微分截面公式:

dσdΩ=bsinθ∣dbdθ∣=α216E21sin4θ2\displaystyle \frac{\mathrm{d}\sigma}{\mathrm{d}\Omega} = \frac{b}{\sin\theta} \left| \frac{\mathrm{d}b}{\mathrm{d}\theta} \right| = \frac{\alpha^{2}}{16 E^{2}} \frac{1}{\sin^{4}\dfrac{\theta}{2}}

这就是著名的Rutherford 散射公式。其特点是:

  1. 向小角度方向散射的粒子数远多于大角度;
  2. 微分截面与能量的平方成反比, EE 越大, 散射截面越小。
例题 6: 利用 Rutherford 公式估算散射概率

设入射粒子具有确定能量 EE, 以库仑势 V(r)=αrV(r) = \dfrac{\alpha}{r} 与重核散射。

  1. 在立体角区间 θ∈[θ1,θ2]\theta \in [\theta_{1}, \theta_{2}] 内的散射截面为

    σ(θ1,θ2)=∫θ1θ2α216E21sin4θ2⋅2πsinθdθ\displaystyle \sigma(\theta_{1}, \theta_{2}) = \int_{\theta_{1}}^{\theta_{2}} \frac{\alpha^{2}}{16 E^{2}} \frac{1}{\sin^{4}\dfrac{\theta}{2}} \cdot 2 \pi \sin\theta \,\mathrm{d}\theta

  2. 令

    x=θ2\displaystyle x = \frac{\theta}{2}

    则

    dθ=2dx,sinθ=2sinxcosx\displaystyle \mathrm{d}\theta = 2 \,\mathrm{d}x, \qquad \sin\theta = 2 \sin x \cos x

    积分可化简为

    σ(θ1,θ2)=πα22E2(cot2θ12−cot2θ22)\displaystyle \sigma(\theta_{1}, \theta_{2}) = \frac{\pi \alpha^{2}}{2 E^{2}} \left( \cot^{2}\frac{\theta_{1}}{2} - \cot^{2}\frac{\theta_{2}}{2} \right)

  3. 给定束流强度 II 与靶厚度等实验条件, 便可估算在该角区间内探测到的散射粒子数。

质心系与实验室系的散射角关系

在实验中, 通常在实验室系下测量散射角, 而理论分析多在质心系中进行, 因此需要两者之间的转换。

设入射粒子质量为 m1m_{1}, 靶粒子质量为 m2m_{2}, 且靶粒子初始静止。
实验室系中:

v⃗1i=v⃗,v⃗2i=0\displaystyle \vec{v}_{1\text{i}} = \vec{v}, \qquad \vec{v}_{2\text{i}} = 0

质心速度:

V⃗=m1v⃗1i+m2v⃗2im1+m2=m1m1+m2v⃗\displaystyle \vec{V} = \frac{m_{1} \vec{v}_{1\text{i}} + m_{2} \vec{v}_{2\text{i}}}{m_{1} + m_{2}} = \frac{m_{1}}{m_{1} + m_{2}} \vec{v}

在质心系中, 入射与散射满足

v⃗1′=−v⃗2′,∣v⃗1′∣=∣v⃗2′∣=v′\displaystyle \vec{v}_{1}^{\,\prime} = - \vec{v}_{2}^{\,\prime}, \qquad \left| \vec{v}_{1}^{\,\prime} \right| = \left| \vec{v}_{2}^{\,\prime} \right| = v^{\prime}

散射角 θ\theta 定义为质心系中相对速度偏转的角度。
实验室系中, 入射粒子散射角记为 Θ1\Theta_{1}, 靶粒子反冲角记为 Θ2\Theta_{2}。

通过速度叠加:

v⃗1f=V⃗+v⃗1f′,v⃗2f=V⃗+v⃗2f′\displaystyle \vec{v}_{1\text{f}} = \vec{V} + \vec{v}_{1\text{f}}^{\,\prime}, \qquad \vec{v}_{2\text{f}} = \vec{V} + \vec{v}_{2\text{f}}^{\,\prime}

可以推得关系(略去详细代数过程):

tanΘ1=sinθcosθ+m1m2\displaystyle \tan\Theta_{1} = \frac{\sin\theta}{\cos\theta + \dfrac{m_{1}}{m_{2}}}

以及

tanΘ2=sinθm2m1+cosθ\displaystyle \tan\Theta_{2} = \frac{\sin\theta}{\dfrac{m_{2}}{m_{1}} + \cos\theta}

特殊情形 m1=m2m_{1} = m_{2} 时:

  1. 有

    Θ1+Θ2=π2\displaystyle \Theta_{1} + \Theta_{2} = \frac{\pi}{2}

    即两粒子散射轨迹在实验室系中互成直角。

  2. 若在质心系中散射角 θ=π\theta = \pi (正向反弹), 则实验室系中入射粒子以 90∘90^{\circ} 偏转, 靶粒子以 0∘0^{\circ} 偏转(沿入射方向前进)。

例题 7: 等质量粒子的弹性散射

考虑 m1=m2m_{1} = m_{2} 的两粒子弹性散射, 靶粒子初始静止。

  1. 质心系中, 入射和散射速度大小相同, 且总动量为零。

  2. 若质心系散射角为 θ\theta, 则实验室系中入射粒子散射角为

    tanΘ1=sinθcosθ+1\displaystyle \tan\Theta_{1} = \frac{\sin\theta}{\cos\theta + 1}

    靶粒子反冲角为

    tanΘ2=sinθ1+cosθ\displaystyle \tan\Theta_{2} = \frac{\sin\theta}{1 + \cos\theta}

    两式给出的角度满足

    Θ1+Θ2=π2\displaystyle \Theta_{1} + \Theta_{2} = \frac{\pi}{2}

    表明两个粒子的散射轨迹互相垂直。

  3. 特别地, 若 θ=π\theta = \pi, 即质心系中发生正向反弹, 则实验室系中入射粒子以 90∘90^{\circ} 偏转, 靶粒子沿原入射方向前进。这一结果在很多低能核散射实验中得到验证。

总结

  1. 通过质心与相对坐标, 两体问题可以约化为质心自由运动与约化质量在中心势中的单体问题。
  2. 在中心力场中, 内禀角动量守恒导致运动限于一平面, 易于采用极坐标描述。
  3. 引入有效势 U(r)U(r), 径向运动化为一维问题, 可以用图像方法定性分析束缚态与散射态。
  4. 轨道方程可通过 Binet 方程求解, 其中开普勒势与简谐势产生闭合轨道。
  5. 开普勒问题中轨道为圆锥曲线, 其形状由能量与角动量(或离心率)决定, 并存在 LRL 矢量这一额外守恒量。
  6. 在散射问题中, 瞄准距离与散射角之间的关系决定微分散射截面; 硬球势与库仑势给出重要的典型范例(Rutherford 散射)。
  7. 实验室系与质心系的散射角之间存在确定的几何关系, 在分析实际散射实验数据时必须加以转换。
