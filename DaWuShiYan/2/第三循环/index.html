半导体 PN 结✅

原理

常温下PN结正向电流随正向电压按指数变化.

PN结有禁带宽，取决于半导体能带结构. 半导体禁带宽度有负的温度系数.

实验步骤

测kBk_B

保持温度不变，测量PN结电流与电压关系，计算玻尔兹曼常数kBk_B.

  1. 将温控装置的设定温度设为25.0℃，改变流过PN结的电流II，观察PN结正向电压UU的变化，确定细测的范围（也就是U∝lnIU \propto lnI的范围），且II过小时仪器读数不准会一直跳动，II不能太小.
  2. 在选定的范围内，仔细测量PN结正向电压UU与电流II的关系. （为了使数据点的分布相对均匀，对于II应该前密后疏）
  3. 对实验数据作最小二乘法拟合，计算玻尔兹曼常数kBk_B，并利用残差图分析实验结果（U−lnIU-lnI公式应包含高次项，由残差图实测斜率偏大，测得的kBk_B偏大）.
  4. 将温控装置的设定温度改为50℃，重复内容2和3（注意：保持电流值相同）. 对实验结果作分析比较. （应取3和4这两个实验中较小的kBk_B为实验结果）

测EgE_g

保持流过PN结的电流不变，测量PN结正向电压与温度的关系，计算禁带宽度EgE_g.

  1. 根据前面实验的结果，选择3组电流值. 【电流数值差距大些，如可选0.2、1.0、5.0微安】
  2. 在保持电流不变的情况下，在10℃到80℃范围内，每隔10℃，测量对应的PN结两端电压UU. 注意温度记录精度要和仪器一致.
  3. 对实验数据作最小二乘法拟合，利用残差图分析实验结果，并计算0K时PN结材料的禁带宽度EgE_g.
  4. 比较不同电流条件下所得结果之间的差异，分析电流大小对实验的影响. （结论：由残差图可知测得的截距偏大，也就是EgE_g偏大. 所以应取最小的. 分析原因：电流大发热大，实验温度高残差偏差更大）

公式

I=Is(EeUKT−1)\displaystyle I = I_s (E^{\frac{eU}{KT}} - 1)

实验中EeUKT≫1E^{\frac{eU}{KT}} \gg 1，近似有

I=IsEeUKT\displaystyle I = I_s E^{\frac{eU}{KT}}

所以推导kBk_B时使用的公式是：

$$ \displaystyle U = \frac{k_B T}{e}lnI + C $$

Is=I0TγeEgKT\displaystyle I_s = I_0 T^\gamma e^{\frac{E_g}{KT}}

所以有：I=I0TγeEg+eUKT\displaystyle I = I_0 T^\gamma e^{\frac{E_g + eU}{KT}}

TγT^\gamma 的变化率小于Eg+eUKT\frac{E_g + eU}{KT}，可以忽略. 所以计算EgE_g的公式为：

$$ \displaystyle U = \frac{C\cdot k_B T - E_g}{e} $$

思考题中，如果不忽略TγT^\gamma，则计算公式为：

$$ \displaystyle U = \frac{k_B T (C - \gamma lnT) - E_g}{e} $$

其中 II 是 PN 结的正向电流，IsI_s是反向饱和电流，T是热力学温度，e是单位电荷，U为PN结正向压降，K为玻尔兹曼常数，γ\gamma是材料绝热系数，EgE_g为PN结禁带宽，CC是常数.

参考数据

kB=1.380649⋅10−23J/Kk_B = 1.380649 \cdot 10^{-23} J/K , Eg=1.166eVE_g = 1.166eV

液氮比汽化热✅

原理

液氮吸热和总的放热平衡，要去掉液氮静置时的吸热. 实验还要考虑和观察莱顿弗罗斯特效应.

实验步骤

  1. 调节电子天平，并测出小铜柱的质量.
  2. 将盛有液氮的保温杯和小铜柱放在天平的托盘上（都要放托盘上！）.
  3. 将电子天平连接到计算机，测量待测物（液氮+容器+铜柱）的总质量MM随时间减少的情况.
  4. 用温度计测量室温T1T_1（测室温时，温度计应放在电子天平内）. 将铜柱轻轻地放入保温杯中，记下放入的时刻tbt_b.
  5. 铜柱投入后立即引起液氮大量汽化，瓶口冒出大量白雾，随时间的推移，白雾越来越少，约一分钟左右. 瓶口突然冒出大量白雾，随即白雾很快消失，立即记下此时的时刻tct_c. （此时铜柱与液氮接触且液氮没有快速汽化，温度相同）
  6. 继续测量M随时间减少的情况.
  7. 编写程序，画出M-t曲线图，从图中计算mNm_N值. （tbt_b、tct_c在图中就是两个拐点）
  8. 根据实验室给出的参数，计算小铜柱放出的热量QQ.
  9. 计算液氮的比汽化热LL.

注意：整个测量过程中，即ab→bc→cd整段时间，应连续记录，千万不能有间断.

公式

比汽化热L=Q/mNL = Q / m_N

对于QQ的测量有3种方法：

  1. 焓变法：

    H=U+pVH = U + pV，对于铜Δ(pV)=0\Delta (pV) = 0，可以用铜的对应温度比焓得Q.

    铜ΔH=m(H300−H78)\Delta H = m(H_{300} - H_{78})

  2. 用比热容：

    Q=m∫T0T1CpdTQ = m\int_{T_0}^{T_1} C_p dT，但是需要注意CpC_p不是常数，实验中以10℃为一个台阶求和计算. 这里用的是铜间接计算.

  3. 用德拜固体比热容模型计算：

    U=9NkBT4TD3∫0TDTx3ex−1dxJ/mol\displaystyle U = \frac{9Nk_BT^4}{T_D^3}\int_0^{\frac{T_D}{T}} \frac{x^3}{e^x - 1} dx \quad J/mol

    其中TDT_D是德拜温度，与材料种类有关；TD,Cu=343KT_{D, Cu} = 343K；NN是原子数目.

mNm_N是汽化的液氮质量，用ab段和cd端分别拟合得到. c点可以稍微向后取一点，防止c处测量不准.

结论

焓变法只能用铜在300℃和78℃的比焓近似，不精确. 德拜公式在高温端计算的U偏小，使测得L偏小. 比热容法误差在于积分近似，但是相对较准. 参考：LN=0.199kJ/gL_N = 0.199 kJ/g

ab段和cd段斜率不同：铜柱增大了液氮表面积

液氮灌太多会被氮气冲出

玻璃折射率温度系数✅

C区是中空的，B区在中间，A区只有中间一块

原理

A. 玻璃折射率温度系数测量原理
利用 A 区测量玻璃折射率温度系数. 将光垂直射向 A 区，光将在玻璃上、下表面 r₁、r₂ 分别反射，反射光垂直向上传播并产生干涉条纹. 温度变化时有：

$$\Delta(2nL) = 2\left(n\frac{\Delta L}{L\cdot\Delta T}L + L\frac{\Delta n}{\Delta T}\right)\Delta T = 2L(n\beta + \gamma)\Delta T \tag{1}$$

光程差会发生变化，γ 即折射率温度系数. 又有Δ(2nL)=mλ\Delta(2nL) = m\lambda，m 为由于光程差改变条纹移动的数目，带入(1)得到：

$$\gamma = \frac{m\lambda}{2L\Delta T} - n\beta \tag{2}$$

产生随时间正弦变化的干涉条纹，在条纹处放置光强探测仪，记录下每个波峰和波谷对应的温度并线性拟合即可计算m−Tm - T图像的斜率，从而求出γ.

B. 玻璃热膨胀系数测量原理
利用 C 区测量玻璃折射率温度系数. 将光垂直 射向 C 区，B 的下表面和B’的上表面的反射光会干涉，光程差为 2L 且随温度升高，B 和 B’ 会膨胀使光程差变化：

$$\Delta(2L) = 2\frac{\Delta L}{\Delta T}\Delta T = 2\frac{\Delta L}{L\cdot\Delta T}L\Delta T = 2\beta L\Delta T \tag{3}$$

β 为玻璃的热膨胀系数，又有Δ(2L)=mλ\Delta(2L) = m\lambda，m 为由于光程差改变条纹移动的数目，带入(3)得到：

$$\beta = \frac{m\lambda}{2L\Delta T} \tag{4}$$

同样产生随时间正弦变化的干涉条纹，在条纹处放置光强探测仪，记录下每个波峰和波谷对应的温度并线性拟合即可计算m−Tm - T图像的斜率，从而求出β.

实验步骤

求折射率温度系数γ\gamma

先把水平仪放在光具座上，通过光具座底部三个调节螺丝调节其水平. 在平台上从左到右依次是氦氖激光器、半透半反镜和平面镜、光屏. 开启激光器并照射到半透半反镜，在光具座上放一个平面镜使光经平面镜反射后能沿原光路返回出光口，此时半透半反镜与水平面成 45∘45^\circ，第一次反射后的光线竖直. 注意 A 区的位置，将大铝块倾斜，在铝块口放置玻璃样品，缓慢滑至底部. 将加热炉放在半平，调节位置正好射向 A 区，且反射光再经半反半透镜反射后正好射入激光出射口. 放入小铝块并微调其位置使激光正好从大孔射入，将温度传感器放入小孔. 调节半反半透镜上方的平面反射镜的角度使得光屏上呈现清晰明亮干涉条纹，将光强传感器吸在光屏上，调节位置使光斑正好对准其中央狭缝，狭缝方向应垂直于条纹放置. 打开加热开关，先观察 30-40℃时数据采集器工作是否正常，光强极大值应大于极小值的 5 倍，相邻波谷的距离应近似相等.
再测量 40-80℃下光强的极大与极小值对应的温度，分别计算斜率作比较，根据已知的λ\lambda，nn，LL，β\beta，求γ\gamma.

求玻璃热膨胀系数β\beta

然后冷却至 30℃以下，取出小铝块，旋转加热台使激光射向 C 区，放入小铝块并微调其位置使激光正好从大孔射入并于小孔放入温度传感器，调整平面反射镜的角度使干涉条纹与激光直接透射半反半透镜产生的亮斑有较大距离（大于光强传感器狭缝长度），此时能观察到 3 个垂直于条纹排列的干涉条纹，中间的较亮，两侧的由 B，B’自身上下表面反射光干涉产生，因 B 和 B’上下表面不平行，所以干涉条纹较暗且位于两侧. 将光强传感器垂直于条纹放置于中央明亮干涉条纹处，重复前述操作以获得 40-80℃下光强的极大与极小值对应的温度，分别计算斜率作比较，根据已知的λ\lambda，LL，求得β\beta.

公式

$L=10.12 \pm 0.05mm,\,𝜆=632.8nm$，

γ=mλ2LΔT−nβ\displaystyle \gamma = \frac{m\lambda}{2L\Delta T} - n\beta

β=mλ2LΔT\displaystyle \beta = \frac{m\lambda}{2L\Delta T}

结论

折射率温度系数数$\gamma=1.71(9)\cdot 10^{−6\circ}C^{−1}$，热膨胀系数$\beta=7.47(10)\cdot 10^{−6\circ} C^{−1}$.

不要忘了做残差图分析！残差图表示公式里忽略了高阶项.
