随机变量的数字特征

包括：数学期望，方差，协方差和协方差矩阵，相关系数，矩，分位数.

数学期望

定义

离散型：

E[X]=∑ixipi=∑ixipX(xi)E[X] = \sum_i x_i p_i = \sum_i x_i p_X(x_i)

连续型：

E[X]=∫0∞xdFX(x)=∫0∞xpX(x)dxE[X] = \int_{0}^{\infty} x dF_X(x) = \int_{0}^{\infty} x p_X(x)dx

实值r.v.：

E[X]=E[X+]−E[X−]=∫−∞∞xdFX(x)=∫−∞∞xpX(x)dxE[X] = E[X^{+}] - E[X^{-}] = \int_{-\infty}^{\infty} x dF_X(x) = \int_{-\infty}^{\infty} x p_X(x) dx

几个常见分布的数学期望

离散分布：

  1. Bernoulli 分布：P(X=1)=pP(X = 1) = p，P(X=0)=1−pP(X = 0) = 1 - p，E[X]=pE[X] = p.

  2. Poisson 分布（λ\lambda）：P(X=k)=λkk!e−λ\displaystyle P(X = k) = \frac{\lambda^k}{k!} e^{-\lambda}，E[X]=λE[X] = \lambda.

  3. 几何分布：XX 为第一次成功的实验次数：P(X=k)=(1−p)k−1p\displaystyle P(X = k) = (1 - p)^{k-1} p，E[X]=1pE[X] = \frac{1}{p}.

  4. 二项分布(n,p)(n,p)：P(X=k)=Cnkpk(1−p)n−kP(X = k) = C_n^k p^k (1 - p)^{n - k}，E[X]=npE[X] = np.

  5. Pascal分布(r,p)(r,p)：XX为rr次成功时，失败的实验次数：P(X=k)=Cr+k−1kpr(1−p)kpP(X = k) = C_{r + k - 1}^k p^r (1 - p)^k p，E[X]=rp−r\displaystyle E[X] = \frac{r}{p} - r.

连续分布：

  1. 均匀分布：a,b∈R,a<ba, b \in \mathbb{R}, a < b，概率密度函数
    p(x)={1b−ax∈[a,b],0x∉[a,b].\displaystyle p(x) = \begin{cases} \dfrac{1}{b-a} & x \in [a, b], \\ 0 & x \notin [a, b]. \end{cases}

    期望 E[X]=a+b2\displaystyle E[X] = \frac{a + b}{2}.

  2. 标准正态分布：Z∼N(0,1)Z \sim N(0, 1)，E[Z]=0E[Z] = 0.

    正态分布：X∼N(μ,σ2)X \sim N(\mu, \sigma^2)，则Z=X−μσ∼N(0,1)\displaystyle Z = \frac{X - \mu}{\sigma} \sim N(0, 1)，E[X]=σE[Z]+μ=μE[X] = \sigma E[Z] + \mu = \mu.

    对数正态分布：Y=eXY = e^X，X∼N(μ,σ)X \sim N(\mu, \sigma)，E[Y]=eμ+12σ2\displaystyle E[Y] = e^{\mu + \frac{1}{2} \sigma^2}.

  3. 指数分布 Exp(λ)Exp(\lambda)：概率密度函数p(x)={λe−λx,x>00,x≤0 p(x) = \begin{cases} \displaystyle \lambda e^{-\lambda x}, & x > 0 \\ 0, & x \leq 0 \end{cases}

    E[X]=1λ \displaystyle E[X] = \frac{1}{\lambda}

  4. Γ\Gamma分布(r,1α)\displaystyle(r, \frac{1}{\alpha})：p(x)={αrΓ(r)xr−1e−αxx>0,0x≤0.p\left(x\right) = \left\{\begin{array}{cc} \frac{\alpha^r}{\Gamma(r)} x^{r - 1} e^{-\alpha x} & x > 0, \\ 0 & x \leq 0. \end{array}\right.，E[X]=rα\displaystyle E[X] = \frac{r}{\alpha}.

  5. Rayleigh分布(σ)(\sigma)：R=X2+Y2R = \sqrt{X^2 + Y^2}，XX与Y∼N(0,σ2)Y \sim N(0, \sigma^2)且独立，E[R]=σπ2\displaystyle E[R] = \sigma \sqrt{\frac{\pi}{2}}.

  3. E[c1X1+c2X2]=c1E[X1]+c2E[X2]E[c_1 X_1 + c_2 X_2] = c_1 E[X_1] + c_2 E[X_2], c1,c2∈Rc_1, c_2 \in R,

方差

定义

D(X)=Var(X):=E[(X−E[X])2]D(X) = Var(X) := E[(X - E[X])^2]

σ(X)=Var(X)\displaystyle \sigma(X) = \sqrt{Var(X)}

性质

设 X,YX, Y 为随机变量，a,b∈Ra, b \in \mathbb{R}，

Var(X)=E[X2]−(E[X])2\text{Var}(X) = E[X^2] - (E[X])^2

$\text{若 } C \text{ 为常数，Var}(C) = 0$

Var(bX+a)=b2Var(X)\text{Var}(bX + a) = b^2\text{Var}(X)

D(X+Y)=D(X)+D(Y)+2E[(X−E[X])(Y−E[Y])]D(X + Y) = D(X) + D(Y) + 2E[(X - E[X])(Y - E[Y])]若 X,YX, Y 独立，则D(X+Y)=D(X)+D(Y)D(X + Y) = D(X) + D(Y)

常见分布的方差

  1. Bernoulli 分布：若随机变量 X∼B(1,p)X \sim B(1,p)，则其方差为

    D(X)=p−p2.D(X) = p - p^2.

  2. 二项分布：若随机变量 X∼B(n,p)X \sim B(n, p)，则其方差为

    D(X)=np(1−p).D(X) = np(1 - p).

  3. Poisson 分布：若随机变量 X∼P(λ)X \sim P(\lambda)，则其方差为

    D(X)=λ.D(X) = \lambda.

  4. 均匀分布：若随机变量 ξ∼U(a,b)\xi \sim U(a,b)，则其方差为

    D(X)=(b−a)212.\displaystyle D(X) = \frac{(b - a)^2}{12}.

  5. 正态分布：若随机变量 X∼N(μ,σ2)X \sim N(\mu, \sigma^2)，令标准化变量 Z=X−μσ\displaystyle Z = \frac{X - \mu}{\sigma}，则 Z∼N(0,1)Z \sim N(0, 1)，且

    D(Z)=1,D(X)=σ2.D(Z) = 1, \quad D(X) = \sigma^2.

  6. 指数分布：若随机变量 X∼Exp(λ)X \sim Exp(\lambda)，则其方差为

    D(X)=1λ2.\displaystyle D(X) = \frac{1}{\lambda^2}.

  7. Γ\Gamma 分布：若随机变量 X∼Γ(r,α)X \sim \Gamma(r, \alpha)，则其方差为

    D(X)=rα2.\displaystyle D(X) = \frac{r}{\alpha^2}.

  8. Rayleigh 分布：若随机变量 XX 服从 Rayleigh 分布，则其方差为

    D(X)=4−π2σ2.\displaystyle D(X) = \frac{4 - \pi}{2}\sigma^2.

  • 独立正态分布随机变量的线性组合性质
    设随机变量列 {Xi}\{X_i\} 相互独立，且每个 XiX_i 服从正态分布 N(μi,σi2)N(\mu_i, \sigma_i^2)（i=1,2,⋯,ni = 1, 2, \cdots, n）. 对于任意实数 Ci∈RC_i \in \mathbb{R}，其线性组合 C1X1+C2X2+⋯+CnXnC_1X_1 + C_2X_2 + \cdots + C_nX_n 仍服从正态分布，即

    C1X1+C2X2+⋯+CnXn∼N(∑i=1nCiμi,∑i=1nCi2σi2).\displaystyle C_1X_1 + C_2X_2 + \cdots + C_nX_n \sim N\left( \sum_{i=1}^n C_i\mu_i, \sum_{i=1}^n C_i^2\sigma_i^2 \right).

    其中，均值由各分量的均值加权求和得到，方差由各分量的方差乘以系数平方后求和得到.

数学期望和方差的几个不等式

  1. Cauchy-Schwarz不等式

    若 X,YX, Y 平方期望存在，则 X⋅YX \cdot Y 绝对可积，且

    ∣E[X⋅Y]∣≤E[X2]⋅E[Y2] \displaystyle \left|E\left[X \cdot Y\right]\right| \leq \sqrt{E[X^2] \cdot E[Y^2]}

  2. Markov不等式

    设 XX 为实值随机变量，若 h:R→[0,+∞)h: \mathbb{R} \to [0, +\infty) 为非负函数，对于实数 m>0m > 0，

    P(h(X)≥m)≤E[h(X)]m \displaystyle P(h(X) \geq m) \leq \frac{E[h(X)]}{m}

  3. 切比雪夫 (Chebyshev) 不等式

    若 XX 的数学期望与方差均存在，记 D(X)=σ2D(X) = \sigma^2，则对于实数 ε>0\varepsilon > 0，

    P(∣X−E[X]∣≥ε)≤σ2ε2 \displaystyle P(\left|X - E[X]\right| \geq \varepsilon) \leq \frac{\sigma^2}{\varepsilon^2}

  4. 方差的性质

    $ \displaystyle Var(X) = 0 \text{ 当且仅当 } P(X = E[X]) = 1 $

协方差和相关系数

定义

XX 与 YY 为两个平方可积的随机变量，其协方差为

Cov(X,Y):=E[(X−E[X])(Y−E[Y])].\displaystyle Cov(X,Y) := E[(X - E[X])(Y - E[Y])].

协方差矩阵

考虑 nn 维随机向量 X=(X1,⋯,Xn)X = (X_1, \cdots, X_n)，定义协方差矩阵如下：

C(X)=(c11c12⋯c1nc21c22⋯c2n⋯⋯⋯⋯cn1cn2⋯cnn),\displaystyle C\left(X\right) = \begin{pmatrix} c_{11} & c_{12} & \cdots & c_{1n} \\ c_{21} & c_{22} & \cdots & c_{2n} \\ \cdots & \cdots & \cdots & \cdots \\ c_{n1} & c_{n2} & \cdots & c_{nn} \end{pmatrix},

其中

cij=Cov(Xi,Xj),i≠j;cii=D(Xi).\displaystyle c_{ij} = \text{Cov}(X_i, X_j), \, i \neq j; \quad c_{ii} = D(X_i).

相关系数为

ρXY=Cov(X,Y)Var(X)Var(Y).\displaystyle \rho_{XY} = \frac{Cov(X,Y)}{\sqrt{Var(X)}\sqrt{Var(Y)}}.

几个性质

协方差性质

  1. Cov(X,Y)=E[XY]−E[X]E[Y]Cov(X,Y) = E[XY] - E[X]E[Y].
  2. Cov(X,Y)=Cov(Y,X)Cov(X,Y) = Cov(Y,X), Cov(X,X)=D(X)≥0Cov(X,X) = D(X) \geq 0.
  3. Cov(aX+c,bY+d)=abCov(X,Y)Cov(aX + c, bY + d) = abCov(X,Y).
  4. Cov(X1+X2,Y)=Cov(X1,Y)+Cov(X2,Y)Cov(X_1 + X_2, Y) = Cov(X_1, Y) + Cov(X_2, Y).

对于两个随机变量 XX 与 YY，

$$ \rho(X,Y)=0 \iff X \text{与} Y \text{不相关. } $$ $$ \displaystyle \text{Var}(X+Y) = \text{Var}(X) + \text{Var}(Y) \iff X \text{与} Y \text{不相关. } $$

多维正态分布随机变量

定义
设 Z=(Z1,⋯,Zn)′Z = (Z_1, \cdots, Z_n)'，其中 ZiZ_i 相互独立，均服从 N(0,1)N(0,1)。对于 m≤nm \leq n，考虑矩阵 A=(aij)1≤i≤m,1≤j≤n\displaystyle A = (a_{ij})_{1 \leq i \leq m, 1 \leq j \leq n} 和向量 μ=(μi)1≤i≤m′\mu = (\mu_i)_{1 \leq i \leq m}'，定义

$X = AZ + \mu$，则 $E[X] = \mu$，协方差矩阵为 $\text{Cov}(X) = \Sigma = AA^T$。

此时称 XX 服从 mm-维正态分布 N(μ,Σ)N(\mu, \Sigma)。若 Σ\Sigma 是正定的（其逆存在），则 XX 的联合密度函数为（设 x=(x1,⋯,xm)′x = (x_1, \cdots, x_m)'）：

$$\displaystyle f\left(x\right) = \frac{1}{(2\pi)^{\frac{m}{2}} \sqrt{\det\Sigma}} \exp\left[ -\frac{1}{2} (x - \mu)^T \Sigma^{-1} (x - \mu) \right]$$

性质

  1. XX 的每个分量 XiX_i 服从正态分布。
  2. XX 的分量的线性组合服从正态分布。
  3. 线性变换 Y=BX+ηY = BX + \eta 服从多维正态分布，BB 为 l×ml \times m 矩阵，η\eta 为 ll-维向量。
  4. XX 分量相互独立的充要条件为 Σ\Sigma 为对角矩阵，即正态 r.v. XX 分量不相关等价于 XX 分量独立。

矩

设 XX 与 YY 为随机变量，

  • 若 E[Xk]E[X^k]，k=2,3,⋯k = 2,3,\cdots，存在，则称为 XX 的 kk 阶原点矩。
  • 若 E[(X−E[X])k]E[(X - E[X])^k]，k=2,3,⋯k = 2,3,\cdots，存在，则称为 XX 的 kk 阶中心矩。
  • 若 E[XkYl]E[X^k Y^l]，k,l=2,3,⋯k,l = 2,3,\cdots，存在，则称为 XX 和 YY 的 k+lk + l 阶混合矩。
  • 若 E[(X−E[X])k(Y−E[Y])l]E[(X - E[X])^k (Y - E[Y])^l]，k,l=2,3,⋯k,l = 2,3,\cdots，存在，则称为 XX 和 YY 的 k+lk + l 阶中心矩。
矩的计算及应用

约定：随机变量 XX。若连续，密度 fX(x)f_X(x)；若离散，概率质量 pX(x)p_X(x)。只在相应矩存在时讨论。

基础定义（尽量记住这几条）

  • kk 阶原点矩（raw moment）

mk=E[Xk]m_k=\mathbb E[X^k]

  • kk 阶中心矩（central moment）

μk=E[(X−E[X])k]\mu_k=\mathbb E[(X-\mathbb E[X])^k]

特别地 μ2=Var(X)\mu_2=\mathrm{Var}(X)。

  • 绝对矩（判断矩是否存在常用）

E[∣X∣k]\mathbb E[|X|^k]

  • 阶乘矩（计数型 X∈{0,1,2,⋯}X\in\{0,1,2,\cdots\} 常用）

(X)k=X(X−1)⋯(X−k+1),E[(X)k](X)_k=X(X-1)\cdots(X-k+1),\qquad \mathbb E[(X)_k]

矩的存在性与常用比较（用来“能不能算/能不能用”）

  • 若 E[∣X∣k]<∞\mathbb E[|X|^k]<\infty，则所有 0≤r≤k0\le r\le k 的 E[∣X∣r]\mathbb E[|X|^r] 也有限。
  • Markov（用矩控尾）：

P(∣X∣≥t)≤E[∣X∣k]tk\mathbb P(|X|\ge t)\le \frac{\mathbb E[|X|^k]}{t^k}

  • Chebyshev（由二阶中心矩）：

P(∣X−E[X]∣≥t)≤Var(X)t2\mathbb P(|X-\mathbb E[X]|\ge t)\le \frac{\mathrm{Var}(X)}{t^2}

例题：用高阶矩给尾部上界（Markov）

已知 E[X4]≤16\mathbb E[X^4]\le 16，求 P(∣X∣≥3)\mathbb P(|X|\ge 3) 上界。

对非负变量 ∣X∣4|X|^4 用 Markov：

P(∣X∣≥3)=P(∣X∣4≥34)≤E[∣X∣4]34≤1681\mathbb P(|X|\ge 3)=\mathbb P(|X|^4\ge 3^4)\le \frac{\mathbb E[|X|^4]}{3^4}\le \frac{16}{81}

答案：$\boxed{\mathbb P(|X|\ge 3)\le 16/81}$。

计算 kk 阶矩的常见套路（按“最常用优先”）

1）定义法：积分 / 求和

  • 连续型：

E[Xk]=∫−∞∞xkfX(x)dx\mathbb E[X^k]=\int_{-\infty}^{\infty}x^k f_X(x)\,dx

  • 离散型：

E[Xk]=∑xxkpX(x)\mathbb E[X^k]=\sum_x x^k p_X(x)

常用 Gamma/Beta 积分：

∫0∞xa−1e−bxdx=Γ(a)ba(a>0,b>0)\int_0^\infty x^{a-1}e^{-bx}\,dx=\frac{\Gamma(a)}{b^a}\quad(a>0,b>0)

∫01xa−1(1−x)b−1dx=B(a,b)=Γ(a)Γ(b)Γ(a+b)\int_0^1 x^{a-1}(1-x)^{b-1}\,dx=B(a,b)=\frac{\Gamma(a)\Gamma(b)}{\Gamma(a+b)}

例题：定义法积分（指数分布的 $k$ 阶矩）

若 X∼Exp(λ)X\sim \mathrm{Exp}(\lambda)（rate λ\lambda），求 E[Xk]\mathbb E[X^k]。

密度 f(x)=λe−λx,x>0f(x)=\lambda e^{-\lambda x},x>0：

E[Xk]=∫0∞xkλe−λxdx\mathbb E[X^k]=\int_0^\infty x^k \lambda e^{-\lambda x}\,dx

换元 u=λx⇒x=u/λ,dx=du/λu=\lambda x\Rightarrow x=u/\lambda,\,dx=du/\lambda：

E[Xk]=1λk∫0∞uke−udu=1λkΓ(k+1)=k!λk\mathbb E[X^k]=\frac{1}{\lambda^k}\int_0^\infty u^k e^{-u}\,du =\frac{1}{\lambda^k}\Gamma(k+1)=\frac{k!}{\lambda^k}

答案：$\boxed{\mathbb E[X^k]=k!/\lambda^k}$。

2）递推：积分分部 / 参数递推（常见分布很省事）

典型：
X∼N(0,1)X\sim N(0, 1) 有

E(X2k)=(2k−1)E(X2k−2),k≥1E(X^{2k}) = (2k-1) E(X^{2k-2}),\quad k \geq 1

奇数阶矩为0.

X∼Γ(α,λ)X\sim \Gamma(\alpha,\lambda)（shape-rate）有

E[Xk]=α+k−1λE[Xk−1]\mathbb E[X^k]=\frac{\alpha+k-1}{\lambda}\,\mathbb E[X^{k-1}]

并且闭式：

E[Xk]=Γ(α+k)Γ(α)⋅1λk\mathbb E[X^k]=\frac{\Gamma(\alpha+k)}{\Gamma(\alpha)}\cdot\frac{1}{\lambda^k}

例题：递推法（Gamma 分布矩递推 + 闭式）

设 X∼Γ(α,λ)X\sim\Gamma(\alpha,\lambda)，证明

E[Xk]=α+k−1λE[Xk−1]\mathbb E[X^k]=\frac{\alpha+k-1}{\lambda}\mathbb E[X^{k-1}]

并写出闭式。

利用 Γ(z+1)=zΓ(z)\Gamma(z+1)=z\Gamma(z)：

E[Xk]E[Xk−1]=Γ(α+k)Γ(α+k−1)⋅1λ=α+k−1λ\frac{\mathbb E[X^k]}{\mathbb E[X^{k-1}]} = \frac{\Gamma(\alpha+k)}{\Gamma(\alpha+k-1)}\cdot\frac{1}{\lambda} =\frac{\alpha+k-1}{\lambda}

故递推成立；闭式为

$$ \boxed{\mathbb E[X^k]=\frac{\Gamma(\alpha+k)}{\Gamma(\alpha)}\cdot\frac{1}{\lambda^k}} $$

3）条件期望：先算“条件矩”，再全期望合并

若 X∣YX|Y 更简单：

E[Xk]=E[E[Xk∣Y]]\mathbb E[X^k]=\mathbb E\big[\mathbb E[X^k\mid Y]\big]

在混合分布里非常常见。

例题：条件期望拆解（Poisson-Gamma 混合的均值方差）

设 X∣Λ∼Poisson(Λ)X\mid \Lambda\sim \mathrm{Poisson}(\Lambda)，且 Λ∼Γ(α,λ)\Lambda\sim \Gamma(\alpha,\lambda)（shape-rate）。求 E[X]\mathbb E[X]、Var(X)\mathrm{Var}(X)。

先用已知条件矩：E[X∣Λ]=Λ\mathbb E[X\mid\Lambda]=\Lambda，Var(X∣Λ)=Λ\mathrm{Var}(X\mid\Lambda)=\Lambda。

全期望：

E[X]=E[E[X∣Λ]]=E[Λ]=αλ\mathbb E[X]=\mathbb E[\mathbb E[X\mid\Lambda]]=\mathbb E[\Lambda]=\frac{\alpha}{\lambda}

方差分解：

Var(X)=E[Var(X∣Λ)]+Var(E[X∣Λ])=E[Λ]+Var(Λ)=αλ+αλ2\mathrm{Var}(X)=\mathbb E[\mathrm{Var}(X\mid\Lambda)]+\mathrm{Var}(\mathbb E[X\mid\Lambda]) =\mathbb E[\Lambda]+\mathrm{Var}(\Lambda) =\frac{\alpha}{\lambda}+\frac{\alpha}{\lambda^2}

答案：$\boxed{\mathbb E[X]=\alpha/\lambda,\ \mathrm{Var}(X)=\alpha/\lambda+\alpha/\lambda^2}$。

4）和的矩：独立时优先用生成函数；不想用也可用展开

  • 若独立：MX+Y(t)=MX(t)MY(t)M_{X+Y}(t)=M_X(t)M_Y(t)，或 KX+Y(t)=KX(t)+KY(t)K_{X+Y}(t)=K_X(t)+K_Y(t)（更好用）。
  • 直接展开（不要求独立，只要能处理交叉项）：

E[(X+Y)k]=∑j=0k(kj)E[XjYk−j]\mathbb E[(X+Y)^k]=\sum_{j=0}^k \binom{k}{j}\mathbb E[X^jY^{k-j}]

若独立则 E[XjYk−j]=E[Xj]E[Yk−j]\mathbb E[X^jY^{k-j}]=\mathbb E[X^j]\mathbb E[Y^{k-j}]。

例题：两独立正态相加，用 CGF 秒算均值方差

若 X∼N(μ1,σ12)X\sim N(\mu_1,\sigma_1^2)，Y∼N(μ2,σ22)Y\sim N(\mu_2,\sigma_2^2) 独立，求 X+YX+Y 的均值与方差。

正态的 CGF：

KX(t)=μ1t+σ12t22,KY(t)=μ2t+σ22t22K_X(t)=\mu_1 t+\frac{\sigma_1^2 t^2}{2},\quad K_Y(t)=\mu_2 t+\frac{\sigma_2^2 t^2}{2}

独立可加：

KX+Y(t)=KX(t)+KY(t)=(μ1+μ2)t+(σ12+σ22)t22K_{X+Y}(t)=K_X(t)+K_Y(t)=(\mu_1+\mu_2)t+\frac{(\sigma_1^2+\sigma_2^2)t^2}{2}

因此

$$ \boxed{\mathbb E[X+Y]=\mu_1+\mu_2,\quad \mathrm{Var}(X+Y)=\sigma_1^2+\sigma_2^2} $$

生成函数：MGF / CGF / PGF / CF（它们和“矩”怎么连）

MGF（矩母函数）

MX(t)=E[etX]M_X(t)=\mathbb E[e^{tX}]

若在 t=0t=0 邻域存在：

E[Xk]=MX(k)(0)\mathbb E[X^k]=M_X^{(k)}(0)

且（形式上）：

MX(t)=∑k=0∞E[Xk]k!tkM_X(t)=\sum_{k=0}^\infty \frac{\mathbb E[X^k]}{k!}t^k

CGF（累积量生成函数）

KX(t)=logMX(t),κk=KX(k)(0)K_X(t)=\log M_X(t),\qquad \kappa_k=K_X^{(k)}(0)

κ1=E[X]\kappa_1=\mathbb E[X]，κ2=Var(X)\kappa_2=\mathrm{Var}(X)，独立可加特别省事。

PGF（概率生成函数，计数型）

GX(s)=E[sX]G_X(s)=\mathbb E[s^X]

GX(k)(1)=E[(X)k]G_X^{(k)}(1)=\mathbb E[(X)_k]

CF（特征函数）

φX(t)=E[eitX]\varphi_X(t)=\mathbb E[e^{itX}]

总存在；若矩存在且允许求导：

E[Xk]=1ikφX(k)(0)\mathbb E[X^k]=\frac{1}{i^k}\varphi_X^{(k)}(0)

例题：MGF 求矩（正态的 $\mathbb E[X]$ 与 $\mathrm{Var}(X)$）

X∼N(μ,σ2)X\sim N(\mu,\sigma^2)。MGF

MX(t)=exp(μt+σ2t22)M_X(t)=\exp\!\left(\mu t+\frac{\sigma^2 t^2}{2}\right)

MX′(t)=(μ+σ2t)MX(t)⇒MX′(0)=μM_X'(t)=(\mu+\sigma^2 t)M_X(t)\Rightarrow M_X'(0)=\mu

MX′′(t)=(σ2+(μ+σ2t)2)MX(t)⇒MX′′(0)=σ2+μ2M_X''(t)=(\sigma^2+(\mu+\sigma^2 t)^2)M_X(t)\Rightarrow M_X''(0)=\sigma^2+\mu^2

所以

Var(X)=E[X2]−E[X]2=MX′′(0)−MX′(0)2=σ2\mathrm{Var}(X)=\mathbb E[X^2]-\mathbb E[X]^2=M_X''(0)-M_X'(0)^2=\sigma^2

答案：$\boxed{\mathbb E[X]=\mu,\ \mathrm{Var}(X)=\sigma^2}$。

例题：PGF 求阶乘矩（Poisson 的 $\mathbb E[(X)_k]$）

若 X∼Poisson(λ)X\sim \mathrm{Poisson}(\lambda)，PGF

GX(s)=exp(λ(s−1))G_X(s)=\exp(\lambda(s-1))

GX(k)(s)=λkexp(λ(s−1))⇒GX(k)(1)=λkG_X^{(k)}(s)=\lambda^k \exp(\lambda(s-1))\Rightarrow G_X^{(k)}(1)=\lambda^k

因此

$$ \boxed{\mathbb E[(X)_k]=\lambda^k} $$

并且 X2=X(X−1)+XX^2=X(X-1)+X，所以

E[X]=λ,E[X2]=λ2+λ\mathbb E[X]=\lambda,\quad \mathbb E[X^2]=\lambda^2+\lambda

常用“矩之间转换”（不想重新积分时很有用）

原点矩 ↔\leftrightarrow 中心矩（低阶最常用）

令 μ=m1=E[X]\mu=m_1=\mathbb E[X]：

μ2=m2−μ2\mu_2=m_2-\mu^2

μ3=m3−3μm2+2μ3\mu_3=m_3-3\mu m_2+2\mu^3

μ4=m4−4μm3+6μ2m2−3μ4\mu_4=m_4-4\mu m_3+6\mu^2 m_2-3\mu^4

例题：由原点矩求方差与三阶中心矩

已知 m1=1m_1=1，m2=3m_2=3，m3=10m_3=10。求 μ2\mu_2 与 μ3\mu_3。

μ2=m2−m12=3−1=2\mu_2=m_2-m_1^2=3-1=2

μ3=m3−3m1m2+2m13=10−9+2=3\mu_3=m_3-3m_1m_2+2m_1^3=10-9+2=3

答案：$\boxed{\mu_2=2,\ \mu_3=3}$。

阶乘矩 ↔\leftrightarrow 原点矩（计数型非常常见）

第二类 Stirling 数 S(k,j)S(k,j)：

Xk=∑j=0kS(k,j)(X)j⇒E[Xk]=∑j=0kS(k,j)E[(X)j]X^k=\sum_{j=0}^k S(k,j)\,(X)_j \Rightarrow \mathbb E[X^k]=\sum_{j=0}^k S(k,j)\,\mathbb E[(X)_j]

第一类 Stirling 数 s(k,j)s(k,j)：

(X)k=∑j=0ks(k,j)Xj⇒E[(X)k]=∑j=0ks(k,j)E[Xj](X)_k=\sum_{j=0}^k s(k,j)X^j \Rightarrow \mathbb E[(X)_k]=\sum_{j=0}^k s(k,j)\mathbb E[X^j]

（实际做题时：Poisson/二项先算 E[(X)k]\mathbb E[(X)_k] 往往更短。）

除了“算矩”，矩的常见应用（更贴近做题）

A）矩估计法（Method of Moments）

用样本矩

m^k=1n∑i=1nXik\hat m_k=\frac1n\sum_{i=1}^n X_i^k

匹配理论矩 mk(θ)m_k(\theta) 解参数 θ\theta。

例题：Gamma(shape-rate) 的矩估计 $(\hat\alpha,\hat\lambda)$

样本来自 Γ(α,λ)\Gamma(\alpha,\lambda)（shape-rate）。设样本均值 X¯\bar X，样本二阶中心矩

S2=1n∑i=1n(Xi−X¯)2S^2=\frac1n\sum_{i=1}^n (X_i-\bar X)^2

理论：

E[X]=αλ,Var(X)=αλ2\mathbb E[X]=\frac{\alpha}{\lambda},\quad \mathrm{Var}(X)=\frac{\alpha}{\lambda^2}

匹配：

X¯=αλ,S2=αλ2⇒λ^=X¯S2,α^=X¯2S2\bar X=\frac{\alpha}{\lambda},\quad S^2=\frac{\alpha}{\lambda^2} \Rightarrow \hat\lambda=\frac{\bar X}{S^2},\quad \hat\alpha=\frac{\bar X^2}{S^2}

答案：$\boxed{\hat\lambda=\bar X/S^2,\ \hat\alpha=\bar X^2/S^2}$。

B）二阶矩方法（Second Moment Method）：证明“有正概率发生”

对非负随机变量 ZZ，常用不等式（由 Cauchy-Schwarz 推出）：

P(Z>0)≥E[Z]2E[Z2]\mathbb P(Z>0)\ge \frac{\mathbb E[Z]^2}{\mathbb E[Z^2]}

这是组合概率、随机图里特别常见的套路。

例题：二阶矩方法下界 $\mathbb P(Z>0)$

设 Z≥0Z\ge 0，且 E[Z]=2\mathbb E[Z]=2、E[Z2]=10\mathbb E[Z^2]=10。给出 P(Z>0)\mathbb P(Z>0) 的下界。

直接代入：

P(Z>0)≥E[Z]2E[Z2]=410=25\mathbb P(Z>0)\ge \frac{\mathbb E[Z]^2}{\mathbb E[Z^2]}=\frac{4}{10}=\frac{2}{5}

答案：$\boxed{\mathbb P(Z>0)\ge 2/5}$。

C）用矩控制收敛（从“矩很小”推出“概率很小”）

最常见：若 E[(Xn−a)2]→0\mathbb E[(X_n-a)^2]\to 0，则 Xn→aX_n\to a 依概率收敛。
理由：Chebyshev

P(∣Xn−a∣≥ε)≤E[(Xn−a)2]ε2→0\mathbb P(|X_n-a|\ge \varepsilon)\le \frac{\mathbb E[(X_n-a)^2]}{\varepsilon^2}\to 0

例题：由二阶矩推出依概率收敛

若 E[(Xn−3)2]≤1n\mathbb E[(X_n-3)^2]\le \frac{1}{n}，证明 Xn→3X_n\to 3 依概率收敛。

对任意 ε>0\varepsilon>0，由 Chebyshev：

$$ \mathbb P(|X_n-3|\ge \varepsilon)\le \frac{\mathbb E[(X_n-3)^2]}{\varepsilon^2} \le \frac{1}{n\varepsilon^2}\xrightarrow[n\to\infty]{}0 $$

因此 Xn→3X_n\to 3 依概率收敛。

D）方差分解（很多模型题就是这一条）

Var(X)=E[Var(X∣Y)]+Var(E[X∣Y])\mathrm{Var}(X)=\mathbb E[\mathrm{Var}(X\mid Y)]+\mathrm{Var}(\mathbb E[X\mid Y])

（上面 Poisson-Gamma 混合那题就是典型用法。）

E）用低阶矩做“分布近似 / 匹配”

常见：用均值+方差选一个简单分布去近似（如用 Gamma 近似正值随机变量，或用正态近似总和）。这在工程统计里很常用。

例题：用均值方差匹配 Gamma 近似

某正值随机变量 Y>0Y>0（未知分布），但测得 E[Y]=10\mathbb E[Y]=10、Var(Y)=20\mathrm{Var}(Y)=20。用 Γ(α,λ)\Gamma(\alpha,\lambda)（shape-rate）做矩匹配近似，给出 (α,λ)(\alpha,\lambda)。

Gamma(shape-rate) 满足：

E[Y]=αλ=10,Var(Y)=αλ2=20\mathbb E[Y]=\frac{\alpha}{\lambda}=10,\quad \mathrm{Var}(Y)=\frac{\alpha}{\lambda^2}=20

解得

λ=E[Y]Var(Y)=1020=12,α=λE[Y]=12⋅10=5\lambda=\frac{\mathbb E[Y]}{\mathrm{Var}(Y)}=\frac{10}{20}=\frac12,\quad \alpha=\lambda\mathbb E[Y]=\frac12\cdot 10=5

答案：$\boxed{\alpha=5,\ \lambda=1/2}$。

常见分布矩（速查，只留最常用几条）

  • Bernoulli(p)(p)：E[X]=p, Var(X)=p(1−p)\mathbb E[X]=p,\ \mathrm{Var}(X)=p(1-p)，且 k≥1k\ge 1 时 E[Xk]=p\mathbb E[X^k]=p。
  • Binomial(n,p)(n,p)：PGF G(s)=(1−p+ps)nG(s)=(1-p+ps)^n，阶乘矩 E[(X)k]=(n)kpk\mathbb E[(X)_k]=(n)_k p^k。
  • Poisson(λ)(\lambda)：PGF G(s)=eλ(s−1)G(s)=e^{\lambda(s-1)}，阶乘矩 E[(X)k]=λk\mathbb E[(X)_k]=\lambda^k。
  • Exponential(λ)(\lambda)：E[Xk]=k!/λk\mathbb E[X^k]=k!/\lambda^k。
  • Gamma(α,λ)(\alpha,\lambda)：E[Xk]=Γ(α+k)/Γ(α)⋅λ−k\mathbb E[X^k]=\Gamma(\alpha+k)/\Gamma(\alpha)\cdot \lambda^{-k}。
  • Normal(μ,σ2)(\mu,\sigma^2)：MGF M(t)=exp(μt+σ2t2/2)M(t)=\exp(\mu t+\sigma^2 t^2/2)，中心偶数矩 μ2k=(2k−1)!!σ2k\mu_{2k}=(2k-1)!!\,\sigma^{2k}，奇数中心矩为 0。

做题选招（超短版）

  • 能积分就积分；遇到 xa−1e−bxx^{a-1}e^{-bx} 优先想 Gamma。
  • 计数分布优先 PGF：先算 E[(X)k]\mathbb E[(X)_k] 再转 E[Xk]\mathbb E[X^k]。
  • 独立和/叠加优先 CGF（可加）；不独立就展开交叉项或条件化。
  • “要证明概率很小”优先 Chebyshev/Markov；“要证明有正概率发生”优先二阶矩法。

分位数, α∈(0,1)\alpha \in (0,1)

① XX为连续型随机变量，具有分布函数F(x)F(x)，和密度函数p(x)p(x)：

  • α\alpha分位数或下α\alpha分位数：xα∈Rx_\alpha \in \mathbb{R}，满足

P(X≤xα)=F(xα)=∫−∞xαp(x)dx=α,\displaystyle P(X \leq x_\alpha) = F(x_\alpha) = \int_{-\infty}^{x_\alpha} p(x)dx = \alpha,

  • 上α\alpha分位数：xα∈Rx_\alpha \in \mathbb{R}，满足

P(X>xα)=1−F(xα)=∫xα∞p(x)dx=α.\displaystyle P(X > x_\alpha) = 1 - F(x_\alpha) = \int_{x_\alpha}^{\infty} p(x)dx = \alpha.

$$ \displaystyle P(X < x_\alpha) \leq \alpha, \text{ 且 } P(X \leq x_\alpha) \geq \alpha. $$
