概统appendix

常用缩写

i.i.d 独立同分布

r.v. 随机变量

a.s. 几乎必然

二维正态分布推导

从独立变量开始

如果 X1X_1 和 X2X_2 独立，且都服从正态分布：

  • X1∼N(μ1,σ12)X_1 \sim N(\mu_1, \sigma_1^2)
  • X2∼N(μ2,σ22)X_2 \sim N(\mu_2, \sigma_2^2)

那么联合密度就是乘积：

f(x1,x2)=f1(x1)⋅f2(x2)=12πσ1σ2exp(−(x1−μ1)22σ12−(x2−μ2)22σ22)f(x_1, x_2) = f_1(x_1) \cdot f_2(x_2) = \frac{1}{2\pi\sigma_1\sigma_2} \exp\left(-\frac{(x_1-\mu_1)^2}{2\sigma_1^2} - \frac{(x_2-\mu_2)^2}{2\sigma_2^2}\right)

引入相关性的条件概率方法

线性相关模型

最简单的相关性模型是线性相关：

$X_2 = a + bX_1 + \text{噪声}$

当给定 X1=x1X_1 = x_1 时，X2X_2 的条件分布应该是：

X2∣X1=x1∼N(μ2+ρσ2σ1(x1−μ1),σ22(1−ρ2))X_2|X_1=x_1 \sim N(\mu_2 + \rho\frac{\sigma_2}{\sigma_1}(x_1-\mu_1), \sigma_2^2(1-\rho^2))

其中 ρ\rho 是相关系数，−1<ρ<1-1 < \rho < 1。

联合密度构建

根据条件概率公式：

f(x1,x2)=f(x1)⋅f(x2∣x1)f(x_1, x_2) = f(x_1) \cdot f(x_2|x_1)

代入得到：

f(x1,x2)=12πσ1e−(x1−μ1)22σ12⋅12πσ21−ρ2exp(−[x2−μ2−ρσ2σ1(x1−μ1)]22σ22(1−ρ2))f(x_1, x_2) = \frac{1}{\sqrt{2\pi}\sigma_1} e^{-\frac{(x_1-\mu_1)^2}{2\sigma_1^2}} \cdot \frac{1}{\sqrt{2\pi}\sigma_2\sqrt{1-\rho^2}} \exp\left(-\frac{[x_2-\mu_2-\rho\frac{\sigma_2}{\sigma_1}(x_1-\mu_1)]^2}{2\sigma_2^2(1-\rho^2)}\right)

化简过程

展开第二个指数并合并同类项，最终得到：

f(x1,x2)=12πσ1σ21−ρ2exp[−12(1−ρ2)((x1−μ1)2σ12−2ρ(x1−μ1)(x2−μ2)σ1σ2+(x2−μ2)2σ22)]f(x_1, x_2) = \frac{1}{2\pi\sigma_1\sigma_2\sqrt{1-\rho^2}} \exp\left[ -\frac{1}{2(1-\rho^2)} \left( \frac{(x_1-\mu_1)^2}{\sigma_1^2} - 2\rho\frac{(x_1-\mu_1)(x_2-\mu_2)}{\sigma_1\sigma_2} + \frac{(x_2-\mu_2)^2}{\sigma_2^2} \right) \right]

几何直观

令 f(x1,x2)=cf(x_1, x_2) = c，得到等概率线方程：

$\frac{(x_1-\mu_1)^2}{\sigma_1^2} - 2\rho\frac{(x_1-\mu_1)(x_2-\mu_2)}{\sigma_1\sigma_2} + \frac{(x_2-\mu_2)^2}{\sigma_2^2} = \text{常数}$

这是椭圆方程：

  • ρ=0\rho = 0 时：圆（独立）
  • ρ>0\rho > 0 时：椭圆向右上方倾斜
  • ρ<0\rho < 0 时：椭圆向右下方倾斜

唯一性说明

满足以下条件的分布形式是唯一的：

  1. 边缘分布必须是一维正态分布
  2. 条件分布也必须是正态分布
  3. 线性相关性（最简单的相关形式）
  4. 概率密度积分为1

因此，我们得到的二维正态分布形式是唯一满足这些条件的分布。

Khinchin弱大数律的证明

定理：设 {Xn}\{X_n\} 是独立同分布的随机变量序列，E[X1]=μ<∞E[X_1] = \mu < \infty，则

$$ \displaystyle \frac{1}{n}\sum_{k=1}^n X_k \xrightarrow{P} \mu $$

证明：

设 Sn=∑k=1nXkS_n = \sum_{k=1}^n X_k，Yn=SnnY_n = \frac{S_n}{n}。我们需要证明 $Y_n \xrightarrow{P} \mu$。

由于 XnX_n 独立同分布，设 ϕ(t)\phi(t) 为 X1X_1 的特征函数。

由特征函数的性质，SnS_n 的特征函数为 ϕSn(t)=[ϕ(t)]n\phi_{S_n}(t) = [\phi(t)]^n。

因此 YnY_n 的特征函数为：

ϕYn(t)=E[eitYn]=E[eitSn/n]=ϕSn(t/n)=[ϕ(t/n)]n\displaystyle \phi_{Y_n}(t) = E[e^{itY_n}] = E[e^{itS_n/n}] = \phi_{S_n}(t/n) = [\phi(t/n)]^n

由于 E[X1]=μ<∞E[X_1] = \mu < \infty，ϕ(t)\phi(t) 在 t=0t=0 处可导，且 ϕ′(0)=iμ\phi'(0) = i\mu。

由泰勒展开，当 t→0t \to 0 时：

ϕ(t)=ϕ(0)+ϕ′(0)t+o(t)=1+iμt+o(t)\displaystyle \phi(t) = \phi(0) + \phi'(0)t + o(t) = 1 + i\mu t + o(t)

因此，当 n→∞n \to \infty 时，对固定的 tt：

ϕYn(t)=[1+iμtn+o(tn)]n\displaystyle \phi_{Y_n}(t) = \left[1 + i\mu \frac{t}{n} + o\left(\frac{t}{n}\right)\right]^n

取对数：

lnϕYn(t)=nln[1+iμtn+o(tn)]\displaystyle \ln \phi_{Y_n}(t) = n \ln\left[1 + i\mu \frac{t}{n} + o\left(\frac{t}{n}\right)\right]

利用 ln(1+x)=x+o(x)\ln(1+x) = x + o(x)：

lnϕYn(t)=n[iμtn+o(tn)]=iμt+n⋅o(tn)\displaystyle \ln \phi_{Y_n}(t) = n\left[i\mu \frac{t}{n} + o\left(\frac{t}{n}\right)\right] = i\mu t + n \cdot o\left(\frac{t}{n}\right)

当 n→∞n \to \infty 时，n⋅o(tn)→0n \cdot o\left(\frac{t}{n}\right) \to 0，所以：

limn→∞ϕYn(t)=eiμt\displaystyle \lim_{n \to \infty} \phi_{Y_n}(t) = e^{i\mu t}

注意到 eiμte^{i\mu t} 是常数随机变量 μ\mu 的特征函数。

$$ \displaystyle Y_n \xrightarrow{d} \mu $$

由于收敛到常数等价于依概率收敛，所以 $Y_n \xrightarrow{P} \mu$。

Levy中心极限定理的证明

定理：设 {Xn}\{X_n\} 是独立同分布的随机变量序列，E[X1]=μE[X_1] = \mu，Var(X1)=σ2<∞Var(X_1) = \sigma^2 < \infty，则

$$ \displaystyle \frac{\sum_{k=1}^n X_k - n\mu}{\sigma\sqrt{n}} \xrightarrow{d} N(0,1) $$

证明：

设 Sn=∑k=1nXk\displaystyle S_n = \sum_{k=1}^n X_k，Zn=Sn−nμσn\displaystyle Z_n = \frac{S_n - n\mu}{\sigma\sqrt{n}}。

设 Yk=Xk−μY_k = X_k - \mu，则 E[Yk]=0E[Y_k] = 0，Var(Yk)=σ2Var(Y_k) = \sigma^2，且 {Yk}\{Y_k\} 独立同分布。

于是 Zn=∑k=1nYkσn\displaystyle Z_n = \frac{\sum_{k=1}^n Y_k}{\sigma\sqrt{n}}。

设 ψ(t)\psi(t) 为 Y1Y_1 的特征函数。由于 E[Y1]=0E[Y_1] = 0，E[Y12]=σ2<∞E[Y_1^2] = \sigma^2 < \infty，ψ(t)\psi(t) 在 t=0t=0 处二阶可导。

由泰勒展开：

ψ(t)=ψ(0)+ψ′(0)t+ψ′′(0)2t2+o(t2)=1−σ2t22+o(t2)\displaystyle \psi(t) = \psi(0) + \psi'(0)t + \frac{\psi''(0)}{2}t^2 + o(t^2) = 1 - \frac{\sigma^2 t^2}{2} + o(t^2)

ZnZ_n 的特征函数为：

ϕZn(t)=E[eitZn]=E[eit∑k=1nYk/(σn)]=[ψ(tσn)]n\displaystyle \phi_{Z_n}(t) = E[e^{itZ_n}] = E\left[e^{it\sum_{k=1}^n Y_k/(\sigma\sqrt{n})}\right] = \left[\psi\left(\frac{t}{\sigma\sqrt{n}}\right)\right]^n

将泰勒展开代入：

ϕZn(t)=[1−σ22(tσn)2+o(t2n)]n=[1−t22n+o(t2n)]n\displaystyle \phi_{Z_n}(t) = \left[1 - \frac{\sigma^2}{2}\left(\frac{t}{\sigma\sqrt{n}}\right)^2 + o\left(\frac{t^2}{n}\right)\right]^n = \left[1 - \frac{t^2}{2n} + o\left(\frac{t^2}{n}\right)\right]^n

取对数：

lnϕZn(t)=nln[1−t22n+o(t2n)]\displaystyle \ln \phi_{Z_n}(t) = n \ln\left[1 - \frac{t^2}{2n} + o\left(\frac{t^2}{n}\right)\right]

利用 ln(1+x)=x+o(x)\ln(1+x) = x + o(x)：

lnϕZn(t)=n[−t22n+o(t2n)]=−t22+n⋅o(t2n)\displaystyle \ln \phi_{Z_n}(t) = n\left[-\frac{t^2}{2n} + o\left(\frac{t^2}{n}\right)\right] = -\frac{t^2}{2} + n \cdot o\left(\frac{t^2}{n}\right)

当 n→∞n \to \infty 时，n⋅o(t2n)→0n \cdot o\left(\frac{t^2}{n}\right) \to 0，所以：

limn→∞ϕZn(t)=e−t2/2\displaystyle \lim_{n \to \infty} \phi_{Z_n}(t) = e^{-t^2/2}

注意到 e−t2/2e^{-t^2/2} 是标准正态分布 N(0,1)N(0,1) 的特征函数。

$$ \displaystyle Z_n \xrightarrow{d} N(0,1) $$

即：

$$ \displaystyle \frac{\sum_{k=1}^n X_k - n\mu}{\sigma\sqrt{n}} \xrightarrow{d} N(0,1) $$

证毕。
