拉格朗日量：

L=12m1r⃗˙12+12m2r⃗˙22−V(r⃗1−r⃗2)L = \frac{1}{2} m_1 \dot{\vec{r}}_1^2 + \frac{1}{2} m_2 \dot{\vec{r}}_2^2 - V(\vec{r}_1 - \vec{r}_2)

其中不含可遗坐标.

引入质心位矢 R⃗\vec{R} 和相对位矢 r⃗\vec{r}：

{R⃗=m1r⃗1+m2r⃗2m1+m2r⃗=r⃗1−r⃗2\begin{cases} \vec{R} = \frac{m_1 \vec{r}_1 + m_2 \vec{r}_2}{m_1 + m_2} \\ \vec{r} = \vec{r}_1 - \vec{r}_2 \end{cases}

总质量 M=m1+m2M = m_1 + m_2，约化质量 μ\mu 满足：

1μ=1m1+1m2\frac{1}{\mu} = \frac{1}{m_1} + \frac{1}{m_2}

位置关系：

{r⃗1=R⃗+m2Mr⃗=R⃗+μm1r⃗r⃗2=R⃗−m1Mr⃗=R⃗−μm2r⃗\begin{cases} \vec{r}_1 = \vec{R} + \frac{m_2}{M} \vec{r} = \vec{R} + \frac{\mu}{m_1} \vec{r} \\ \vec{r}_2 = \vec{R} - \frac{m_1}{M} \vec{r} = \vec{R} - \frac{\mu}{m_2} \vec{r} \end{cases}

拉格朗日量分解：

L=12MR⃗˙2+12μr⃗˙2−V(r⃗)=Lc+L′L = \frac{1}{2} M \dot{\vec{R}}^2 + \frac{1}{2} \mu \dot{\vec{r}}^2 - V(\vec{r}) = L_c + L'

解耦：将两体问题等效为两个单体问题：整体运动和相对运动

现在可考虑动能和角动量分解：

T=Tc+T′=12MR⃗˙2+12μr⃗˙2T = T_c + T' = \frac{1}{2} M \dot{\vec{R}}^2 + \frac{1}{2} \mu \dot{\vec{r}}^2

J⃗=J⃗c+J⃗′=R⃗×MR⃗˙+r⃗×μr⃗˙\vec{J} = \vec{J}_c + \vec{J}' = \vec{R} \times M \dot{\vec{R}} + \vec{r} \times \mu \dot{\vec{r}}

相对运动拉格朗日量（极坐标）：

L′=12μ(r˙2+r2θ˙2)−V(r)L' = \frac{1}{2} \mu ( \dot{r}^2 + r^2 \dot{\theta}^2 ) - V(r)

运动方程：

{μ(r¨−rθ˙2)=−∂V∂r=f(r)μ(rθ¨+2r˙θ˙)=0\begin{cases} \mu ( \ddot{r} - r \dot{\theta}^2 ) = -\frac{\partial V}{\partial r} = f(r) \\ \mu ( r \ddot{\theta} + 2 \dot{r} \dot{\theta} ) = 0 \end{cases}

守恒量：

  1. 角动量守恒：

    ∂L∂θ=0\frac{\partial L}{\partial \theta} = 0

pθ=μr2θ˙=l(const.)p_\theta = \mu r^2 \dot{\theta} = l \quad (\text{const.})

  1. 两体内能量守恒：

    ∂L∂t=0\frac{\partial L}{\partial t} = 0

H=12μ(r˙2+r2θ˙2)+V(r)=E(const.)H = \frac{1}{2} \mu ( \dot{r}^2 + r^2 \dot{\theta}^2 ) + V(r) = E \quad (\text{const.})

将以上两个守恒式联立得：

12μr˙2+l22μr2+V(r)=E\frac{1}{2} \mu \dot{r}^2 + \frac{l^2}{2 \mu r^2} + V(r) = E

定义有效势能：

U(r)=l22μr2+V(r)U(r) = \frac{l^2}{2 \mu r^2} + V(r)

其中 l22μr2\frac{l^{2}}{2\mu r^{2}} 可视为有效离心势能

则有径向运动方程：

12μr˙2+U(r)=E\frac{1}{2} \mu \dot{r}^2 + U(r) = E

等效为质点 μ\mu 在等效势场 UU 中的一维运动

轨道方程（Binet方程）的推导：

drdt=±2μ[E−U(r)]\frac{\mathrm{d}r}{\mathrm{d}t} = \pm \sqrt{\frac{2}{\mu}\left[ E - U(r) \right]}

t−t0=±μ2∫r0rdrE−U(r)⟶r=r(t)t - t_0 = \pm \sqrt{\frac{\mu}{2}} \int_{r_0}^r \frac{\mathrm{d}r}{\sqrt{E - U(r)}} \longrightarrow r = r(t)

dθdt=lμr2\frac{\mathrm{d}\theta}{\mathrm{d}t} = \frac{l}{\mu r^2}

θ−θ0=lμ∫t0tdt[r(t)]2⟶θ=θ(t) \theta - \theta_0 = \frac{l}{\mu} \int_{t_0}^t \frac{\mathrm{d}t}{[r(t)]^2} \longrightarrow \theta = \theta(t)

drdθ=±μr2l2μ[E−U(r)] \frac{\mathrm{d}r}{\mathrm{d}\theta} = \pm \frac{\mu r^2}{l} \sqrt{\frac{2}{\mu}\left[ E - U(r) \right]}

代换：

u=1rdudθ=∓1l2μ[E−U(1u)]u = \frac{1}{r} \quad \frac{\mathrm{d}u}{\mathrm{d}\theta} = \mp \frac{1}{l} \sqrt{2\mu \left[ E - U\left( \frac{1}{u} \right) \right]}

θ−θ0=∓∫u0uldu2μ[E−U(1/u)]1r=u=u(θ)r=r(θ)\theta - \theta_0 = \mp \int_{u_0}^u \frac{l \mathrm{d}u}{\sqrt{2\mu \left[ E - U\left( 1/u \right) \right]}} \quad \frac{1}{r} = u = u(\theta) \quad r = r(\theta)

当 $( V® \propto \frac{1}{r} )，( \frac{1}{r^2} ) $时，可解得轨道闭合. 这两种情形为万有引力 胡克定律

μ(r¨−rθ˙2)=f(r)1r=u\mu (\ddot{r} - r \dot{\theta}^2) = f(r) \quad \frac{1}{r} = u

r˙=drdt=dθdtdrdθ=θ˙drdθ=lμr2drdθ=−lμdudθ\dot{r} = \frac{\mathrm{d}r}{\mathrm{d}t} = \frac{\mathrm{d}\theta}{\mathrm{d}t} \frac{\mathrm{d}r}{\mathrm{d}\theta} = \dot{\theta} \frac{\mathrm{d}r}{\mathrm{d}\theta} = \frac{l}{\mu r^2} \frac{\mathrm{d}r}{\mathrm{d}\theta} = -\frac{l}{\mu} \frac{\mathrm{d}u}{\mathrm{d}\theta}

r¨=dr˙dt=dθdtdr˙dθ=θ˙dr˙dθ=lμr2ddθ(−lμdudθ)=−l2u2μ2d2udθ2\ddot{r} = \frac{\mathrm{d}\dot{r}}{\mathrm{d}t} = \frac{\mathrm{d}\theta}{\mathrm{d}t} \frac{\mathrm{d}\dot{r}}{\mathrm{d}\theta} = \dot{\theta} \frac{\mathrm{d}\dot{r}}{\mathrm{d}\theta} = \frac{l}{\mu r^2} \frac{\mathrm{d}}{\mathrm{d}\theta} \left( -\frac{l}{\mu} \frac{\mathrm{d}u}{\mathrm{d}\theta} \right) = -\frac{l^2 u^2}{\mu^2} \frac{\mathrm{d}^2 u}{\mathrm{d}\theta^2}

rθ˙2=r(lμr2)2=l2u3μ2r \dot{\theta}^2 = r \left( \frac{l}{\mu r^2} \right)^2 = \frac{l^2 u^3}{\mu^2}
