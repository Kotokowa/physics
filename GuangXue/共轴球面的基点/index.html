共轴球面的基点

主要包括：主焦点，主点，节点.

1. 主点与主面

任何复杂的共轴球面系统，在近轴区域内，都可以等效为一个简单的“黑箱”系统。这个系统的成像性质可以由几对特殊的点和面完全描述，它们被称为基点和基面。

主面 将系统对光线的多次的偏折等效为主面对光线产生的一次偏折. 物空间和像空间主面有各自对应的焦点，成像规律与薄透镜相同. 它是一对共轭平面。设第一主面为 H\mathcal{H}，第二主面为 H′\mathcal{H}'。从 H\mathcal{H} 上任意一点 PP 发出的任意光线，经过系统后，必然从 H′\mathcal{H}' 上的共轭点 P′P' 射出，且 PP 和 P′P' 的高度相等，即 y=y′y=y'。

这意味着，主面是系统中横向放大率 β=+1\beta = +1 的一对共轭平面。

主点 是主面与光轴的交点。第一主点 HH 和第二主点 H′H' 分别是第一主面 H\mathcal{H} 和第二主面 H′\mathcal{H}' 与光轴的交点。

主点和主面的引入，将一个复杂系统的成像问题简化为：光线在第一主面发生一次折射，然后在两主面之间沿平行于光轴的方向传播，最后在第二主面发生第二次折射。

2. 主点、主面及焦点位置计算

我们采用 Zemax 或工程光学的符号约定：

  • 光线从左向右传播。
  • 沿光线传播方向为正。
  • 光轴之上为正。
  • 曲率中心在像方（界面右侧）时，曲率半径 rr 为正。

2.1 单独厚透镜

对于一个折射率为 nn，置于空气（n1=n2=1n_1=n_2=1）中，厚度为 dd 的厚透镜，其两个表面的曲率半径分别为 r1r_1 和 r2r_2。

系统的光焦度 Φ\Phi 和焦距 f′f' 为：

Φ=(n−1)(1r1−1r2+(n−1)dnr1r2)\displaystyle \Phi = (n-1)\left( \frac{1}{r_1} - \frac{1}{r_2} + \frac{(n-1)d}{n r_1 r_2} \right)

f′=1Φ\displaystyle f' = \frac{1}{\Phi}

主点位置（从透镜表面顶点算起，右侧为正）的计算公式为：

第一主点 HH 到第一表面顶点的距离 lHl_H：

lH=−f′(n−1)dnr2\displaystyle l_H = -\frac{f'(n-1)d}{n r_2}

第二主点 H′H' 到第二表面顶点的距离 lH′l_{H'}：

lH′=−f′(n−1)dnr1\displaystyle l_{H'} = -\frac{f'(n-1)d}{n r_1}

2.2 薄透镜组

对于两个光焦度分别为 Φ1\Phi_1 和 Φ2\Phi_2 的薄透镜，它们之间的空气间隔为 dd。

系统的总光焦度 Φ\Phi 为：

Φ=Φ1+Φ2−dΦ1Φ2\Phi = \Phi_1 + \Phi_2 - d \Phi_1 \Phi_2

系统焦距 f′=1/Φf' = 1/\Phi。

第一主点 HH 到第一个薄透镜的距离 lHl_H：

lH=f′dΦ2Φ\displaystyle l_H = \frac{f' d \Phi_2}{\Phi}

第二主点 H′H' 到第二个薄透镜的距离 lH′l_{H'}：

lH′=−f′dΦ1Φ\displaystyle l_{H'} = -\frac{f' d \Phi_1}{\Phi}

3. 节点

节点 是光轴上另一对特殊的共轭点，记为第一节点 NN 和第二节点 N′N'。

其定义为：任意一条穿过第一节点 NN 的入射光线，其共轭出射光线必定穿过第二节点 N′N'，并且与入射光线平行（即方向不变）。

这意味着，节点是系统中角放大率 γ=1\gamma = 1 的一对共轭点。

节点位置与主点位置的关系为：

HN=f′+fHN = f' + f

H′N′=f′+fH'N' = f' + f

其中 ff 是物方焦距，f′f' 是像方焦距。它们的关系为 f′=−n′nf\displaystyle f' = -\frac{n'}{n}f。

当系统物方和像方介质相同时（n=n′n=n'），有 f′=−ff'=-f，此时 HN=0HN=0 且 H′N′=0H'N'=0。这意味着节点与主点重合。

4. 作图法及其性质

利用基点基面，我们可以通过追踪三条特殊光线来快速确定像的位置和大小。

  1. 平行光线：平行于光轴的入射光线，经过系统后，其出射光线必然通过第二焦点 F′F'。
  2. 焦点光线：通过第一焦点 FF 的入射光线，经过系统后，其出射光线必然平行于光轴。
  3. 节点光线：通过第一节点 NN 的入射光线，经过系统后，其出射光线通过第二节点 N′N' 且方向不变。

补充性质：

  • 主面共轭性：如前所述，第一主面 H\mathcal{H} 上的任意点 P(y)P(y) 与第二主面 H′\mathcal{H}' 上的点 P′(y′)P'(y') 共轭，且 y′=yy'=y。这是主面的定义，也是作图法中光线在两主面间“平移”的依据。
  • 焦面共轭性：第一焦面 F\mathcal{F} 上的任意点（轴外点）与无穷远处的一个特定方向共轭。反之，来自无穷远处某一方向的平行光束，将会聚于第二焦面 F′\mathcal{F}' 上的一个对应点。
  • 节点共轭性：节点 NN 和 N′N' 是一对共轭点，保证了光线方向的连续性。

5. 高斯公式与牛顿公式的验证

引入主点和主面的概念，正是为了统一和简化成像公式。因此，在这些定义下，经典的高斯公式和牛顿公式依然成立，但距离的起点需要明确。

  • 高斯公式：物距 ll 从第一主点 HH 算起，像距 l′l' 从第二主点 H′H' 算起。

    n′l′−nl=n′f′=Φ\displaystyle \frac{n'}{l'} - \frac{n}{l} = \frac{n'}{f'} = \Phi

    对于空气中的系统，n=n′=1n=n'=1，简化为：

    1l′−1l=1f′\displaystyle \frac{1}{l'} - \frac{1}{l} = \frac{1}{f'}

  • 牛顿公式：物距 xx 从第一焦点 FF 算起，像距 x′x' 从第二焦点 F′F' 算起。

    xx′=ff′x x' = f f'

验证：这两个公式是近轴光学成像的基本规律。主点和主面的定义是构造性的，它们被定义为一对 β=+1\beta=+1 的共轭平面。在这样的定义下，可以严格推导出上述两个公式依然成立。所以，答案是肯定的，只要我们正确地从对应的基点开始测量距离，这两个公式就完全正确。

6. 光焦度与放大率公式

  • 光焦度：表征系统汇聚光线能力的物理量。

    Φ=n′f′=−nf\displaystyle \Phi = \frac{n'}{f'} = -\frac{n}{f}

  • 三个放大率：
    1. 横向放大率 β\beta：像高与物高之比。

      β=y′y=−fx=−x′f′=nl′n′l\displaystyle \beta = \frac{y'}{y} = -\frac{f}{x} = -\frac{x'}{f'} = \frac{n l'}{n' l}

    2. 轴向放大率 α\alpha：沿光轴方向的微小线段成像后的长度之比。

      α=dx′dx=−x′x=n′nβ2\displaystyle \alpha = \frac{dx'}{dx} = -\frac{x'}{x} = \frac{n'}{n} \beta^2

    3. 角放大率 γ\gamma：出射光线与光轴夹角 u′u' 和入射光线与光轴夹角 uu 之比。

      γ=u′u=xf′=fx′=nn′1β\displaystyle \gamma = \frac{u'}{u} = \frac{x}{f'} = \frac{f}{x'} = \frac{n}{n'} \frac{1}{\beta}

  • 三个放大率之间的关系：
    这三个放大率并非独立，它们之间存在一个重要的关系：

    α⋅γ=β\alpha \cdot \gamma = \beta

    或者更常见的形式：

    α=βγ\alpha = \beta \gamma

    同时，从公式中也可以看出：

    β⋅γ=nn′\displaystyle \beta \cdot \gamma = \frac{n}{n'}

    对于物像方介质相同的系统（n=n′n=n'），有 β⋅γ=1\beta \cdot \gamma = 1。
