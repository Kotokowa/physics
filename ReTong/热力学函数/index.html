热力学函数

热力学函数的引入

热力学系统的宏观状态原始参量：几何𝑉、力学𝑝、电磁学、化学参量𝑁

从热力学定律可推出高级参量：温度𝑇、内能𝑈、熵𝑆、焓𝐻

高级参量可以表示为原始参量的函数 eg.: 𝑈 = 𝑈(𝑝,𝑉) = 𝑈(𝑇,𝑉) = ···

进一步简化，以高级参量作为自变量，可以有 𝑈 = 𝑈(𝑆,𝑉) = ···

这个函数只依赖系统参量而无需外界参数，更加方便。

同样，引入更多的热力学函数 eg.: 𝐹 = 𝐹(𝑇,𝑉) = 𝑈 −𝑇𝑆

Legendre 变换

引入

若有函数 𝑓(𝑥,···)，则 x 和 𝑦 = ∂x\partial_x𝑓(𝑥,···) 一般称为共轭参量。例如：哈密顿量 𝐻(𝒓,𝒑) 包含了 𝒓 和 𝒑 随时间的改变 v = r˙\dot{\bold{r}} = 𝜕𝒑𝐻，𝑭 = p˙\dot{\bold{p}} = −𝜕𝒓𝐻

目的：使用共轭参量为自变量时，改变函数形式，使得函数里包含的信息保持不变。于是提出 Legendre 变换

设有 y=y(x)=∂xf(x)y = y(x) = \partial_x f(x)，反解得到 x=x(y)x = x(y)

直接令 f∗(y)=f(x(y))f^*(y) = f(x(y))

以 f(x)=ax2+bx+cf(x) = ax^2 + bx + c 为例，代入 y=∂xf(x)y = \partial_x f(x)，得到 y=2ax+by = 2ax + b 或 x=y−b2ax = \frac{y-b}{2a}

得到 f∗(y)f^*(y)，再代入 x=∂yf∗(y)x = \partial_y f^*(y)，得到 x=y2ax = \frac{y}{2a}

只保留了斜率信息，而截距丢失！

对于这个例子，Legendre 变换 f~(y)=f∗(y)−xy\tilde{f}(y) = f^*(y) - xy，此时信息不丢失，f(x)=f~~f(x) = \tilde{\tilde{f}}

经典力学表述和热力学表述

经典力学中的 Legendre 变换：哈密顿描述 ⇔ 拉格朗日描述

𝐻 = 𝐻(𝒓, 𝒑)
𝐿 = 𝐿(𝒓,𝒗)

𝐻 = 𝒑 · 𝒗 − 𝐿

有 v=r˙=∂pH,F=p˙=−∂rHv = \dot{r} = \partial_p H,\quad F = \dot{p} = -\partial_r H （H 分别对各个变量的偏导）

p=∂vL,p˙=∂rLp = \partial_v L,\quad \dot{p} = \partial_r L （L 分别对各个变量的偏导）

可以验证各自变量的形式保持不变，哈密顿描述和拉格朗日描述二者完全等价

热力学或多变量 Legendre 变换比经典力学表述仅相差一个符号

L~=L~(X1,⋯,yα,⋯)=L(X1,⋯,Xα(X1,⋯,yα,⋯),⋯)−Xαyα\tilde{L} =\tilde{L}(X_1, \cdots, y_\alpha, \cdots) = L(X_1, \cdots, X_\alpha(X_1, \cdots, y_\alpha, \cdots), \cdots) - X_\alpha y_\alpha

即：dL~=y1dX1+y2dX2+⋯−Xαdyα+⋯d\tilde{L} = y_1 dX_1 + y_2 dX_2 + \cdots - X_\alpha dy_\alpha + \cdots

微分1-形式

定义与外微分：
微分1-形式表示为：Ω = y₁dx₁ + y₂dx₂ + … = Σyᵢdxᵢ
其外微分为：dΩ = Σ(∂yᵢ/∂xⱼ)dxⱼ∧dxᵢ，其中外积满足反交换律：dxᵢ∧dxⱼ = -dxⱼ∧dxᵢ
形式性质：
闭形式：如果 dΩ = 0，则Ω是闭的
恰当形式：如果 Ω = df = Σ(∂f/∂xᵢ)dxᵢ，则 Ω 是恰当的，此时 f 称为"势函数"
关键关系：恰当1-形式总是闭的，但闭1-形式不一定是恰当的，这取决于状态空间流形的拓扑性质

用 Maxwell 关系推导热力学关系

一、 Maxwell关系的物理含义与记忆

Maxwell关系不仅仅是数学上的等式变换，其物理含义在于联系了不同热力学过程中的相对改变量。例如：

  1. 绝热过程 vs 等容过程
    绝热过程体积对温度相对改变 vs 等容过程中放热对压强的相对改变：

    (∂V∂T)S=−(∂S∂p)V\left(\frac{\partial V}{\partial T}\right)_S = -\left(\frac{\partial S}{\partial p}\right)_V

    这将绝热膨胀中的体积变化与等容加热引起的压强变化联系了起来。
  2. 等容过程 vs 等温过程
    等容过程压强对温度的变化 vs 等温过程吸热对体积的变化：

    (∂p∂T)V=(∂S∂V)T\left(\frac{\partial p}{\partial T}\right)_V = \left(\frac{\partial S}{\partial V}\right)_T

  3. 等压过程 vs 等温过程
    等压过程体积对温度的变化 vs 等温过程放热对压强的变化：

    (∂V∂T)p=−(∂S∂p)T\left(\frac{\partial V}{\partial T}\right)_p = -\left(\frac{\partial S}{\partial p}\right)_T

记忆方法：
记住特性函数（如内能 UU、自由能 FF 等）的定义及其微分形式（如 dU=TdS−pdVdU=TdS-pdV）即可推导。

能流密度 JJ 与能量密度 uu 的关系：
对所有方向取平均：J(ω)=⟨S⟩k=c4u(ω)J(\omega) = \langle S \rangle_k = \frac{c}{4} u(\omega)

辐射压强 pp：

利用电磁波能动量张量 Tij=−DiEj−HiBj+IijuT_{ij} = -D_i E_j - H_i B_j + I_{ij} u。
压强 p=⟨Tii⟩=−⟨DiEi+HiBi⟩+3up = \langle T_{ii} \rangle = -\langle D_i E_i + H_i B_i \rangle + 3u。
对于各向同性辐射，⟨DiEi⟩=⟨ε0Ei2⟩=13ε0E2=u3\langle D_i E_i \rangle = \langle \varepsilon_0 E_i^2 \rangle = \frac{1}{3} \varepsilon_0 E^2 = \frac{u}{3}。
因此：

p=−23u+u=u3p = -\frac{2}{3}u + u = \frac{u}{3}

结论：仅从力学平衡无法确定能量密度 uu 的具体表达式。

热力学平衡与空腔辐射

热力学平衡条件

设真空中辐射能流密度为 J(ω)J(\omega)，入射到温度为 TT 的物体上。
反射部分：R(T,ω)J(ω)R(T, \omega)J(\omega)
吸收部分：ϵ(T,ω)J(ω)\epsilon(T, \omega)J(\omega)，其中 ϵ=1−R\epsilon = 1 - R 为吸收系数。

物体自身辐射为 Je(T,ω)J_e(T, \omega)。平衡时入射=出射：

J(ω)=RJ(ω)+Je⇒ϵJ(ω)=JeJ(\omega) = R J(\omega) + J_e \Rightarrow \epsilon J(\omega) = J_e

即：

J(ω)=Je(T,ω)ϵ(T,ω)J(\omega) = \frac{J_e(T, \omega)}{\epsilon(T, \omega)}

理想黑体：ϵ≡1\epsilon \equiv 1，此时 J(ω)=Je(T,ω)J(\omega) = J_e(T, \omega)。即平衡热辐射等同于理想黑体辐射。

空腔辐射的普适性
空腔中的热辐射是最简单的处于热力学平衡的热辐射。
普适函数：u(ω,T)u(\omega, T) 是普适函数，与空腔器壁性质（组成、外形）无关。
证明（反证法）：假设两个同温空腔能量密度不同，导致 J1>J2J_1 > J_2。用带通滤波器连接，会有净能流从空腔1流向2，导致温差，从而构造出单一热源热机，违反热力学第二定律。

热力学平衡态下的状态方程推导

基本关系式
内能：U(T,V)=u(T)VU(T, V) = u(T)V
压强：p(T,V)=u(T)3p(T, V) = \frac{u(T)}{3}
能流密度：J(T)=c4u(T)J(T) = \frac{c}{4}u(T)

能量密度 u(T)u(T) 的推导

由热力学基本方程 dF=−SdT−pdVdF = -SdT - pdV。
利用 U=F+TSU = F + TS，对体积求偏导：

(∂U∂V)T=(∂F∂V)T+T(∂S∂V)T\left( \frac{\partial U}{\partial V} \right)_T = \left( \frac{\partial F}{\partial V} \right)_T + T \left( \frac{\partial S}{\partial V} \right)_T

代入已知关系：

u=−p+T(∂p∂T)Vu = -p + T \left( \frac{\partial p}{\partial T} \right)_V

（此处用到Maxwell关系 (∂S∂V)T=(∂p∂T)V\left( \frac{\partial S}{\partial V} \right)_T = \left( \frac{\partial p}{\partial T} \right)_V）
将 p=u/3p = u/3 代入：

u=−u3+T⋅13dudTu = -\frac{u}{3} + T \cdot \frac{1}{3} \frac{du}{dT}

整理得：

43u=T3dudT⇒duu=4dTT\frac{4}{3}u = \frac{T}{3} \frac{du}{dT} \Rightarrow \frac{du}{u} = 4 \frac{dT}{T}

积分得：

u(T)=u0T4u(T) = u_0 T^4

这就是 Stefan-Boltzmann 定律。

热力学函数的计算
内能：U=u0T4VU = u_0 T^4 V

熵 SS：
由 TdS=dU+pdV=d(u0T4V)+u0T43dVTdS = dU + pdV = d(u_0 T^4 V) + \frac{u_0 T^4}{3} dV

TdS=4u0T3VdT+43u0T4dV=Td(43u0T3V)TdS = 4u_0 T^3 V dT + \frac{4}{3} u_0 T^4 dV = T d\left( \frac{4}{3} u_0 T^3 V \right)

S(T,V)=43u0T3VS(T, V) = \frac{4}{3} u_0 T^3 V

（根据热力学第三定律，积分常数为0）

自由能 FF：

F=U−TS=u0T4V−T(43u0T3V)=−13u0T4VF = U - TS = u_0 T^4 V - T \left( \frac{4}{3} u_0 T^3 V \right) = -\frac{1}{3} u_0 T^4 V

Gibbs自由能 GG：

G=F+pV=−13u0T4V+u0T43V=0G = F + pV = -\frac{1}{3} u_0 T^4 V + \frac{u_0 T^4}{3} V = 0

这表明光子数不守恒，化学势 μ=0\mu = 0。

Stefan-Boltzmann常数

J(T)=σT4J(T) = \sigma T^4

其中 σ=π2kB460c2ℏ3≈5.67×10−8W/(m2K4)\sigma = \frac{\pi^2 k_B^4}{60 c^2 \hbar^3} \approx 5.67 \times 10^{-8} W/(m^2 K^4)。
u(T)u(T) 与 J(T)J(T) 的关系：u(T)=4cσT4u(T) = \frac{4}{c} \sigma T^4。

辐射传热

基本公式
物体发射强度：Je=ϵσT4J_e = \epsilon \sigma T^4（发射系数 ϵ\epsilon 等于吸收系数）。
单个平面辐射传热：Q˙=AϵσT4\dot{Q} = A \epsilon \sigma T^4。

两平行平面的辐射传热
设两平面温度为 T1,T2T_1, T_2，吸收系数 ϵ1,ϵ2\epsilon_1, \epsilon_2，反射系数 R1=1−ϵ1,R2=1−ϵ2R_1 = 1-\epsilon_1, R_2 = 1-\epsilon_2。
平面1的总出射 J1J_1 包括自身发射 J1eJ_{1e} 和对平面2辐射的反射 R1J2R_1 J_2：

{J1=J1e+R1J2J2=J2e+R2J1\begin{cases} J_1 = J_{1e} + R_1 J_2 \\ J_2 = J_{2e} + R_2 J_1 \end{cases}

代入 J1e=ϵ1σT14,J2e=ϵ2σT24J_{1e} = \epsilon_1 \sigma T_1^4, J_{2e} = \epsilon_2 \sigma T_2^4，解方程组：

(1−R1−R21)(J1J2)=(ϵ1σT14ϵ2σT24)\begin{pmatrix} 1 & -R_1 \\ -R_2 & 1 \end{pmatrix} \begin{pmatrix} J_1 \\ J_2 \end{pmatrix} = \begin{pmatrix} \epsilon_1 \sigma T_1^4 \\ \epsilon_2 \sigma T_2^4 \end{pmatrix}

解得：

J1−J2=ϵ1ϵ2ϵ1+ϵ2−ϵ1ϵ2σ(T14−T24)J_1 - J_2 = \frac{\epsilon_1 \epsilon_2}{\epsilon_1 + \epsilon_2 - \epsilon_1 \epsilon_2} \sigma (T_1^4 - T_2^4)

故传热速率：

Q˙=A(J1−J2)=Aϵ1ϵ2ϵ1+ϵ2−ϵ1ϵ2σ(T14−T24)\dot{Q} = A(J_1 - J_2) = A \frac{\epsilon_1 \epsilon_2}{\epsilon_1 + \epsilon_2 - \epsilon_1 \epsilon_2} \sigma (T_1^4 - T_2^4)

  • 特殊情况：
    • 黑体 (ϵ1=ϵ2=1\epsilon_1 = \epsilon_2 = 1)：Q˙=Aσ(T14−T24)\dot{Q} = A \sigma (T_1^4 - T_2^4)。
    • 亮体 (ϵ1=ϵ2=ϵ≪1\epsilon_1 = \epsilon_2 = \epsilon \ll 1)：Q˙≈ϵ2Aσ(T14−T24)\dot{Q} \approx \frac{\epsilon}{2} A \sigma (T_1^4 - T_2^4)。

黑体辐射的热流和熵流

熵流密度 JSJ_S
由 S=43u0T3VS = \frac{4}{3} u_0 T^3 V 和 J=c4uJ = \frac{c}{4} u，可得熵流密度：

JS=c3Tu=43σT3J_S = \frac{c}{3T} u = \frac{4}{3} \sigma T^3

面积 AA 上的熵流率：S˙=A43σT3\dot{S} = A \frac{4}{3} \sigma T^3。

物体通过辐射导致的熵变
物体以 Q˙=AσT4\dot{Q} = A \sigma T^4 辐射热量。
若简单地用 Q˙T\frac{\dot{Q}}{T} 计算熵流，结果是 AσT3A \sigma T^3。
实际上，从物体表面热辐射带走的熵流为 S˙rad=43AσT3\dot{S}_{rad} = \frac{4}{3} A \sigma T^3。

  • 解释：物体在辐射过程中，不仅放出热量，还产生了熵。辐射过程是不可逆过程。辐射场携带的熵流大于物体本身的熵减（Q˙T\frac{\dot{Q}}{T}），差值即为熵产生。
