几率法：Boltzmann 统计

基本概念

为了阐明 Boltzmann 统计的定义，需要先了解近独立子系组成的系统，全同和非全同粒子的系统态（多粒子态）和单粒子态

近独立子系组成的系统

考虑系统运动时，可以把系统分成不同的子系统，这些子系统之间相互作用可以忽略不计。于是有总能量

\begin{aligned} \hat{H} = \sum_{i = 1}^N \hat{h}_i (\boldsymbol{r}_i, \boldsymbol{p}_i) + H_I \backsimeq \sum_{i = 1}^N h_i (\boldsymbol{r}_i, \boldsymbol{p}_i) \end{aligned}

子系 hih_i 可以是来自于组成系统的粒子、或者粒子的某个自由度、或者是元激发形式的能量

若考虑 NN 个近独立粒子组成的系统，这些粒子具有相同的运动形式，单粒子的 Hamiltonian 相同，即单组元系统，粒子间没有相互作用，则

\begin{aligned} \hat{H} = \sum_i \hat{h}_i (\hat{\boldsymbol{r}}_i, \hat{\boldsymbol{p}}_i | V, H, \cdots) \end{aligned}

微观描述：系统的波函数∣Ψ(r1,r2,⋯,rN,t)⟩|\Psi(r_1, r_2, \cdots, r_N, t)\rangle

热力学平衡态物理量（特别是能量）不随时间改变 ⇒ 只要考虑物理量（特别是能量）不变的态⇒能量本征态

对于无相互作用的系统，粒子之间没有关系，求系统的本征态/波函数时可以通过分离变量，先求出单粒子的本征态/波函数，然后组合出系统本征态/波函数。

单粒子态和系统态（多粒子态）

单粒子问题：

H^=∑ih^i=∑ih^(ri,pi∣V)\hat{H} = \sum_i \hat{h}_i = \sum_i \hat{h}(\mathbf{r}_i, \mathbf{p}_i|V)

\begin{aligned} \hat{h}|\psi_s\rangle &= \varepsilon_s(V)|\psi_s\rangle \quad \boxed{\text{态描述，V为体积，即外界参数}} \\ &= \varepsilon_l(V)|\psi_{1\alpha}\rangle \quad \boxed{\text{能级描述：}\alpha=1,2,\cdot,\omega_l；\omega_l\text{简并度}} \end{aligned}

且有：

⟨ψs∣ψs′⟩=δss′⟨ψ1α∣ψ1′α′⟩=δll′δαα′\langle\psi_s|\psi_{s'}\rangle = \delta_{ss'} \quad \langle\psi_{1\alpha}|\psi_{1'\alpha'}\rangle = \delta_{ll'}\delta_{\alpha\alpha'}

系统问题：

H^=∑ih^i=∑ih^(ri,pi)\hat{H} = \sum_i \hat{h}_i = \sum_i \hat{h}(\mathbf{r}_i, \mathbf{p}_i)

h^∣ψs⟩=εs∣ψs⟩=εl∣ψlα⟩\hat{h}|\psi_s\rangle = \varepsilon_s|\psi_s\rangle = \varepsilon_l|\psi_{l\alpha}\rangle

H^∣ΨS⟩=ES∣ΨS⟩\hat{H}|\Psi_S\rangle = E_S|\Psi_S\rangle

统计假设中微观态的数目指的是系统态，不是单粒子态。单粒子态只不过是得到系统态的手段。

从单粒子态得到系统态时需要考虑粒子全同性（见后文）

统计物理的基本假设和方法对所有系统都相同，不依赖于组成系统的粒子遵循的微观规律，也不要考虑粒子是否全同，以及具有什么样的全同性。

全同性仅仅体现在从单粒子态构造系统态的方法不一样。

全同和非全同粒子

非全同粒子：系统本征波函数是单粒子态的简单直积

∣ΨS(1,2,⋯,N)⟩=∣ψs1(1)⟩⊗∣ψs2(2)⟩⋯⊗∣ψsN(N)⟩|\Psi_S(1,2,\cdots,N)\rangle = |\psi_{s_1}(1)\rangle \otimes |\psi_{s_2}(2)\rangle \cdots \otimes |\psi_{s_N}(N)\rangle

=∣ψs1(1)ψs2(2)⋯ψsN(N)⟩= |\psi_{s_1}(1)\psi_{s_2}(2)\cdots\psi_{s_N}(N)\rangle

=∣ψl1α1(1)ψl2α2(2)⋯ψlNαN(N)⟩= |\psi_{l_1\alpha_1}(1)\psi_{l_2\alpha_2}(2)\cdots\psi_{l_N\alpha_N}(N)\rangle

H^∣ΨS⟩=ES∣ΨS⟩=∑iεsi∣ΨS⟩=∑iεli∣ΨS⟩\hat{H}|\Psi_S\rangle = E_S|\Psi_S\rangle = \sum_i \varepsilon_{s_i}|\Psi_S\rangle = \sum_i \varepsilon_{l_i}|\Psi_S\rangle

ES=∑iεsi=∑iεliE_S = \sum_i \varepsilon_{s_i} = \sum_i \varepsilon_{l_i}

全同粒子：系统本征波函数是单粒子波函数简单直积后再对称化（波色子）或者反对称化（费米子）

$$|\Psi_S\rangle = \frac{1}{C} \sum_P (\pm)^P |\psi_{s_1}(1)\psi_{s_2}(2)\cdots\psi_{s_N}(N)\rangle \quad \boxed{\text{P: 交换算符，$N!$个}}$$

下面以两个粒子为例讨论全同和非全同粒子的性质以及宏观态分布

设两个单粒子态 ∣ψ1⟩,∣ψ2⟩|\psi_1\rangle, |\psi_2\rangle

三种情况：

  1. 非全同粒子：多粒子态（系统态）无对称性要求

    多粒子态 ∣Ψ1⟩=∣ψ1(1)ψ2(2)⟩∣Ψ2⟩=∣ψ2(1)ψ1(2)⟩|\Psi_1\rangle = |\psi_1(1)\psi_2(2)\rangle \qquad |\Psi_2\rangle = |\psi_2(1)\psi_1(2)\rangle

    当 ∣ψ1⟩=∣ψ2⟩|\psi_1\rangle = |\psi_2\rangle 时，只有一个多粒子态（系统态）；反之，有两个不同的多粒子态

  2. 全同波色子(Boson)：交换对称的多粒子波函数

    ∣Ψ⟩=1C[∣ψ1(1)ψ2(2)⟩+∣ψ1(2)ψ2(1)⟩]=1C[∣ψ1(1)ψ2(2)⟩+∣ψ2(1)ψ1(2)⟩]|\Psi\rangle = \frac{1}{C}[|\psi_1(1)\psi_2(2)\rangle + |\psi_1(2)\psi_2(1)\rangle] = \frac{1}{C}[|\psi_1(1)\psi_2(2)\rangle + |\psi_2(1)\psi_1(2)\rangle]

    不管 ∣ψ1⟩,∣ψ2⟩|\psi_1\rangle , |\psi_2\rangle 是否相同，都只有一个多粒子态

  3. 全同费米子(Fermion)：交换反对称的多粒子波函数

    ∣Ψ⟩=1C[∣ψ1(1)ψ2(2)⟩−∣ψ1(2)ψ2(1)⟩]=1C[∣ψ1(1)ψ2(2)⟩−∣ψ2(1)ψ1(2)⟩]|\Psi\rangle = \frac{1}{C}[|\psi_1(1)\psi_2(2)\rangle - |\psi_1(2)\psi_2(1)\rangle] = \frac{1}{C}[|\psi_1(1)\psi_2(2)\rangle - |\psi_2(1)\psi_1(2)\rangle]

    当 ∣ψ1⟩=∣ψ2⟩|\psi_1\rangle = |\psi_2\rangle 时，此多粒子态不存在（Pauli 不相容原理）

    当 ∣ψ1⟩≠∣ψ2⟩|\psi_1\rangle \neq |\psi_2\rangle 时，一个多粒子态

分布函数：

微观描述：系统态、多粒子波函数 ∣ψ⟩|\psi\rangle
宏观描述：内能密度 u(r)u(r)，粒子数密度 ρ(r)，总能量…
分布函数：每个态/能级上有几个粒子占据 as/ala_s/a_l

从分布函数可以得到宏观描述

ρ(r)=⟨Ψ∣∑i=1Nδ(r−ri)∣Ψ⟩=∑i=1N∣ψsi(r)∣2=∑sas∣ψs(r)∣2\rho(r) = \langle\Psi|\sum_{i=1}^{N}\delta(r-r_i)|\Psi\rangle = \sum_{i=1}^{N}|\psi_{s_i}(r)|^2 = \sum_{s}a_s|\psi_s(r)|^2

求密度时，不关心具体粒子，只要知道每个态上的粒子数as即可

u(r)=⟨Ψ∣∑i=1Nεsiδ(r−ri)∣Ψ⟩=∑i=1Nεsi∣ψsi(r)∣2=∑sasεs∣ψs(r)∣2u(r) = \langle\Psi|\sum_{i=1}^{N}\varepsilon_{s_i}\delta(r-r_i)|\Psi\rangle = \sum_{i=1}^{N}\varepsilon_{s_i}|\psi_{s_i}(r)|^2 = \sum_{s}a_s\varepsilon_s|\psi_s(r)|^2

U=⟨Ψ∣H^∣Ψ⟩=∑i=1Nεsi=∑sεsas=∑lεlalU = \langle\Psi|\hat{H}|\Psi\rangle = \sum_{i=1}^{N}\varepsilon_{s_i} = \sum_{s}\varepsilon_s a_s = \sum_{l}\varepsilon_l a_l

N=∑i1=∑sas=∑lalN = \sum_{i}1 = \sum_{s}a_s = \sum_{l}a_l

不同的分布函数代表不同的宏观态，可以是平衡态，也可以是非平衡态

Boltzmann 统计

等几率假设与微观态计数

等几率假设（基本思想）

对孤立系统，宏观态（用分布 {al}\{a_l\} 表示）出现的几率满足：

p(al)∝Ω(E,N,V,{al})p({a_l}) \propto \Omega(E,N,V,\{a_l\})

其中 Ω\Omega 是该分布对应的系统微观态数目。

非全同粒子系统的微观态数 Ω(E,N,V,{al})\Omega(E,N,V,\{a_l\})

设能级（或组）用 ll 标记，第 ll 组有简并度 ωl\omega_l，该组被占据的粒子数为 ala_l。

  • 粒子数约束

    N=∑i1=∑sas=∑lalN=\sum_i 1=\sum_s a_s=\sum_l a_l

  • 能量约束

    E=∑iεsi=∑sasεs=∑lalεlE=\sum_i \varepsilon_{s_i}=\sum_s a_s\varepsilon_s=\sum_l a_l\varepsilon_l

对非全同粒子，先做“把 NN 个粒子分到各组”的组合计数，再乘上各组内部简并态的选取数：

Ω(E,N,V,al)=(Na1)(N−a1a2)⋯(N−a1−⋯−al−1al)⋯×ω1a1ω2a2⋯ωlal⋯\Omega(E,N,V,{a_l}) = \binom{N}{a_1}\binom{N-a_1}{a_2}\cdots \binom{N-a_1-\cdots-a_{l-1}}{a_l}\cdots \times \omega_1^{a_1}\omega_2^{a_2}\cdots\omega_l^{a_l}\cdots

把连乘的组合数化简（经典结果）：

Ω=N!a1!a2!⋯al!⋯∏lωlal=N!∏lal!∏lωlal\Omega= \frac{N!}{a_1!a_2!\cdots a_l!\cdots} \prod_l \omega_l^{a_l}= \frac{N!}{\prod_l a_l!}\prod_l \omega_l^{a_l}

热力学平衡态 ⇔ 最可几态（最大 Ω\Omega 的分布）

取对数并用 Stirling 公式：

lnΩ=lnN!−∑llnal!+∑lallnωl\ln\Omega = \ln N!-\sum_l \ln a_l!+\sum_l a_l\ln\omega_l

当 ala_l 很大时用 Stirling 公式：

lnn!≃nlnn−n\ln n!\simeq n\ln n-n

于是

lnΩ≃(NlnN−N)−∑l(allnal−al)+∑lallnωl\ln\Omega \simeq (N\ln N-N)-\sum_l(a_l\ln a_l-a_l)+\sum_l a_l\ln\omega_l

整理（用 ∑lal=N\sum_l a_l=N）：

lnΩ=(∑lal)ln(∑lal)−∑lal(lnal−lnωl)\ln\Omega = \Big(\sum_l a_l\Big)\ln\Big(\sum_l a_l\Big) -\sum_l a_l(\ln a_l-\ln\omega_l)

变分：最大化 lnΩ\ln\Omega：

对分布做微扰 al→al+δala_l\to a_l+\delta a_l。用

(xlnx)′=lnx+1(x\ln x)'=\ln x+1

可得

0=δlnΩ=∑l[lnN−ln(alωl)]δal0=\delta\ln\Omega= \sum_l\Big[\ln N-\ln\Big(\frac{a_l}{\omega_l}\Big)\Big]\delta a_l

但 δal\delta a_l 不独立，因为还要满足约束：

0=δE=∑lεl,δal,0=δN=∑lδal0=\delta E=\sum_l \varepsilon_l,\delta a_l,\qquad 0=\delta N=\sum_l \delta a_l

拉格朗日乘子法（使 δal\delta a_l 视作独立）：

构造：

0=δlnΩ−β,δE−α~,δN0=\delta\ln\Omega-\beta,\delta E-\tilde\alpha,\delta N

其中 α~,β\tilde\alpha,\beta 是待定的 Lagrange 乘子。代入上式：

0=∑l[lnN−ln(alωl)−α~−βεl]δal0=\sum_l\Big[\ln N-\ln\Big(\frac{a_l}{\omega_l}\Big)-\tilde\alpha-\beta\varepsilon_l\Big]\delta a_l

由于 δal\delta a_l 可视为独立，括号必须逐项为 0：

ln(alωl)=lnN−α~−βεl\ln\Big(\frac{a_l}{\omega_l}\Big)=\ln N-\tilde\alpha-\beta\varepsilon_l

指数化得到最可几态（平衡态）分布：

a¯l=Nωle−α~−βεl\bar a_l=N\omega_l e^{-\tilde\alpha-\beta\varepsilon_l}

常把 α~\tilde\alpha 吸收到 α\alpha 中：

α=α~−lnN⟺α~=α+lnN\alpha=\tilde\alpha-\ln N \quad\Longleftrightarrow\quad \tilde\alpha=\alpha+\ln N

于是

$$ \boxed{\bar a_l=\omega_l e^{-\alpha-\beta\varepsilon_l}} $$ 并且对单粒子微观态 $s$（无简并度因子写法）： $$ \boxed{\bar a_s=e^{-\alpha-\beta\varepsilon_s}} $$

Boltzmann 分布与乘子物理意义

由分布：

a¯l=ωle−α−βεl\bar a_l=\omega_l e^{-\alpha-\beta\varepsilon_l}

配合约束方程：

E=∑la¯lεl=∑lεlωle−α−βεl,N=∑la¯l=∑lωle−α−βεlE=\sum_l \bar a_l\varepsilon_l=\sum_l \varepsilon_l\omega_l e^{-\alpha-\beta\varepsilon_l}, \qquad N=\sum_l \bar a_l=\sum_l \omega_l e^{-\alpha-\beta\varepsilon_l}

直接解出 α,β\alpha,\beta 通常很麻烦，因此引入特性函数（配分函数）与 Legendre 变换来简化。

乘子与态密度导数关系

由于平衡态满足

0=δlnΩ−βδE−α~δN0=\delta\ln\Omega-\beta\delta E-\tilde\alpha\delta N

可推出

β=(∂lnΩ∂E)N,α~=(∂lnΩ∂N)E\beta=\left(\frac{\partial\ln\Omega}{\partial E}\right)_N, \qquad \tilde\alpha=\left(\frac{\partial\ln\Omega}{\partial N}\right)_E

并写成微分形式：

dlnΩ=β,dE+α~,dNd\ln\Omega=\beta,dE+\tilde\alpha,dN

Legendre 变换：从 (E,N,V)(E,N,V) 到 (β,N,V)(\beta,N,V)

定义：

lnZ(β,N,V)=lnΩ−βE\ln Z(\beta,N,V)=\ln\Omega-\beta E

微分：

dlnZ=d(lnΩ−βE)=−E,dβ+α~,dNd\ln Z=d(\ln\Omega-\beta E)= -E,d\beta+\tilde\alpha,dN

这使得平衡态计算从“能量确定的孤立系统”转为“温度确定的封闭系统”，在求平衡态物理量时等价。

配分函数 zz 与系统配分函数 ZZ

由

a¯l=ωle−α−βεl\bar a_l=\omega_l e^{-\alpha-\beta\varepsilon_l}

定义单粒子配分函数：

$$ \boxed{z(\beta,V)=\sum_l \omega_l e^{-\beta\varepsilon_l} =\sum_s e^{-\beta\varepsilon_s}} $$

由粒子数约束：

N=∑la¯l=∑lωle−α−βεl=e−α∑lωle−βεl=e−αzN=\sum_l \bar a_l=\sum_l \omega_l e^{-\alpha-\beta\varepsilon_l} =e^{-\alpha}\sum_l \omega_l e^{-\beta\varepsilon_l} =e^{-\alpha}z

所以

e−α=Nz,α=−lnNz=−lnN+lnze^{-\alpha}=\frac{N}{z},\qquad \alpha=-\ln\frac{N}{z}=-\ln N+\ln z

从而

α~=α+lnN=lnz\tilde\alpha=\alpha+\ln N=\ln z

进一步可推出系统特性函数：

lnZ=lnΩ−βE\ln Z=\ln\Omega-\beta E

代入并化简（PDF 中完整代数推导）最终得到

$$ \boxed{\ln Z = N\ln z} \quad\Longleftrightarrow\quad \boxed{Z=z^N} $$

因此：

  • zz：单粒子配分函数
  • ZZ：系统配分函数（在这里得到 Z=zNZ=z^N）

物理量（内能、压强、熵）

后续计算中常把 a¯l,a¯s\bar a_l,\bar a_s 直接写作 al,asa_l,a_s。

内能 UU（= 最可几态能量）

U=E=∑lalεl=∑lωle−α−βεlεl=e−α∑lεlωle−βεlU=E=\sum_l a_l\varepsilon_l =\sum_l \omega_l e^{-\alpha-\beta\varepsilon_l}\varepsilon_l =e^{-\alpha}\sum_l \varepsilon_l\omega_l e^{-\beta\varepsilon_l}

用 e−α=N/ze^{-\alpha}=N/z：

U=Nz∑lεlωle−βεlU=\frac{N}{z}\sum_l \varepsilon_l\omega_l e^{-\beta\varepsilon_l}

把 εle−βεl\varepsilon_l e^{-\beta\varepsilon_l} 写成对 β\beta 的导数：

−∂∂βe−βεl=εle−βεl-\frac{\partial}{\partial\beta}e^{-\beta\varepsilon_l} =\varepsilon_l e^{-\beta\varepsilon_l}

于是

U=−Nz(∂∂β∑lωle−βεl)V=−Nz(∂z∂β)VU=-\frac{N}{z}\left(\frac{\partial}{\partial\beta}\sum_l \omega_l e^{-\beta\varepsilon_l}\right)_V =-\frac{N}{z}\left(\frac{\partial z}{\partial\beta}\right)_V

U=−N(∂lnz∂β)V=−(∂lnZ∂β)N,VU=-N\left(\frac{\partial\ln z}{\partial\beta}\right)_V = -\left(\frac{\partial\ln Z}{\partial\beta}\right)_{N,V}

又因为

β=1kBT\beta=\frac{1}{k_BT}

可改写成对 TT 的形式：

U=NkBT2(∂lnz∂T)V=kBT2(∂lnZ∂T)N,VU=Nk_BT^2\left(\frac{\partial\ln z}{\partial T}\right)_V = k_BT^2\left(\frac{\partial\ln Z}{\partial T}\right)_{N,V}

压强 pp（由能级对体积的依赖）

从内能变分：

δU=δE¯=∑lεl(V),δal+∑lal,δεl(V)\delta U=\delta \bar E =\sum_l \varepsilon_l(V),\delta a_l+\sum_l a_l,\delta\varepsilon_l(V)

解释：

  • 体积不变导致的内能变化 ⇒\Rightarrow 吸热
  • 体积改变导致的内能变化 ⇒\Rightarrow 做功

压强定义为：

p=−∑lal(∂εl∂V)p=-\sum_l a_l\left(\frac{\partial\varepsilon_l}{\partial V}\right)

代入 al=ωle−α−βεla_l=\omega_l e^{-\alpha-\beta\varepsilon_l}：

p=−∑lωle−α−βεl(∂εl∂V)p=-\sum_l \omega_l e^{-\alpha-\beta\varepsilon_l}\left(\frac{\partial\varepsilon_l}{\partial V}\right)

转写为对 zz 的导数（关键步骤：把 ∂Ve−βεl\partial_V e^{-\beta\varepsilon_l} 提出来）：

p=Nz1β(∂∂V∑lωle−βεl)β=Nβz(∂z∂V)βp=\frac{N}{z}\frac{1}{\beta}\left(\frac{\partial}{\partial V}\sum_l \omega_l e^{-\beta\varepsilon_l}\right)_\beta =\frac{N}{\beta z}\left(\frac{\partial z}{\partial V}\right)_\beta

$$ \boxed{p=\frac{N}{\beta}\left(\frac{\partial\ln z}{\partial V}\right)_\beta =\frac{1}{\beta}\left(\frac{\partial\ln Z}{\partial V}\right)_{\beta,N}} $$

用 β=1/(kBT)\beta=1/(k_BT)：
p=NkBT(∂lnz∂V)T=kBT(∂lnZ∂V)T,N p=Nk_BT\left(\frac{\partial\ln z}{\partial V}\right)_T = k_BT\left(\frac{\partial\ln Z}{\partial V}\right)_{T,N}

并与热力学第一定律一致：

dU=dQ−dW=dQ−p,dVdU=dQ-dW=dQ-p,dV

熵 SS：证明 βdQ\beta dQ 是全微分

从

δU=∑lεl,δal+∑lal,δεl\delta U=\sum_l \varepsilon_l,\delta a_l+\sum_l a_l,\delta\varepsilon_l

识别热量部分：

dQ=∑lεl,daldQ=\sum_l \varepsilon_l,da_l

计算：

βdQ=β∑lεl,dal=∑l[d(βεlal)−al,d(βεl)]\beta dQ=\beta\sum_l \varepsilon_l,da_l =\sum_l\big[d(\beta\varepsilon_l a_l)-a_l,d(\beta\varepsilon_l)\big]

=d(∑lβεlal)−∑lal,d(βεl)=d(βU)−∑lal,d(βεl)= d\Big(\sum_l\beta\varepsilon_l a_l\Big)-\sum_l a_l,d(\beta\varepsilon_l) = d(\beta U)-\sum_l a_l,d(\beta\varepsilon_l)

用 al=e−αωle−βεla_l=e^{-\alpha}\omega_l e^{-\beta\varepsilon_l}：

−∑lal,d(βεl)=−e−α∑lωle−βεld(βεl)=Nz∑lωl,d(e−βεl)-\sum_l a_l,d(\beta\varepsilon_l) =-e^{-\alpha}\sum_l \omega_l e^{-\beta\varepsilon_l}d(\beta\varepsilon_l) = \frac{N}{z}\sum_l \omega_l,d\big(e^{-\beta\varepsilon_l}\big)

=Nz,d(∑lωle−βεl)=Nz,dz=d(Nlnz)=\frac{N}{z},d\Big(\sum_l\omega_l e^{-\beta\varepsilon_l}\Big) =\frac{N}{z},dz = d(N\ln z)

所以：

βdQ=d(βU+Nlnz)\beta dQ=d(\beta U+N\ln z)

它是全微分。类比热力学

dQT=dS\frac{dQ}{T}=dS

可得

β=1kBT\beta=\frac{1}{k_BT}

并得到熵：

βdQ=d(SkB)=d(UkBT+Nlnz)\beta dQ=d\left(\frac{S}{k_B}\right) =d\left(\frac{U}{k_BT}+N\ln z\right)

准经典近似

单粒子量子力学描述

  • 量子态：量子力学中，无法同时确定位置 $ \mathbf{r} $ 和动量 $ \mathbf{p} $，由测不准原理决定。
  • 相空间：相空间中非常靠近的点无法区分，一个量子态占据相空间体积 $ \Delta \mathbf{r} \Delta \mathbf{p} = h $。

推导过程：

  1. 假设相空间中的点由 $ \mathbf{r} $ 和 $ \mathbf{p} $ 两个坐标表示。由于量子力学的测不准原理，不能同时确定 $ \mathbf{r} $ 和 $ \mathbf{p} $。
  2. 一个量子态占据的相空间体积为 $ \Delta \mathbf{r} \Delta \mathbf{p} = h $，其中 $ h $ 为普朗克常数。

由此，单粒子态密度 $ f(\mathbf{r}, \mathbf{p}) $ 可以表示为：

∫f(ri,pi)=1hd∫drdpf(r,p)\int f(\mathbf{r}_i, \mathbf{p}_i) = \frac{1}{h^d} \int d\mathbf{r} d\mathbf{p} f(\mathbf{r}, \mathbf{p})

准经典近似与配分函数

推导过程：

  • 配分函数 $ z $ 定义为系统所有能量状态的求和：

z=∑se−βϵsz = \sum_s e^{-\beta \epsilon_s}

  • 对于准经典近似，可以将量子态的求和转换为积分形式：

zc=∫drdphde−βh(r,p)z_c = \int \frac{d\mathbf{r} d\mathbf{p}}{h^d} e^{-\beta h(\mathbf{r}, \mathbf{p})}

其中，$ \beta = \frac{1}{k_B T} $ 为倒温度。

进一步推导：

z=1h3∫Boxdr∫dpe−βp22mz = \frac{1}{h^3} \int_{\text{Box}} d\mathbf{r} \int d\mathbf{p} \, e^{-\beta \frac{p^2}{2m}}

通过高斯积分得到：

zc=V(2πmkBT/h2)3/2z_c = \frac{V}{(2 \pi m k_B T / h^2)^{3/2}}
