弱简并理想气体

强简并与弱简并气体的比较：

单粒子基态能量 ϵg=0\epsilon_g = 0

as=1eβϵs/λ±1a_s = \frac{1}{e^{\beta \epsilon_s} / \lambda \pm 1}

若 λ=e−α=eβμ≪1\lambda = e^{-\alpha} = e^{\beta \mu} \ll 1：弱简并，全同性不重要，Boltzmann 分布：

强弱简并判据

对于 Boson：ϵs−μ≥0⇒μ≤0\epsilon_s - \mu \geq 0 \Rightarrow \mu \leq 0

Fermionic 系统的能量和压强比经典系统大：

  • 由于 Pauli 不相容原理，Fermion 更倾向于占据高能的单粒子态，导致能量和压强增大。
  • 等价于有个统计上的排斥作用。

Bosonic 系统的能量和压强比经典系统小：

  • Boson 更加抱团一点，都倾向于呆在低能的单粒子态上。
  • 等价于有个统计上的吸引作用。

BEC 相变类型

BEC的相变通常认为是一阶相变，其中热容CVC_V连续但其导数不连续。Bose-Einstein凝聚体的正常相和凝聚相混合在一起，不像一般系统那样有明显的界面区分。

等温情况下体积变动导致的 BEC

  • 当体积大于VcV_c时，粒子主要占据激发态。

6.13 强简并 Fermi 气体

判据

  • 强简并条件：

    λ=eβμ≫1⇒μ/(kBT)≫1⇒μ≫kBT\lambda = e^{\beta \mu} \gg 1 \Rightarrow \mu / (k_B T) \gg 1 \Rightarrow \mu \gg k_B T

  • 弱简并条件：

    λ∼y=12(ℏ22πmkBT)3/2NV≪1\lambda \sim y = \frac{1}{2} \left( \frac{\hbar^2}{2 \pi m k_B T} \right)^{3/2} \frac{N}{V} \ll 1

系统的基态

全同Fermion系统的波函数是单粒子的Slater行列式。每个粒子只能占据一个单粒子态，这就导致了Pauli不相容原理的出现。Fermion系统通过Pauli不相容原理把粒子逐个填充到单粒子态上，最终形成Fermi球。

Fermi面是占据态与非占据态的分界面，其对应的能量被称为Fermi能量。

系统的低能激发态

Fermi面下的占据态被激发到Fermi面以上的空态。由于Pauli不相容原理，系统的基态和低能激发态反映的是Fermi面及其附近态的性质。

低能激发能量：

ΔEex=E−Eg=ϵ2−ϵ1(ϵ2∼ϵF)\Delta E_{\text{ex}} = E - E_g = \epsilon_2 - \epsilon_1 \quad (\epsilon_2 \sim \epsilon_F)

  • 粒子型激发能量：ϵp≥0\epsilon_p \geq 0
  • 空穴型激发能量：ϵh≥0\epsilon_h \geq 0

系统的低能激发态能量可以表示为粒子型激发的能量和空穴型激发的能量之和。

Fermi 面

Fermi面在强简并Fermion体系中扮演着非常重要的角色。定义了一系列与Fermi面相关的物理量：

  • Fermi能量 ϵF\epsilon_F
  • Fermi温度 TF,kBTF=ϵFT_F, \quad k_B T_F = \epsilon_F
  • Fermi动量 pF,ϵF=pF22m,pF=2mϵFp_F, \quad \epsilon_F = \frac{p_F^2}{2m}, \quad p_F = \sqrt{2m \epsilon_F}
  • Fermi波矢 kF=pFℏ=2mϵFℏk_F = \frac{p_F}{\hbar} = \frac{\sqrt{2m \epsilon_F}}{\hbar}
  • Fermi速度 vF=pFm=ℏkFmv_F = \frac{p_F}{m} = \frac{\hbar k_F}{m}

在金属中的电子气是一个典型的强简并Fermion体系，其特性如：

  • TF∼104−105KT_F \sim 10^4 - 10^5 \, \text{K}
  • vF∼106m/sv_F \sim 10^6 \, \text{m/s}

T=0KT = 0 \, \text{K}时的Fermi分布

as=1eβ(ϵs−μ)+1=f(ϵs,T=0)a_s = \frac{1}{e^{\beta (\epsilon_s - \mu)} + 1} = f(\epsilon_s, T = 0)

  • 当能量低于化学势μ\mu时，单粒子态被完全占据；当能量高于μ\mu时，单粒子态为空。

在T=0T = 0 K时，化学势与Fermi能量相同：

分数统计

二维系统中存在介乎玻色和费米子之间的“粒子”——anyons。其波函数满足：

  • Boson：θ=2nπ,ψ(r1,r2)=ψ(r2,r1)\theta = 2n\pi, \, \psi(\mathbf{r_1}, \mathbf{r_2}) = \psi(\mathbf{r_2}, \mathbf{r_1})
