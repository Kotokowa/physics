广义系综理论

广延量与强度量
广延量：X1,X2,⋯X_1, X_2, \cdots
强度量：y1,y2,⋯y_1, y_2, \cdots
熵的表达式：S=S(X1,X2,⋯)S = S(X_1, X_2, \cdots)，微分形式：dS=y1dX1+y2dX2+⋯dS = y_1 dX_1 + y_2 dX_2 + \cdots
微正则系综
定义：系统 + 环境
特点：系统和环境的广延量总量守恒，强度量相同
态矢：∣T⟩=∣s⊗R⟩|T\rangle = |s \otimes R\rangle
广延量算符作用：X^i∣T⟩=X^i∣s⊗R⟩=[Xi(s)+XiR(R)]∣s⊗R⟩\hat{X}_i |T\rangle = \hat{X}_i |s \otimes R\rangle = [X_i(s) + X_i^R(R)] |s \otimes R\rangle
总熵：ST({Xi+XiR})=kBlnΩT({Xi+XiR})S_T(\{X_i + X_i^R\}) = k_B \ln \Omega_T(\{X_i + X_i^R\})
概率分布：

ps=∑∣R⟩1ΩT=ΩR({XiR})ΩT({XiT})=1ΩTeSR({XiT−Xi(s)})/kBp_s = \sum_{|R\rangle} \frac{1}{\Omega_T} = \frac{\Omega_R(\{X_i^R\})}{\Omega_T(\{X_i^T\})} = \frac{1}{\Omega_T} e^{S_R(\{X_i^T - X_i(s)\})/k_B}

展开形式：

ps=1ΩTeSR[{X‾iR−(Xi(s)−X‾i)}]/kB=1ΩTexp{S‾RkB−Xi(s)−X‾ikB(∂S‾R∂X‾iR)+⋯}p_s = \frac{1}{\Omega_T} e^{S_R[\{\overline{X}_i^R - (X_i(s) - \overline{X}_i) \}]/k_B} = \frac{1}{\Omega_T} \exp\left\{ \frac{\overline{S}_R}{k_B} - \frac{X_i(s) - \overline{X}_i}{k_B} \left( \frac{\partial \overline{S}_R}{\partial \overline{X}_i^R} \right) + \cdots \right\}

最终形式：

ps=1ΩTeS‾R/kB−∑iyi(Xi−X‾i)/kB=1Ξe−∑iyiXi(s)/kBp_s = \frac{1}{\Omega_T} e^{\overline{S}_R/k_B - \sum_i y_i (X_i - \overline{X}_i)/k_B} = \frac{1}{\Xi} e^{-\sum_i y_i X_i(s)/k_B}

广义配分函数：Ξ(y1,y2,⋯)=∑se−∑iyiXi(s)/kB\Xi(y_1, y_2, \cdots) = \sum_s e^{-\sum_i y_i X_i(s)/k_B}

系综类型

正则系综（NTV ensemble）
密度矩阵：ρ=1Zexp{−H^kBT}\rho = \frac{1}{Z} \exp\left\{ -\frac{\hat{H}}{k_B T} \right\}
巨正则系综（μ\muTV ensemble）
密度矩阵：ρ=1Ξexp{−H−μN^kBT}\rho = \frac{1}{\Xi} \exp\left\{ -\frac{\mathcal{H} - \mu \hat{N}}{k_B T} \right\}
等温等压系综（Isothermal isobaric ensemble，NTp ensemble）
用途：研究化学反应
密度矩阵：ρ=1Δexp{−H+pVkBT}\rho = \frac{1}{\Delta} \exp\left\{ -\frac{\mathcal{H} + pV}{k_B T} \right\}
配分函数：Δ=Δ(N,T,p)=exp{−G(N,T,p)/kBT}\Delta = \Delta(N, T, p) = \exp\{-G(N, T, p)/k_B T\}

延量的平均值与涨落

平均值

X‾l=∑sXl(s)ps=1Ξ∑sXl(s)e−∑iyiXi(s)/kB\overline{X}_l = \sum_s X_l(s) p_s = \frac{1}{\Xi} \sum_s X_l(s) e^{-\sum_i y_i X_i(s)/k_B}

通过配分函数求导：

X‾l=−kB1Ξ∂Ξ∂yl=−kB(∂lnΞ∂yl){yi≠l}\overline{X}_l = -k_B \frac{1}{\Xi} \frac{\partial \Xi}{\partial y_l} = -k_B \left( \frac{\partial \ln \Xi}{\partial y_l} \right)_{\{y_{i \neq l}\}}

二阶矩

XlXm‾=1Ξ∑sXl(s)Xm(s)e∑iyiXi(s)/kB=kB21Ξ∂2Ξ∂yl∂ym\overline{X_l X_m} = \frac{1}{\Xi} \sum_s X_l(s) X_m(s) e^{\sum_i y_i X_i(s)/k_B} = k_B^2 \frac{1}{\Xi} \frac{\partial^2 \Xi}{\partial y_l \partial y_m}

展开形式：

XlXm‾=kB2∂2lnΞ∂yl∂ym+kB2(∂lnΞ∂yl)(∂lnΞ∂ym)=kB2∂2lnΞ∂yl∂ym+Xl‾Xm‾\overline{X_l X_m} = k_B^2 \frac{\partial^2 \ln \Xi}{\partial y_l \partial y_m} + k_B^2 \left( \frac{\partial \ln \Xi}{\partial y_l} \right) \left( \frac{\partial \ln \Xi}{\partial y_m} \right) = k_B^2 \frac{\partial^2 \ln \Xi}{\partial y_l \partial y_m} + \overline{X_l} \overline{X_m}

涨落

ΔXlΔXm‾=(Xl−Xl‾)(Xm−Xm‾)‾=XlXm‾−Xl‾Xm‾=kB2∂2lnΞ∂yl∂ym\overline{\Delta X_l \Delta X_m} = \overline{(X_l - \overline{X_l})(X_m - \overline{X_m})} = \overline{X_l X_m} - \overline{X_l} \overline{X_m} = k_B^2 \frac{\partial^2 \ln \Xi}{\partial y_l \partial y_m}

另一种形式：

ΔXlΔXm‾=−kB(∂Xl∂ym){yi≠m}=−kB(∂Xm∂yl){yi≠l}\overline{\Delta X_l \Delta X_m} = -k_B \left( \frac{\partial X_l}{\partial y_m} \right)_{\{y_{i \neq m}\}} = -k_B \left( \frac{\partial X_m}{\partial y_l} \right)_{\{y_{i \neq l}\}}

强度量的涨落

强度量与广延量的关系
yi=yi(X1,X2,⋯)y_i = y_i(X_1, X_2, \cdots)
强度量涨落

Δyi=yi({Xl})−yi({Xl‾})=yi({Xl‾+ΔXl})−yi({Xl‾})=∑l(∂yi∂Xl)ΔXl\Delta y_i = y_i(\{X_l\}) - y_i(\{\overline{X_l}\}) = y_i(\{\overline{X_l} + \Delta X_l\}) - y_i(\{\overline{X_l}\}) = \sum_l \left( \frac{\partial y_i}{\partial X_l} \right) \Delta X_l

强度量与广延量的关联涨落

ΔyiΔXj‾=∑l(∂yi∂Xl)ΔXlΔXj‾=∑l(∂yi∂Xl)ΔXlΔXj‾\overline{\Delta y_i \Delta X_j} = \sum_l \overline{\left( \frac{\partial y_i}{\partial X_l} \right) \Delta X_l \Delta X_j} = \sum_l \left( \frac{\partial y_i}{\partial X_l} \right) \overline{\Delta X_l \Delta X_j}

代入涨落公式：

ΔyiΔXj‾=−∑l(∂yi∂Xl){Xm≠l}kB(∂Xl∂yj){yk≠j}=−kB(∂yi∂yj){yk≠j}=−kBδij\overline{\Delta y_i \Delta X_j} = -\sum_l \left( \frac{\partial y_i}{\partial X_l} \right)_{\{X_{m \neq l}\}} k_B \left( \frac{\partial X_l}{\partial y_j} \right)_{\{y_{k \neq j}\}} = -k_B \left( \frac{\partial y_i}{\partial y_j} \right)_{\{y_{k \neq j}\}} = -k_B \delta_{ij}

强度量自身的涨落

ΔyiΔyj‾=Δyi∑m(∂yj∂Xm)ΔXm‾=∑m(∂yj∂Xm)ΔyiΔXm‾\overline{\Delta y_i \Delta y_j} = \overline{\Delta y_i \sum_m \left( \frac{\partial y_j}{\partial X_m} \right) \Delta X_m} = \sum_m \left( \frac{\partial y_j}{\partial X_m} \right) \overline{\Delta y_i \Delta X_m}

代入关联涨落：

ΔyiΔyj‾=−kB∑m(∂yj∂Xm)δim=−kB(∂yj∂Xi)Xk≠i=−kB(∂yi∂Xj)Xk≠j\overline{\Delta y_i \Delta y_j} = -k_B \sum_m \left( \frac{\partial y_j}{\partial X_m} \right) \delta_{im} = -k_B \left( \frac{\partial y_j}{\partial X_i} \right)_{X_{k \neq i}} = -k_B \left( \frac{\partial y_i}{\partial X_j} \right)_{X_{k \neq j}}

其它物理量的涨落与关联

物理量的涨落
设 f=f({Xl})f = f(\{X_l\})，f‾=f({Xl‾})=f({yl‾})\overline{f} = f(\{\overline{X_l}\}) = f(\{\overline{y_l}\})
涨落：

Δf=f({Xl})−f‾=f({Xl‾+ΔXl})−f‾=∑l(∂f∂Xl)ΔXl\Delta f = f(\{X_l\}) - \overline{f} = f(\{\overline{X_l} + \Delta X_l\}) - \overline{f} = \sum_l \left( \frac{\partial f}{\partial X_l} \right) \Delta X_l

通过强度量表示：

Δf=∑l(∂f∂Xl)∑m(∂Xl∂ym)Δym=∑m[∑l(∂f∂Xl)(∂Xl∂ym)]Δym=∑m(∂f∂ym)Δym\Delta f = \sum_l \left( \frac{\partial f}{\partial X_l} \right) \sum_m \left( \frac{\partial X_l}{\partial y_m} \right) \Delta y_m = \sum_m \left[ \sum_l \left( \frac{\partial f}{\partial X_l} \right) \left( \frac{\partial X_l}{\partial y_m} \right) \right] \Delta y_m = \sum_m \left( \frac{\partial f}{\partial y_m} \right) \Delta y_m

同理，Δg=∑l(∂g∂Xl)ΔXl=∑m(∂g∂ym)Δym\Delta g = \sum_l \left( \frac{\partial g}{\partial X_l} \right) \Delta X_l = \sum_m \left( \frac{\partial g}{\partial y_m} \right) \Delta y_m
物理量的关联涨落

ΔfΔg‾=∑lm(∂f∂Xl)(∂g∂ym)ΔXlΔym‾=−∑lm(∂f∂Xl)(∂g∂ym)kBδlm\overline{\Delta f \Delta g} = \sum_{lm} \left( \frac{\partial f}{\partial X_l} \right) \left( \frac{\partial g}{\partial y_m} \right) \overline{\Delta X_l \Delta y_m} = -\sum_{lm} \left( \frac{\partial f}{\partial X_l} \right) \left( \frac{\partial g}{\partial y_m} \right) k_B \delta_{lm}

简化：

ΔfΔg‾=−kB∑l(∂f∂Xl)(∂g∂yl)=−kB∑l(∂f∂yl)(∂g∂Xl)\overline{\Delta f \Delta g} = -k_B \sum_l \left( \frac{\partial f}{\partial X_l} \right) \left( \frac{\partial g}{\partial y_l} \right) = -k_B \sum_l \left( \frac{\partial f}{\partial y_l} \right) \left( \frac{\partial g}{\partial X_l} \right)

进一步展开：

ΔfΔg‾=−kB∑l∑m(∂f∂Xl)(∂g∂Xm)(∂Xl∂yl)=−kB∑l∑m(∂f∂yl)(∂g∂ym)(∂ym∂Xl)\overline{\Delta f \Delta g} = -k_B \sum_l \sum_m \left( \frac{\partial f}{\partial X_l} \right) \left( \frac{\partial g}{\partial X_m} \right) \left( \frac{\partial X_l}{\partial y_l} \right) = -k_B \sum_l \sum_m \left( \frac{\partial f}{\partial y_l} \right) \left( \frac{\partial g}{\partial y_m} \right) \left( \frac{\partial y_m}{\partial X_l} \right)

广义系综理论几个例子

  1. 熵的微分与系综结果
    熵的微分形式：

    dS=1TdU+pTdV−μTdNdS = \frac{1}{T}dU + \frac{p}{T}dV - \frac{\mu}{T}dN

    正则系综（保持 T,N,VT, N, V 不变）
    能量涨落：

    ΔE2‾=−kB(∂U∂[1/T])NV=kBT2CV\overline{\Delta E^2} = -k_B \left( \frac{\partial U}{\partial [1/T]} \right)_{NV} = k_B T^2 C_V

    巨正则系综（保持 T,μ/T,VT, \mu/T, V 不变）
    粒子数涨落：

    ΔN2‾=−kB(∂N∂[−μ/T])T,V=kBT(∂N∂μ)TV=kBTN2κTV\overline{\Delta N^2} = -k_B \left( \frac{\partial N}{\partial [-\mu/T]} \right)_{T,V} = k_B T \left( \frac{\partial N}{\partial \mu} \right)_{TV} = \frac{k_B T N^2 \kappa_T}{V}

    能量涨落：

    ΔE2‾=−kB(∂U∂[1/T])μ/T,V=kBT2CV+(∂U∂N)TV2ΔN2‾\overline{\Delta E^2} = -k_B \left( \frac{\partial U}{\partial [1/T]} \right)_{\mu/T,V} = k_B T^2 C_V + \left( \frac{\partial U}{\partial N} \right)_{TV}^2 \overline{\Delta N^2}

    能量与粒子数关联涨落：

    ΔEΔN‾=−kB(∂U∂[−μ/T])TV=kBT(∂U∂μ)TV=(∂U∂N)TVΔN2‾\overline{\Delta E \Delta N} = -k_B \left( \frac{\partial U}{\partial [-\mu/T]} \right)_{TV} = k_B T \left( \frac{\partial U}{\partial \mu} \right)_{TV} = \left( \frac{\partial U}{\partial N} \right)_{TV} \overline{\Delta N^2}

  2. 广义系综（保持 T,p/T,NT, p/T, N 不变）
    体积涨落：

    ΔV2‾=−kB(∂V∂[p/T])1/T,N=−kBT(∂V∂p)TN=kBTV−1V(∂V∂p)TN=kBTVκT\overline{\Delta V^2} = -k_B \left( \frac{\partial V}{\partial [p/T]} \right)_{1/T,N} = -k_B T \left( \frac{\partial V}{\partial p} \right)_{TN} = k_B T V \frac{-1}{V} \left( \frac{\partial V}{\partial p} \right)_{TN} = k_B T V \kappa_T

    体积与粒子数涨落的关联：
    由 V=NvV = N v（v=V/Nv = V/N 为摩尔体积），得 ΔV2‾=ΔN2‾v2=kBTN2VκT⋅V2N2=kBTVκT\overline{\Delta V^2} = \overline{\Delta N^2} v^2 = \frac{k_B T N^2}{V} \kappa_T \cdot \frac{V^2}{N^2} = k_B T V \kappa_T
    能量与体积关联涨落：

    ΔUΔV‾=−kB(∂U∂p/T)T,N=−kBT(∂U∂p)T=−kBT(∂(G+TS−pV)∂p)=kBT[T(∂V∂T)p+p(∂V∂p)T]\overline{\Delta U \Delta V} = -k_B \left( \frac{\partial U}{\partial p/T} \right)_{T,N} = -k_B T \left( \frac{\partial U}{\partial p} \right)_T = -k_B T \left( \frac{\partial (G + TS - pV)}{\partial p} \right) = k_B T \left[ T \left( \frac{\partial V}{\partial T} \right)_p + p \left( \frac{\partial V}{\partial p} \right)_T \right]

  3. 巨正则系综中的粒子涨落
    涨落定义：ΔN2=(N−N‾)2‾=N2‾−N‾2\Delta N^2 = \overline{(N - \overline{N})^2} = \overline{N^2} - \overline{N}^2
    粒子数二阶矩：

    N2‾=Tr{N^2ρ^}=1ΞTr{N^2e−βH^+βμN^}\overline{N^2} = Tr\{\hat{N}^2 \hat{\rho}\} = \frac{1}{\Xi} Tr\{\hat{N}^2 e^{-\beta \hat{\mathcal{H}} + \beta \mu \hat{N}}\}

    求导过程：

    =1ΞTr{(∂2∂(βμ)2e−βH^+βμN^)β,V}=1Ξ(∂2∂(βμ)2Tr{e−βH^+βμN^})β,V= \frac{1}{\Xi} Tr\left\{ \left( \frac{\partial^2}{\partial (\beta \mu)^2} e^{-\beta \hat{\mathcal{H}} + \beta \mu \hat{N}} \right)_{\beta,V} \right\} = \frac{1}{\Xi} \left( \frac{\partial^2}{\partial (\beta \mu)^2} Tr\{ e^{-\beta \hat{\mathcal{H}} + \beta \mu \hat{N}} \} \right)_{\beta,V}

    =1Ξ∂2Ξ∂(βμ)2=∂∂βμ(1Ξ∂Ξ∂βμ)−∂(1/Ξ)∂βμ∂Ξ∂βμ=(∂2lnΞ∂(βμ)2)+(∂lnΞ∂βμ)2= \frac{1}{\Xi} \frac{\partial^2 \Xi}{\partial (\beta \mu)^2} = \frac{\partial}{\partial \beta \mu} \left( \frac{1}{\Xi} \frac{\partial \Xi}{\partial \beta \mu} \right) - \frac{\partial (1/\Xi)}{\partial \beta \mu} \frac{\partial \Xi}{\partial \beta \mu} = \left( \frac{\partial^2 \ln \Xi}{\partial (\beta \mu)^2} \right) + \left( \frac{\partial \ln \Xi}{\partial \beta \mu} \right)^2

    结果：

    ΔN2=(∂2lnΞ∂(βμ)2)=(∂N∂βμ)βV=kBT(∂N∂μ)TV\Delta N^2 = \left( \frac{\partial^2 \ln \Xi}{\partial (\beta \mu)^2} \right) = \left( \frac{\partial N}{\partial \beta \mu} \right)_{\beta V} = k_B T \left( \frac{\partial N}{\partial \mu} \right)_{TV}

    结合 Gibbs-Duhem 关系推导
    Gibbs-Duhem 关系：dμ=−sdT+vdpd\mu = -s dT + v dp，df=−sdT−pdvdf = -s dT - p dv，μ=f+pv\mu = f + p v
    推导 (∂N∂μ)TV\left( \frac{\partial N}{\partial \mu} \right)_T V：

    (∂N∂μ)TV=V(∂N/V∂μ)TV=V(∂1/v∂μ)T=−Vv2(∂v∂μ)T=−N2V/(∂μ∂v)T\left( \frac{\partial N}{\partial \mu} \right)_T V = V \left( \frac{\partial N/V}{\partial \mu} \right)_{TV} = V \left( \frac{\partial 1/v}{\partial \mu} \right)_T = -\frac{V}{v^2} \left( \frac{\partial v}{\partial \mu} \right)_T = -\frac{N^2}{V} / \left( \frac{\partial \mu}{\partial v} \right)_T

    • (∂μ∂v)T\left( \frac{\partial \mu}{\partial v} \right)_T 的计算：

    (∂μ∂v)T=(∂(f+pv)∂v)T=(∂f∂v)T+p+v(∂p∂v)T=−p+p−1−v(∂v∂p)T=−1κT\left( \frac{\partial \mu}{\partial v} \right)_T = \left( \frac{\partial (f + p v)}{\partial v} \right)_T = \left( \frac{\partial f}{\partial v} \right)_T + p + v \left( \frac{\partial p}{\partial v} \right)_T = -p + p - \frac{1}{-v \left( \frac{\partial v}{\partial p} \right)_T} = -\frac{1}{\kappa_T}

    粒子数涨落：

    ΔN2=kBTN2κTV∝O(N1)\Delta N^2 = \frac{k_B T N^2 \kappa_T}{V} \propto O(N^1)

    相对涨落：

    ΔNN=kBTκTV=kBTκTNv∝1N\frac{\Delta N}{N} = \sqrt{\frac{k_B T \kappa_T}{V}} = \sqrt{\frac{k_B T \kappa_T}{N v}} \propto \frac{1}{\sqrt{N}}

  4. 巨正则系综中的能量涨落
    能量涨落定义：ΔE2=E2‾−E‾2\Delta E^2 = \overline{E^2} - \overline{E}^2
    配分函数求导：

    ΔE2=∂2lnΞ∂β2=−(∂U∂β)βμ,V=−∂(U,βμ)∂(β,βμ)\Delta E^2 = \frac{\partial^2 \ln \Xi}{\partial \beta^2} = -\left( \frac{\partial U}{\partial \beta} \right)_{\beta \mu, V} = -\frac{\partial (U, \beta \mu)}{\partial (\beta, \beta \mu)}

    链式法则展开：

    =−∂(U,βμ)∂(β,N)∂(β,N)∂(β,βμ)=−[(∂U∂β)N(∂βμ∂N)β−(∂U∂N)β(∂βμ∂β)N](∂N∂βμ)β= -\frac{\partial (U, \beta \mu)}{\partial (\beta, N)} \frac{\partial (\beta, N)}{\partial (\beta, \beta \mu)} = -\left[ \left( \frac{\partial U}{\partial \beta} \right)_N \left( \frac{\partial \beta \mu}{\partial N} \right)_\beta - \left( \frac{\partial U}{\partial N} \right)_\beta \left( \frac{\partial \beta \mu}{\partial \beta} \right)_N \right] \left( \frac{\partial N}{\partial \beta \mu} \right)_\beta

    =−[(∂U∂β)N−(∂U∂N)β(∂βμ∂β)N(∂N∂βμ)β]= -\left[ \left( \frac{\partial U}{\partial \beta} \right)_N - \left( \frac{\partial U}{\partial N} \right)_\beta \left( \frac{\partial \beta \mu}{\partial \beta} \right)_N \left( \frac{\partial N}{\partial \beta \mu} \right)_\beta \right]

    配分函数微分关系：

    dlnΞ=(∂lnΞ∂β)βμdβ+(∂lnΞ∂βμ)βd(βμ)=−Udβ+Nd(βμ)d \ln \Xi = \left( \frac{\partial \ln \Xi}{\partial \beta} \right)_{\beta \mu} d\beta + \left( \frac{\partial \ln \Xi}{\partial \beta \mu} \right)_\beta d(\beta \mu) = -U d\beta + N d(\beta \mu)

    d(lnΞ−Nβμ)=−Udβ−βμdN⇒(∂βμ∂β)N=(∂U∂N)βd(\ln \Xi - N \beta \mu) = -U d\beta - \beta \mu dN \Rightarrow \left( \frac{\partial \beta \mu}{\partial \beta} \right)_N = \left( \frac{\partial U}{\partial N} \right)_\beta

    最终能量涨落：

    ΔE2=−(∂U∂β)N+(∂U∂N)β2(∂N∂βμ)β=kBT2(∂U∂T)NV+(∂U∂N)TV2ΔN2=kBT2CV+(∂U∂N)TV2ΔN2\Delta E^2 = -\left( \frac{\partial U}{\partial \beta} \right)_N + \left( \frac{\partial U}{\partial N} \right)_\beta^2 \left( \frac{\partial N}{\partial \beta \mu} \right)_\beta = k_B T^2 \left( \frac{\partial U}{\partial T} \right)_{NV} + \left( \frac{\partial U}{\partial N} \right)_{TV}^2 \Delta N^2 = k_B T^2 C_V + \left( \frac{\partial U}{\partial N} \right)_{TV}^2 \Delta N^2

    物理意义：保持 NTVNTV 不变时的能量涨落 + 粒子数涨落导致的能量变动

    特殊体系能量涨落

    • 经典理想气体：p=nkBTp = n k_B T，能量涨落为 ΔE2=kBT2CV+u2ΔN2\Delta E^2 = k_B T^2 C_V + u^2 \Delta N^2（uu 为单粒子能量）
    • 强简并 Fermion：ΔE2≃kBT2CV+εF2ΔN2≃ΔN2εF2\Delta E^2 \simeq k_B T^2 C_V + \varepsilon_F^2 \Delta N^2 \simeq \Delta N^2 \varepsilon_F^2（εF\varepsilon_F 为费米能）

  5. 巨正则系综中能量与粒子数涨落的关联
    关联涨落定义：ΔNΔE‾=(N−N‾)(E−E‾)‾=NE‾−N‾E‾\overline{\Delta N \Delta E} = \overline{(N - \overline{N})(E - \overline{E})} = \overline{N E} - \overline{N} \overline{E}
    算符期望值：

    NE‾=Tr{N^H^ρ^}=1ΞTr{N^H^e−βH^+βμN^}=−1Ξ∂2∂β∂(βμ)Tr{e−βH^+βμN^}\overline{N E} = Tr\{ \hat{N} \hat{\mathcal{H}} \hat{\rho} \} = \frac{1}{\Xi} Tr\{ \hat{N} \hat{\mathcal{H}} e^{-\beta \hat{\mathcal{H}} + \beta \mu \hat{N}} \} = -\frac{1}{\Xi} \frac{\partial^2}{\partial \beta \partial (\beta \mu)} Tr\{ e^{-\beta \hat{\mathcal{H}} + \beta \mu \hat{N}} \}

    求导过程：

    =−1Ξ∂2Ξ∂β∂(βμ)=−∂∂β[1Ξ∂Ξ∂βμ]−∂(1/Ξ)∂β∂Ξ∂βμ=−∂∂β[1Ξ∂Ξ∂βμ]−1Ξ2(∂Ξ∂β)βμ(∂Ξ∂βμ)β= -\frac{1}{\Xi} \frac{\partial^2 \Xi}{\partial \beta \partial (\beta \mu)} = -\frac{\partial}{\partial \beta} \left[ \frac{1}{\Xi} \frac{\partial \Xi}{\partial \beta \mu} \right] - \frac{\partial (1/\Xi)}{\partial \beta} \frac{\partial \Xi}{\partial \beta \mu} = -\frac{\partial}{\partial \beta} \left[ \frac{1}{\Xi} \frac{\partial \Xi}{\partial \beta \mu} \right] - \frac{1}{\Xi^2} \left( \frac{\partial \Xi}{\partial \beta} \right)_{\beta \mu} \left( \frac{\partial \Xi}{\partial \beta \mu} \right)_\beta

    =−∂2lnΞ∂β∂(βμ)−(∂lnΞ∂β)βμ(∂lnΞ∂βμ)β= -\frac{\partial^2 \ln \Xi}{\partial \beta \partial (\beta \mu)} - \left( \frac{\partial \ln \Xi}{\partial \beta} \right)_{\beta \mu} \left( \frac{\partial \ln \Xi}{\partial \beta \mu} \right)_\beta

    =−∂2lnΞ∂β∂(βμ)+NE‾= -\frac{\partial^2 \ln \Xi}{\partial \beta \partial (\beta \mu)} + \overline{N E}

    关联涨落结果：

    ΔNΔE‾=−∂2lnΞ∂β∂(βμ)=−(∂N∂β)βμV=(∂U∂βμ)βV=kBT(∂U∂μ)TV=kBT(∂U∂N)TV(∂N∂μ)TV=(∂U∂N)TVΔN2\overline{\Delta N \Delta E} = -\frac{\partial^2 \ln \Xi}{\partial \beta \partial (\beta \mu)} = -\left( \frac{\partial N}{\partial \beta} \right)_{\beta \mu V} = \left( \frac{\partial U}{\partial \beta \mu} \right)_{\beta V} = k_B T \left( \frac{\partial U}{\partial \mu} \right)_{TV} = k_B T \left( \frac{\partial U}{\partial N} \right)_{TV} \left( \frac{\partial N}{\partial \mu} \right)_{TV} = \left( \frac{\partial U}{\partial N} \right)_{TV} \Delta N^2

  6. 温度涨落
    熵微分形式：dS=1TdU+pTdV−μTdNdS = \frac{1}{T}dU + \frac{p}{T}dV - \frac{\mu}{T}dN
    温度涨落：

    Δ(1/T)2‾=(−ΔTT2)2‾=−kB∂1/T∂U=kBT2∂T∂U=kBT2/∂U∂T\overline{\Delta (1/T)^2} = \overline{\left( -\frac{\Delta T}{T^2} \right)^2} = -k_B \frac{\partial 1/T}{\partial U} = \frac{k_B}{T^2} \frac{\partial T}{\partial U} = \frac{k_B}{T^2} / \frac{\partial U}{\partial T}

    温度平方涨落：

    ΔT2‾=kBT2CV\overline{\Delta T^2} = \frac{k_B T^2}{C_V}

    温度与能量关联涨落：

    −kB=Δ(1/T)ΔU‾=(−ΔT/T2)ΔU‾⇒ΔTΔU‾=kBT2-k_B = \overline{\Delta (1/T) \Delta U} = \overline{(-\Delta T / T^2) \Delta U} \Rightarrow \overline{\Delta T \Delta U} = k_B T^2

    温度与体积关联涨落：

    0=Δ(1/T)ΔV‾=−ΔTΔV‾/T2⇒ΔTΔV‾=00 = \overline{\Delta (1/T) \Delta V} = -\overline{\Delta T \Delta V} / T^2 \Rightarrow \overline{\Delta T \Delta V} = 0

涨落的准热力学理论

涨落的准热力学理论

概率分布推导
广义系综中，系统处于状态{Xi}\{X_i\}的概率为：

p({Xi})=∑{s∣Xi(s)=Xi}ps=∑{s∣Xi(s)=Xi}1Ξe−∑iyiXi(s)/kBp(\{X_i\}) = \sum_{\{s|X_i(s)=X_i\}} p_s = \sum_{\{s|X_i(s)=X_i\}} \frac{1}{\Xi} e^{-\sum_i y_i X_i(s)/k_B}

利用熵的表达式S=S({Xi})S = S(\{X_i\})，可将概率改写为：

p({Xi})=Ω({Xi})Ξe−∑iyiXi/kB=1Ξe[S({Xi})−∑iyiXi]/kBp(\{X_i\}) = \frac{\Omega(\{X_i\})}{\Xi} e^{-\sum_i y_i X_i/k_B} = \frac{1}{\Xi} e^{[S(\{X_i\}) - \sum_i y_i X_i]/k_B}

对熵在平均值{Xi‾}\{\overline{X_i}\}附近展开（泰勒展开）：

S({Xi‾+ΔXi})=S({Xi‾})+∑i(∂S∂Xi){X‾j}ΔXi+12∑ij(∂2S∂Xi∂Xj){X‾k}ΔXiΔXj+⋯S(\{\overline{X_i} + \Delta X_i\}) = S(\{\overline{X_i}\}) + \sum_i \left( \frac{\partial S}{\partial X_i} \right)_{\{\overline{X}_j\}} \Delta X_i + \frac{1}{2} \sum_{ij} \left( \frac{\partial^2 S}{\partial X_i \partial X_j} \right)_{\{\overline{X}_k\}} \Delta X_i \Delta X_j + \cdots

由于(∂S∂Xi)=yi\left( \frac{\partial S}{\partial X_i} \right) = y_i（强度量定义），代入后：

p({Xi})=e(−kBlnΞ+S‾−∑iyiXi‾)/kBexp{12kB∑ij∂2S∂Xi∂XjΔXiΔXj+⋯}p(\{X_i\}) = e^{(-k_B \ln \Xi + \overline{S} - \sum_i y_i \overline{X_i})/k_B} \exp\left\{ \frac{1}{2k_B} \sum_{ij} \frac{\partial^2 S}{\partial X_i \partial X_j} \Delta X_i \Delta X_j + \cdots \right\}

忽略高阶项，得到高斯分布形式的涨落概率：

p({ΔXi})=1Cexp{12kB∑ij∂2S∂Xi∂XjΔXiΔXj}p(\{\Delta X_i\}) = \frac{1}{C} \exp\left\{ \frac{1}{2k_B} \sum_{ij} \frac{\partial^2 S}{\partial X_i \partial X_j} \Delta X_i \Delta X_j \right\}

利用热力学关系∂yj∂Xi=∂2S∂Xi∂Xj\frac{\partial y_j}{\partial X_i} = \frac{\partial^2 S}{\partial X_i \partial X_j}（由dS=∑iyidXidS = \sum_i y_i dX_i求导），可改写为：

p({ΔXi})=1Cexp{12kB∑ij(∂yj∂Xi)ΔXiΔXj}=1Cexp{ΔXTAΔX}p(\{\Delta X_i\}) = \frac{1}{C} \exp\left\{ \frac{1}{2k_B} \sum_{ij} \left( \frac{\partial y_j}{\partial X_i} \right) \Delta X_i \Delta X_j \right\} = \frac{1}{C} \exp\{ \Delta X^T \mathcal{A} \Delta X \}

其中ΔX=(ΔX1ΔX2⋮)\Delta X = \begin{pmatrix} \Delta X_1 \\ \Delta X_2 \\ \vdots \end{pmatrix}，矩阵A\mathcal{A}的元素为：

Aij=12kB(∂yj∂Xi)\mathcal{A}_{ij} = \frac{1}{2k_B} \left( \frac{\partial y_j}{\partial X_i} \right)

矩阵方法（雅可比矩阵与关联函数）

矩阵定义
定义涨落向量ΔX\Delta X和矩阵A\mathcal{A}：

ΔX=(ΔX1ΔX2⋮),A=12kB(∂y1∂X1∂y2∂X1⋯∂y1∂X2∂y2∂X2⋯⋮⋮⋱)\Delta X = \begin{pmatrix} \Delta X_1 \\ \Delta X_2 \\ \vdots \end{pmatrix}, \quad \mathcal{A} = \frac{1}{2k_B} \begin{pmatrix} \frac{\partial y_1}{\partial X_1} & \frac{\partial y_2}{\partial X_1} & \cdots \\ \frac{\partial y_1}{\partial X_2} & \frac{\partial y_2}{\partial X_2} & \cdots \\ \vdots & \vdots & \ddots \end{pmatrix}

概率分布为：

p({ΔXi})=1Cexp{−ΔXTAΔX}p(\{\Delta X_i\}) = \frac{1}{C} \exp\{ -\Delta X^T \mathcal{A} \Delta X \}

关联函数计算
关联函数ΔXiΔXj‾\overline{\Delta X_i \Delta X_j}定义为：

ΔXiΔXj‾=∫−∞∞ΔXiΔXjp({ΔXl})dΔX∫−∞∞p({ΔXl})dΔX\overline{\Delta X_i \Delta X_j} = \frac{\int_{-\infty}^{\infty} \Delta X_i \Delta X_j p(\{\Delta X_l\}) d\Delta X}{\int_{-\infty}^{\infty} p(\{\Delta X_l\}) d\Delta X}

利用正交变换UU（UAUT=diag{A1,A2,⋯}U\mathcal{A}U^T = \text{diag}\{A_1, A_2, \cdots\}，且UUT=IUU^T = I），令ΔX′=UΔX\Delta X' = U\Delta X，则ΔXi′=UliΔXl\Delta X_i' = U_{li} \Delta X_l，dΔX=dΔX′d\Delta X = d\Delta X'（雅可比行列式为1）。在正交基下，概率分布为：

p({ΔXi′})∝exp{−∑lAl(ΔXl′)2}p(\{\Delta X_i'\}) \propto \exp\left\{ -\sum_l A_l (\Delta X_l')^2 \right\}

计算正交基下的关联：

ΔXi′ΔXj′‾=∫−∞∞ΔXi′ΔXj′e−∑lAl(ΔXl′)2dΔX′∫−∞∞e−∑lAl(ΔXl′)2dΔX′=−12Aiδij\overline{\Delta X_i' \Delta X_j'} = \frac{\int_{-\infty}^{\infty} \Delta X_i' \Delta X_j' e^{-\sum_l A_l (\Delta X_l')^2} d\Delta X'}{\int_{-\infty}^{\infty} e^{-\sum_l A_l (\Delta X_l')^2} d\Delta X'} = -\frac{1}{2A_i} \delta_{ij}

转换回原基：

ΔXiΔXj‾=∑lmUliUmjΔXl′ΔXm′‾=−12∑lmUliUmj1Alδlm=−12∑lUliUmj1Al\overline{\Delta X_i \Delta X_j} = \sum_{lm} U_{li} U_{mj} \overline{\Delta X_l' \Delta X_m'} = -\frac{1}{2} \sum_{lm} U_{li} U_{mj} \frac{1}{A_l} \delta_{lm} = -\frac{1}{2} \sum_l U_{li} U_{mj} \frac{1}{A_l}

利用UTAU=diag{A1,A2,⋯}U^T \mathcal{A} U = \text{diag}\{A_1, A_2, \cdots\}，即(UTAU)−1=diag{1/A1,1/A2,⋯}(U^T \mathcal{A} U)^{-1} = \text{diag}\{1/A_1, 1/A_2, \cdots\}，得：

ΔXiΔXj‾=−12(UTAU)ij−1=−12(A−1)ij\overline{\Delta X_i \Delta X_j} = -\frac{1}{2} (U^T \mathcal{A} U)^{-1}_{ij} = -\frac{1}{2} (\mathcal{A}^{-1})_{ij}

结合Aij=12kB(∂yj∂Xi)\mathcal{A}_{ij} = \frac{1}{2k_B} \left( \frac{\partial y_j}{\partial X_i} \right)，可得：

ΔXiΔXj‾=−kB(∂Xi∂yj)\overline{\Delta X_i \Delta X_j} = -k_B \left( \frac{\partial X_i}{\partial y_j} \right)

矩阵A\mathcal{A}的逆矩阵为：

A−1=−2kB(∂X1∂y1∂X1∂y2⋯∂X2∂y1∂X2∂y2⋯⋮⋮⋱)\mathcal{A}^{-1} = -2k_B \begin{pmatrix} \frac{\partial X_1}{\partial y_1} & \frac{\partial X_1}{\partial y_2} & \cdots \\ \frac{\partial X_2}{\partial y_1} & \frac{\partial X_2}{\partial y_2} & \cdots \\ \vdots & \vdots & \ddots \end{pmatrix}

Einstein方法

概率分布与热力学关系
从准热力学理论的高斯分布出发，利用热力学基本方程dU=TdS−∑iy~idXidU = TdS - \sum_i \tilde{y}_i dX_i（其中y~i\tilde{y}_i为广义力），可得：

dS=dUT+∑iy~iTdXidS = \frac{dU}{T} + \sum_i \frac{\tilde{y}_i}{T} dX_i

涨落的概率分布与熵变的关系为：

p∝exp{12kB[ΔUΔ(1T)+∑iΔXiΔ(y~iT)]}p \propto \exp\left\{ \frac{1}{2k_B} \left[ \Delta U \Delta \left( \frac{1}{T} \right) + \sum_i \Delta X_i \Delta \left( \frac{\tilde{y}_i}{T} \right) \right] \right\}

展开并整理：

p∝exp{12kB[−ΔUΔTT2−∑iy~iΔXiΔTT2+∑iΔXiΔy~iT]}p \propto \exp\left\{ \frac{1}{2k_B} \left[ -\frac{\Delta U \Delta T}{T^2} - \sum_i \frac{\tilde{y}_i \Delta X_i \Delta T}{T^2} + \sum_i \frac{\Delta X_i \Delta \tilde{y}_i}{T} \right] \right\}

进一步利用dU=TdS−∑iy~idXidU = TdS - \sum_i \tilde{y}_i dX_i，可得：

p∝exp{12kB[−ΔSΔT+∑iΔXiΔy~iT]}∝exp{−ΔSΔT+∑iΔXiΔy~i2kBT}p \propto \exp\left\{ \frac{1}{2k_B} \left[ -\frac{\Delta S \Delta T + \sum_i \Delta X_i \Delta \tilde{y}_i}{T} \right] \right\} \propto \exp\left\{ \frac{-\Delta S \Delta T + \sum_i \Delta X_i \Delta \tilde{y}_i}{2k_B T} \right\}

应用：保持NN不变，求T,VT, V的涨落
保持粒子数NN不变，取X1=TX_1 = T，X2=VX_2 = V，则y~1=−S\tilde{y}_1 = -S（由dF=−SdT−pdVdF = -SdT - pdV，F=U−TSF = U - TS，得dU=TdS−SdT−pdVdU = TdS - SdT - pdV，故y~1=−S\tilde{y}_1 = -S，y~2=−p\tilde{y}_2 = -p）。代入概率分布：

p(ΔT,ΔV)∝exp{−ΔSΔT+ΔVΔ(−p)2kBT}=exp{−ΔSΔT−pΔV2kBT}p(\Delta T, \Delta V) \propto \exp\left\{ \frac{-\Delta S \Delta T + \Delta V \Delta (-p)}{2k_B T} \right\} = \exp\left\{ \frac{-\Delta S \Delta T - p \Delta V}{2k_B T} \right\}

利用dF=−SdT−pdVdF = -SdT - pdV，熵变ΔS\Delta S可展开为：

ΔS=(∂S∂T)VΔT+(∂S∂V)TΔV=CVTΔT+(∂p∂T)VΔV\Delta S = \left( \frac{\partial S}{\partial T} \right)_V \Delta T + \left( \frac{\partial S}{\partial V} \right)_T \Delta V = \frac{C_V}{T} \Delta T + \left( \frac{\partial p}{\partial T} \right)_V \Delta V

代入概率分布并整理：

p(ΔT,ΔV)∝exp{12kBT[−(∂S∂T)VΔT2−(∂S∂V)TΔTΔV+(∂p∂T)VΔTΔV+(∂p∂V)TΔV2]}p(\Delta T, \Delta V) \propto \exp\left\{ \frac{1}{2k_B T} \left[ -\left( \frac{\partial S}{\partial T} \right)_V \Delta T^2 - \left( \frac{\partial S}{\partial V} \right)_T \Delta T \Delta V + \left( \frac{\partial p}{\partial T} \right)_V \Delta T \Delta V + \left( \frac{\partial p}{\partial V} \right)_T \Delta V^2 \right] \right\}

利用热力学关系(∂S∂V)T=(∂p∂T)V\left( \frac{\partial S}{\partial V} \right)_T = \left( \frac{\partial p}{\partial T} \right)_V（麦克斯韦关系），化简后：

p(ΔT,ΔV)∝exp{12kBT[−CVTΔT2−1VκTΔV2]}∝exp{−ΔT22kBT2/CV−ΔV22kBTκTV}p(\Delta T, \Delta V) \propto \exp\left\{ \frac{1}{2k_B T} \left[ -\frac{C_V}{T} \Delta T^2 - \frac{1}{V \kappa_T} \Delta V^2 \right] \right\} \propto \exp\left\{ -\frac{\Delta T^2}{2k_B T^2 / C_V} - \frac{\Delta V^2}{2k_B T \kappa_T V} \right\}

其中κT=−1V(∂V∂p)T\kappa_T = -\frac{1}{V} \left( \frac{\partial V}{\partial p} \right)_T为等温压缩率。

涨落与关联结果
从高斯分布的形式，可直接得到：

ΔT2‾=kBT2CV∝1N,ΔV2‾=kBTVκT∝N,ΔTΔV‾=0\overline{\Delta T^2} = \frac{k_B T^2}{C_V} \propto \frac{1}{N}, \quad \overline{\Delta V^2} = k_B T V \kappa_T \propto N, \quad \overline{\Delta T \Delta V} = 0

涨落的应用：Rayleigh散射强度

散射光强与密度涨落
光强为I0I_0的入射光被体积为VV的介质散射后，总散射光强为：

IscatterI0∝V2(Δϵ)2λ4\frac{I_{\text{scatter}}}{I_0} \propto \frac{V^2 (\Delta \epsilon)^2}{\lambda^4}

其中ϵ\epsilon为介质的介电常数，λ\lambda为入射光波长。利用Clausius-Mossotti关系：

ϵ−1ϵ+2=Aρ\frac{\epsilon - 1}{\epsilon + 2} = A \rho

其中AA为常数，ρ=N/V\rho = N/V为粒子数密度。对密度涨落Δρ\Delta \rho，介电常数的涨落为：

3Δϵ(ϵ+2)2=AΔρ⇒(Δϵ)2=(ϵ+2)49A2(Δρ)2\frac{3\Delta \epsilon}{(\epsilon + 2)^2} = A \Delta \rho \Rightarrow (\Delta \epsilon)^2 = \frac{(\epsilon + 2)^4}{9} A^2 (\Delta \rho)^2

密度涨落的均方值为：

(Δρ)2=ΔN2V2=kBTκTN2V3=kBTκTρ2V(\Delta \rho)^2 = \frac{\Delta N^2}{V^2} = \frac{k_B T \kappa_T N^2}{V^3} = \frac{k_B T \kappa_T \rho^2}{V}

代入散射光强公式：

IscatterI0∝V2(Δϵ)2λ4=kBTκTV9λ4(ϵ+2)4A2ρ2=kBTκTV9λ4(ϵ−1)2(ϵ+2)2(ϵ+2)4=kBTκTV(ϵ−1)29λ4(ϵ+2)2\frac{I_{\text{scatter}}}{I_0} \propto \frac{V^2 (\Delta \epsilon)^2}{\lambda^4} = \frac{k_B T \kappa_T V}{9\lambda^4} (\epsilon + 2)^4 A^2 \rho^2 = \frac{k_B T \kappa_T V}{9\lambda^4} \frac{(\epsilon - 1)^2 (\epsilon + 2)^2}{(\epsilon + 2)^4} = \frac{k_B T \kappa_T V (\epsilon - 1)^2}{9\lambda^4 (\epsilon + 2)^2}

临界乳光现象解释
在临界点附近，等温压缩率κT→∞\kappa_T \to \infty，导致散射光强极大，即临界乳光现象（1910年Einstein解释）。
