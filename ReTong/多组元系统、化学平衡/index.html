多组元系统、化学平衡

多组元单相系统

系统描述与基本参量

系统定义

均匀的、含有 kk 个组元的系统

参量分类

原始参量：

热学参量：TT
力学参量：pp
几何参量：VV
化学参量：N1,N2,⋯,NkN_1, N_2, \cdots, N_k （NiN_i 为第 ii 组元的摩尔数）
用质量表示：Mi=NimiM_i = N_i m_i （mim_i 为第 ii 组元的原子量）

高级参量：

  • 内能 UU，熵 SS，自由能 FF，Gibbs自由能 GG 等。

热力学函数与化学势

基本热力学关系

对于多组元系统，内能 UU、自由能 FF 和 Gibbs 自由能 GG 的全微分表达式如下：
内能 U(S,V,{Ni})U(S, V, \{N_i\})

dU=TdS−pdV+∑iμidNidU = TdS - pdV + \sum_i \mu_i dN_i

自由能 F(T,V,{Ni})F(T, V, \{N_i\})
定义：F=U−TSF = U - TS
微分形式：dF=−SdT−pdV+∑iμidNidF = -SdT - pdV + \sum_i \mu_i dN_i

Gibbs 自由能 G(T,p,{Ni})G(T, p, \{N_i\})
定义：G=F+pVG = F + pV
微分形式：dG=−SdT+Vdp+∑iμidNidG = -SdT + Vdp + \sum_i \mu_i dN_i

化学势 μi\mu_i 的定义

化学势 μi\mu_i 描述在保持其他变量不变时，系统能量随第 ii 组元粒子数的变化率。可通过不同热力学势的偏导数定义：

从内能看：

μi=(∂U∂Ni)S,V,{Nj≠i}\mu_i = \left(\frac{\partial U}{\partial N_i}\right)_{S, V, \{N_{j\neq i}\}}

从自由能看：

μi=(∂F∂Ni)T,V,{Nj≠i}\mu_i = \left(\frac{\partial F}{\partial N_i}\right)_{T, V, \{N_{j\neq i}\}}

从 Gibbs 自由能看：

μi=(∂G∂Ni)T,p,{Nj≠i}\mu_i = \left(\frac{\partial G}{\partial N_i}\right)_{T, p, \{N_{j\neq i}\}}

熵与压强的表达式

熵 SS：

S=−(∂F∂T)V,{Ni}=−(∂G∂T)p,{Ni}S = -\left(\frac{\partial F}{\partial T}\right)_{V, \{N_i\}} = -\left(\frac{\partial G}{\partial T}\right)_{p, \{N_i\}}

压强 pp：

p=−(∂F∂V)T,{Ni}p = -\left(\frac{\partial F}{\partial V}\right)_{T, \{N_i\}}

Maxwell 关系

利用热力学函数的二阶偏导数与求导顺序无关的性质，可以得到以下关系：

基于 dFdF 的关系

  1. (∂S∂V)T,{Ni}=(∂p∂T)V,{Ni}\left(\frac{\partial S}{\partial V}\right)_{T, \{N_i\}} = \left(\frac{\partial p}{\partial T}\right)_{V, \{N_i\}}
  2. (∂S∂Ni)T,V,{Nj≠i}=−(∂μi∂T)V,{Nj}\left(\frac{\partial S}{\partial N_i}\right)_{T, V, \{N_{j\neq i}\}} = -\left(\frac{\partial \mu_i}{\partial T}\right)_{V, \{N_j\}} （化学势随温度的改变）
  3. (∂p∂Ni)T,V,{Nj≠i}=−(∂μi∂V)T,{Nj}\left(\frac{\partial p}{\partial N_i}\right)_{T, V, \{N_{j\neq i}\}} = -\left(\frac{\partial \mu_i}{\partial V}\right)_{T, \{N_j\}} （化学势随体积的改变）
  4. (∂μi∂Nj)T,V,{Nk≠j}=(∂μj∂Ni)T,V,{Nk≠i}\left(\frac{\partial \mu_i}{\partial N_j}\right)_{T, V, \{N_{k\neq j}\}} = \left(\frac{\partial \mu_j}{\partial N_i}\right)_{T, V, \{N_{k\neq i}\}} （组元间的相互影响）

基于 dGdG 的关系

  1. (∂S∂p)T,{Ni}=−(∂V∂T)p,{Ni}\left(\frac{\partial S}{\partial p}\right)_{T, \{N_i\}} = -\left(\frac{\partial V}{\partial T}\right)_{p, \{N_i\}}
  2. (∂S∂Ni)T,p,{Nj≠i}=−(∂μi∂T)p,{Nj}\left(\frac{\partial S}{\partial N_i}\right)_{T, p, \{N_{j\neq i}\}} = -\left(\frac{\partial \mu_i}{\partial T}\right)_{p, \{N_j\}}
  3. (∂V∂Ni)T,p,{Nj≠i}=(∂μi∂p)T,{Nj}\left(\frac{\partial V}{\partial N_i}\right)_{T, p, \{N_{j\neq i}\}} = \left(\frac{\partial \mu_i}{\partial p}\right)_{T, \{N_j\}} （化学势随压强的改变）
  4. (∂μi∂Nj)T,p,{Nk≠j}=(∂μj∂Ni)T,p,{Nk≠i}\left(\frac{\partial \mu_i}{\partial N_j}\right)_{T, p, \{N_{k\neq j}\}} = \left(\frac{\partial \mu_j}{\partial N_i}\right)_{T, p, \{N_{k\neq i}\}}

Gibbs-Duhem 关系

摩尔量与强度量

定义总摩尔数 N=∑NiN = \sum N_i 和摩尔分数 xi=Ni/Nx_i = N_i / N（∑xi=1\sum x_i = 1）。
广延量可以表示为强度量与 NN 的乘积：

U=Nu,S=Ns,V=Nv,G=NgU = Nu, \quad S = Ns, \quad V = Nv, \quad G = Ng

其中 u,s,v,gu, s, v, g 为摩尔量。

关系式推导

由 U=TS−pV+∑NiμiU = TS - pV + \sum N_i \mu_i，两边除以 NN 得：

u=Ts−pv+∑xiμiu = Ts - pv + \sum x_i \mu_i

全微分形式：

du=Tds−pdv+∑μidxidu = Tds - pdv + \sum \mu_i dx_i

这就是 Gibbs-Duhem 关系。它表明在强度量中，总摩尔数 NN 是冗余参数，只有 k−1k-1 个独立的 xix_i。

Gibbs 自由能表达式

G=∑Niμi=N∑xiμi(T,p,{xi})G = \sum N_i \mu_i = N \sum x_i \mu_i(T, p, \{x_i\})

摩尔 Gibbs 自由能：

g=∑xiμig = \sum x_i \mu_i

dg=−sdT+vdp+∑μidxidg = -sdT + vdp + \sum \mu_i dx_i

巨势

定义巨势 JJ 为：

J=J(T,V,{μi})=F−∑Niμi=F−G=−pVJ = J(T, V, \{\mu_i\}) = F - \sum N_i \mu_i = F - G = -pV

微分形式：

dJ=−SdT−pdV−∑NidμidJ = -SdT - pdV - \sum N_i d\mu_i

  • 巨势主要用于处理和粒子源接触的系统，在统计物理中通过巨正则系综计算。
  • 此时压强 p(T,V,{μi})p(T, V, \{\mu_i\}) 是以温度、体积和化学势为自变量的特性函数。

实例：混合理想气体

状态方程与分压定律

设有 kk 个组元，总摩尔数 N=∑NiN = \sum N_i

分压定律：总压强等于各组元分压之和

piV=NiRTp_i V = N_i RT

p=∑pi=∑NiRTV=NRT/Vp = \sum p_i = \frac{\sum N_i RT}{V} = NRT/V

分体积定律：

pVi=NiRT,∑Vi=Vp V_i = N_i RT, \quad \sum V_i = V

混合熵

假设每种气体初始处于纯净分离状态，占据分体积 ViV_i，最终混合占据总体积 VV

Si=Si0+NiRlnVS_i = S_{i0} + N_iR\ln V

单一组元熵变：ΔSi=NiRln(V/Vi)\Delta S_i = N_i R \ln(V/V_i)

总混合熵 ΔSmix\Delta S_{mix}：

Sfinal=Sinit+∑ΔSi=Sinit+∑NiRlnNNiS_{final} = S_{init} + \sum \Delta S_i = S_{init} + \sum N_i R \ln \frac{N}{N_i}

Smix=−NR∑xilnxiS_{mix} = -N R \sum x_i \ln x_i

注：混合后内能不变，ΔU=0\Delta U = 0，故 Ffinal=Finit−TSmixF_{final} = F_{init} - T S_{mix}。

化学势表达式

纯净 ii 组元的化学势为：

μi0(T,p)=gi(T,p)=ϕi(T)+RTlnp\mu_i^0(T, p) = g_i(T, p) = \phi_i(T) + RT \ln p

G=∑iNigi(T,p)+NRT∑ixilnxi=N∑ixi[gi(T,p)+RTlnxi]G = \sum_i N_i g_i(T, p) + NRT \sum_i x_i \ln x_i = N \sum_i x_i [g_i(T, p) + RT \ln x_i]

在混合理想气体中，第 ii 组元的化学势仅与其自身的摩尔分数有关：

μi(T,p,{xj})=gi(T,p)+RTlnxi\mu_i(T, p, \{x_j\}) = g_i(T, p) + RT \ln x_i

展开后可写为：

μi=ϕi(T)+RTlnp+RTlnxi=ϕi(T)+RTln(xip)\mu_i = \phi_i(T) + RT \ln p + RT \ln x_i = \phi_i(T) + RT \ln(x_i p)

物理意义：理想混合气体中，ii 组元在状态 (T,p,xi)(T, p, x_i) 下的化学势，等同于纯净 ii 组元在状态 (T,xip)(T, x_i p) 下的化学势。这一结论对稀溶液和合金也近似成立。

多元复相系统

系统参量与变量

基本定义：系统包含 kk 个组元，分为 ϕ\phi 个部分（每部分内部均匀），即 ϕ\phi 个相。

平衡条件：
热平衡：每个相的温度相同。
力学平衡：不考虑界面影响时，每个相的压强相同。

变量选择：
广延量：T,p,Vσ,{Nσi}T, p, V_\sigma, \{N_{\sigma i}\}（σ\sigma 为相标记，ii 为组元标记）。
独立变量数：2+ϕ×k2 + \phi \times k 个。

强度量表示：

  • 使用摩尔体积 vσ=Vσ/Nσv_\sigma = V_\sigma/N_\sigma 和摩尔分数 xσi=Nσi/Nσx_{\sigma i} = N_{\sigma i}/N_\sigma。
  • 约束条件：∑ixσi=1\sum_i x_{\sigma i} = 1，每个相有 k−1k-1 个独立的 xix_i。
  • 独立参量：T,p,{xσi}T, p, \{x_{\sigma i}\}，共 2+ϕ(k−1)2 + \phi(k-1) 个。
  • 冗余参量：各相总摩尔数 {Nσ}\{N_\sigma\}，共 ϕ\phi 个。

热力学函数

相内定义：每个相内部可定义各自的热力学量（Uσ,Sσ,Fσ,GσU_\sigma, S_\sigma, F_\sigma, G_\sigma）。

基本方程：

  • 内能微分：dUσ=TdSσ−pdVσ+∑iμσidNσidU_\sigma = TdS_\sigma - pdV_\sigma + \sum_i \mu_{\sigma i} dN_{\sigma i}。
  • 吉布斯函数微分：dGσ=−SσdT+Vσdp+∑iμσidNσidG_\sigma = -S_\sigma dT + V_\sigma dp + \sum_i \mu_{\sigma i} dN_{\sigma i}。

总量关系：系统的总广延量为各相广延量之和（如 U=∑σUσ,⋯U = \sum_\sigma U_\sigma,\,\cdots）。

多元复相平衡条件

在无化学反应且恒温恒压下，平衡时同一种物质在所有相里的化学势相同。

μiI=μiII=⋯=μiϕ(i=1,2,⋯,k)\mu^I_i = \mu^{II}_i = \cdots = \mu^\phi_i \quad (i=1, 2, \cdots, k)

推导方法：

  1. 极值法：利用吉布斯自由能 G 在约束 $N_i$ 守恒下的极小值。

    保持温度、压强不变，每个组元可以在不同相里跑动，直到平衡，且无化学反应时

    \begin{aligned} &N_i = \sum_{\sigma} N_i^\sigma \quad \boxed{N_i \text{ 守恒}} \\ &0 = \delta N_i = \sum_{\sigma} \delta N_i^\sigma \implies \delta N_i^I = -\sum_{\sigma=II}^{\phi} \delta N_i^\sigma \\ &dG = -SdT + Vdp + \sum_{\sigma i} \mu_i^\sigma dN_i^\sigma \\ &\delta G = \sum_{i=1}^{k} \sum_{\sigma=1}^{\phi} \mu_i^\sigma \delta N_i^\sigma = \sum_{i=1}^{k} \left[ -\mu_i^I \sum_{\sigma=II}^{\phi} \delta N_i^\sigma + \sum_{\sigma=II}^{\phi} \mu_i^\sigma \delta N_i^\sigma \right] = \sum_{i=1}^{k} \sum_{\sigma=II}^{\phi} (\mu_i^\sigma - \mu_i^I) \delta N_i^\sigma \\ &\text{多元复相平衡条件} \\ &\text{平衡条件} \\ &\delta G = \sum_{i=1}^{k} \sum_{\sigma=II}^{\phi} (\mu_i^\sigma - \mu_i^I) \delta N_i^\sigma \\ &\mu_i^I = \mu_i^{II} = \cdots = \mu_i^\phi \quad i = 1,2,\cdots,k \\ &\textbf{}{平衡时，同一种物质在所有相里的的化学势都相同} \\ &\textbf{一共有 } (\phi - 1) \times k \textbf{ 个条件} \end{aligned}
  2. 拉格朗日乘子法：引入乘子 $\mu_i$，在约束 $\sum_\sigma N_{\sigma i} = N_i$ 下求极值。 引入 𝑘 个 Langrange 乘子 $\mu_i$， $$ \begin{align*} 0 &= \delta G - \sum_i \mu_i \delta N_i = \sum_{i\sigma} \mu_i^\sigma \delta N_i^\sigma - \sum_i \mu_i \sum_\sigma \delta N_i^\sigma \\ &= \sum_{i\sigma} (\mu_i^\sigma - \mu_i) \delta N_i^\sigma \end{align*} $$

    引入Langrange乘子之后，δNiσ\delta N_i^\sigma 可以看成是独立的，因此上式成立的条件是 μiσ−μi=0\mu_i^\sigma - \mu_i = 0

Gibbs 相律

公式：

独立强度量个数：2+(k−1)ϕ2 + (k-1)\phi。

平衡约束条件数：(ϕ−1)k(\phi-1)k。

自由度 DD：$D = \text{独立强度量} - \text{约束条件} = 2 + k - \phi$

单组元系统 (k=1k=1)：

  • 单相 (ϕ=1\phi=1)：D=2D=2，T,pT, p 均可独立变动。
  • 两相 (ϕ=2\phi=2)：D=1D=1，两相共存线（如熔点、沸点曲线），T=T(p)T=T(p)。
  • 三相点 (ϕ=3\phi=3)：D=0D=0，T,pT, p 均确定。

两组元系统 (k=2k=2)：

  • 单相 (ϕ=1\phi=1)：D=3D=3，独立变量为 T,p,xT, p, x（浓度）。
  • 两相 (ϕ=2\phi=2)：D=2D=2（如饱和溶液+盐，溶液+冰，溶液+蒸汽）。
  • 三相 (ϕ=3\phi=3)：D=1D=1。
  • 四相 (ϕ=4\phi=4)：D=0D=0，温度、压强和浓度均确定（如饱和溶液+盐+冰+蒸汽）。

典型例子与应用

渗透压：

现象：纯水与盐水通过半透膜平衡，化学势相等导致盐水侧压强升高。

公式：Δp=−RTvwlnxw≈RTρwx\Delta p = -\frac{RT}{v_w} \ln x_w \approx RT \rho_w x

溶质摩尔浓度 x=1−xwx = 1 - x_w

实例：室温下生理盐水渗透压约 6 atm（≈60\approx 60 米水柱）。

杂质对相变点的影响：

原理：溶液里有杂质，μl(T,p,xj)=gl(T,p)+RTlnxl<gl(T,p)\mu_l(T, p, {x_j}) = g_l(T, p) + RT \ln x_l < g_l (T, p)，溶液中溶剂化学势降低 ⇒\Rightarrow 冰点降低，沸点升高。

应用：撒盐化雪。

有 gs=μl(T,p,x)g_s = \mu_l(T, p, x)，

gs(T∗,p)−ssΔT=gl(T∗,p)−slΔT+R(T∗+ΔT)ln(1−x)g_s(T^*, p) - s_s \Delta T = g_l(T^*, p) - s_l \Delta T + R(T^* + \Delta T) \ln(1 - x)

在纯净物熔点 T∗,gs(T∗,p)=gl(T∗,p)T^*,\,g_s(T^*, p) = g_l(T^*, p)

化简得凝固点降低：ΔT≈−RT∗xL+RT∗xT∗\Delta T \approx -\frac{RT^* x}{L + RT^* x} T^*。

简单二元系统相关系：

类型 I（固-液）：固体完全不互溶，溶液任意混合（如金-铊）。
特征：低共熔点，合金熔点降低（如青铜）。

类型 II（气-液）：气相和液相均可任意混合。
Raoult 定律：蒸汽中某成分分压 xivpx^v_i p 等于该成分饱和蒸气压 pi∗p^*_i 与溶液浓度 xilx^l_i 之积。
特征：气相和液相成分不同，易挥发成分在气相中比例更高。

化学平衡

化学反应的描述

反应式改写：∑iνiAi=0\sum_i \nu_i A_i = 0。

规定：生成物系数为正，反应物系数为负。

AiA_i 为分子式，νi\nu_i 为化学反应系数。

化学反应度 ξ\xi：ΔNi=νiξ\Delta N_i = \nu_i \xi。ξ\xi 越大，生成物越多。

反应方向与平衡条件

吉布斯自由能变：恒温恒压下，ΔG=(∑iνiμi)ξ=−Aξ≤0\Delta G = (\sum_i \nu_i \mu_i) \xi = -A \xi \le 0。

化学亲和势 AA：A=−∑iνiμiA = -\sum_i \nu_i \mu_i。

A>0A > 0：正向反应自发进行。
A<0A < 0：逆向反应自发进行。
A=0A = 0：反应达到平衡。

平衡常数与控制方法

理想气体假设：
化学势形式：μi=gi(T,p)+RTlnxi\mu_i = g_i(T, p) + RT \ln x_i。
平衡时：A=−∑iνiμi=0⇒lnK=−∑iνigi(T,p)RTA=-\sum_i \nu_i \mu_i=0 \Rightarrow \ln K = -\frac{\sum_i \nu_i g_i(T, p)}{RT}。
平衡常数：K=∏ixiνiK = \prod_i x_i^{\nu_i}。KK 越大，平衡时生成物越多。

温度对平衡的影响：
关系式：(∂lnK∂T)p=ΔhRT2(\frac{\partial \ln K}{\partial T})_p = \frac{\Delta h}{RT^2}（Δh\Delta h 为反应热）。
吸热反应 (Δh>0\Delta h > 0)：温度升高，KK 增大，正向反应。
放热反应 (Δh<0\Delta h < 0)：温度升高，KK 减小，逆向反应。

压强对平衡的影响：
关系式：(∂lnK∂p)T=−∑iνiviRT=−∑iνip(\frac{\partial \ln K}{\partial p})_T = -\frac{\sum_i \nu_i v_i}{RT} = -\frac{\sum_i \nu_i}{p}。
体积减小反应：压强升高，KK 增大，正向反应。
体积增加反应：压强升高，KK 减小，逆向反应。

Le Chatelier 原理：系统变化总是倾向于抵消外界的改变。

应用实例

人工合成氨 (N2+3H2⇌2NH3N_2 + 3H_2 \rightleftharpoons 2NH_3)：
特征：放热反应，体积减小反应。
热力学条件：低温、高压有利于正向反应。
背景：Haber 法合成氨解决了肥料短缺问题。

物理体系中的应用：
溶液 pH 值：H++OH−⇌H2OH^+ + OH^- \rightleftharpoons H_2O，加入酸碱改变浓度但乘积 xH+xOH−x_{H^+}x_{OH^-} 保持不变。
半导体：e+h⇌γe + h \rightleftharpoons \gamma（电子-空穴对），nenhn_e n_h 乘积保持不变。
宇宙学：原初核合成、电子-正电子湮灭。
