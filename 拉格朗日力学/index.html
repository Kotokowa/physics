Part1 拉格朗日力学

约束与自由度

约束分为完整约束（可积约束）和非完整约束（不可积约束）：

  • 完整约束：约束方程可表示为坐标和时间的函数，且不包含速度，或虽包含速度但可通过积分转化为坐标关系。每个完整约束方程同时减少1个独立坐标和1个独立速度分量。
  • 非完整约束：约束方程无法通过积分转化为坐标关系（如微分约束），仅减少1个独立速度分量，不减少独立坐标。
    自由度 s=3n−ks = 3n - k，其中 nn 为质点数， kk 为完整约束数。

广义坐标

广义坐标 qαq_\alpha ( α=1,2,⋯,s\alpha = 1, 2, \cdots, s )是确定力学体系位形的任意一组独立变量，需满足：

  1. 独立性：各广义坐标相互独立；
  2. 唯一性：能唯一确定体系的位形；
  3. 整体性：属于整个体系，而非单个质点。

不稳定约束

若约束方程显含时间 tt（即 f(ri,r˙i,t)=0f(\mathbf{r}_i, \dot{\mathbf{r}}_i, t) = 0），称为不稳定约束。其全微分为：

df=∂f∂ridri+∂f∂r˙idr˙i+∂f∂tdt=0df = \frac{\partial f}{\partial \mathbf{r}_i} d\mathbf{r}_i + \frac{\partial f}{\partial \dot{\mathbf{r}}_i} d\dot{\mathbf{r}}_i + \frac{\partial f}{\partial t} dt = 0

等时变分定义为：

δf:=df(ri+dri,r˙i+dr˙i,t)−f(ri,r˙i,t)=∂f∂r˙idr˙i\delta f := df(\mathbf{r}_i + d\mathbf{r}_i, \dot{\mathbf{r}}_i + d\dot{\mathbf{r}}_i, t) - f(\mathbf{r}_i, \dot{\mathbf{r}}_i, t) = \frac{\partial f}{\partial \dot{\mathbf{r}}_i} d\dot{\mathbf{r}}_i

虚功原理

在理想约束（约束力做功之和为零，即∑i=1nRi⋅δri=0\sum_{i=1}^n \mathbf{R}_i \cdot \delta \mathbf{r}_i = 0）下，主动力做功之和为零：

∑i=1nFi⋅δri=0\sum_{i=1}^n \mathbf{F}_i \cdot \delta \mathbf{r}_i = 0

其中 δri=∑α=1s∂ri∂qαδqα\delta \mathbf{r}_i = \sum_{\alpha=1}^s \frac{\partial \mathbf{r}_i}{\partial q_\alpha} \delta q_\alpha 为虚位移。代入得：

∑α=1s(∑i=1nFi⋅∂ri∂qα)δqα=0\sum_{\alpha=1}^s \left( \sum_{i=1}^n \mathbf{F}_i \cdot \frac{\partial \mathbf{r}_i}{\partial q_\alpha} \right) \delta q_\alpha = 0

由于 δqα\delta q_\alpha 独立，故广义主动力 Qα=∑i=1nFi⋅∂ri∂qα=0Q_\alpha = \sum_{i=1}^n \mathbf{F}_i \cdot \frac{\partial \mathbf{r}_i}{\partial q_\alpha} = 0（α=1,2,⋯,s\alpha = 1, 2, \cdots, s）。

达朗贝尔原理

由牛顿第二定律 Fi+Ri=mir¨i\mathbf{F}_i + \mathbf{R}_i = m_i \ddot{\mathbf{r}}_i，引入惯性力 −mir¨i-m_i \ddot{\mathbf{r}}_i，得虚功形式的平衡方程：

∑i=1n(Fi−mir¨i)⋅δri=0\sum_{i=1}^n (\mathbf{F}_i - m_i \ddot{\mathbf{r}}_i) \cdot \delta \mathbf{r}_i = 0

代入虚位移表达式，化简得：

∑α=1s(Qα−∑i=1nmir¨i⋅∂ri∂qα)δqα=0\sum_{\alpha=1}^s \left( Q_\alpha - \sum_{i=1}^n m_i \ddot{\mathbf{r}}_i \cdot \frac{\partial \mathbf{r}_i}{\partial q_\alpha} \right) \delta q_\alpha = 0

因 δqα\delta q_\alpha 独立，故：

Qα=∑i=1nmir¨i⋅∂ri∂qα(α=1,2,⋯,s)Q_\alpha = \sum_{i=1}^n m_i \ddot{\mathbf{r}}_i \cdot \frac{\partial \mathbf{r}_i}{\partial q_\alpha} \quad (\alpha = 1, 2, \cdots, s)

经典拉格朗日关系与拉格朗日方程

经典 Lagrange 关系

质点的速度可表示为广义坐标和时间的函数：

r˙i=∑α=1s∂ri∂qαq˙α+∂ri∂t=vi\dot{\mathbf{r}}_i = \sum_{\alpha=1}^s \frac{\partial \mathbf{r}_i}{\partial q_\alpha} \dot{q}_\alpha + \frac{\partial \mathbf{r}_i}{\partial t} = v_i

其中 q˙α\dot{q}_\alpha 为广义速度，(q,q˙,t)(q, \dot{q}, t) 表示广义坐标、广义速度和时间的关系。

对速度 r˙i\dot{\mathbf{r}}_i 取全微分，得：

dr˙i=∑α=1sd(∂ri∂qα)q˙α+∑α=1s∂ri∂qαdq˙α+d(∂ri∂t)d\dot{\mathbf{r}}_i = \sum_{\alpha=1}^s d\left( \frac{\partial \mathbf{r}_i}{\partial q_\alpha} \right) \dot{q}_\alpha + \sum_{\alpha=1}^s \frac{\partial \mathbf{r}_i}{\partial q_\alpha} d\dot{q}_\alpha + d\left( \frac{\partial \mathbf{r}_i}{\partial t} \right)

进一步，位置矢量的全微分可表示为：

dri=∑α=1sddt(∂ri∂qα)dqα+∑α=1s∂ri∂qαdqα+d(∂ri∂t)d\mathbf{r}_i = \sum_{\alpha=1}^s \frac{d}{dt}\left( \frac{\partial \mathbf{r}_i}{\partial q_\alpha} \right) dq_\alpha + \sum_{\alpha=1}^s \frac{\partial \mathbf{r}_i}{\partial q_\alpha} dq_\alpha + d\left( \frac{\partial \mathbf{r}_i}{\partial t} \right)

进一步，位置矢量的全微分也可表示为：

dri=∑α=1s∂ri∂qαdqα+∑α=1s∂ri∂q˙αdq˙α+∂ri∂tdtd\mathbf{r}_i = \sum_{\alpha=1}^s \frac{\partial \mathbf{r}_i}{\partial q_\alpha} dq_\alpha + \sum_{\alpha=1}^s \frac{\partial \mathbf{r}_i}{\partial \dot{q}_\alpha} d\dot{q}_\alpha + \frac{\partial \mathbf{r}_i}{\partial t} dt

对比上述两式的系数，可得拉格朗日关系：

  1. 对广义坐标的偏导数交换顺序不变：

    ddt(∂ri∂qα)=∂r˙i∂qα\frac{d}{dt} \left( \frac{\partial \mathbf{r}_i}{\partial q_\alpha} \right) = \frac{\partial \dot{\mathbf{r}}_i}{\partial q_\alpha}

  2. 对广义速度的偏导数等于位置矢量对广义坐标的偏导数：

    ∂r˙i∂q˙α=∂ri∂qα\frac{\partial \dot{\mathbf{r}}_i}{\partial \dot{q}_\alpha} = \frac{\partial \mathbf{r}_i}{\partial q_\alpha}

稳定约束下拉格朗日关系推导
1. 速度表达式

质点位置矢量 \mathbf{r}_i = \mathbf{r}_i(q_1, q_2, \dots, q_s, t)

对时间求导得速度：

r˙i=∑α=1s∂ri∂qαq˙α+∂ri∂t\dot{\mathbf{r}}_i = \sum_{\alpha=1}^s \frac{\partial \mathbf{r}_i}{\partial q_\alpha} \dot{q}_\alpha + \frac{\partial \mathbf{r}_i}{\partial t}

2. 加速度表达式

对速度再求时间导数：

r¨i=ddtr˙i=∑α=1s[ddt(∂ri∂qα)q˙α+∂ri∂qαq¨α]+ddt(∂ri∂t)\ddot{\mathbf{r}}_i = \frac{d}{dt}\dot{\mathbf{r}}_i = \sum_{\alpha=1}^s \left[ \frac{d}{dt}\left(\frac{\partial \mathbf{r}_i}{\partial q_\alpha}\right) \dot{q}_\alpha + \frac{\partial \mathbf{r}_i}{\partial q_\alpha} \ddot{q}_\alpha \right] + \frac{d}{dt}\left(\frac{\partial \mathbf{r}_i}{\partial t}\right)

3. 展开全导数

ddt(∂ri∂qα)=∑β=1s∂2ri∂qβ∂qαq˙β+∂2ri∂t∂qα\frac{d}{dt}\left(\frac{\partial \mathbf{r}_i}{\partial q_\alpha}\right) = \sum_{\beta=1}^s \frac{\partial^2 \mathbf{r}_i}{\partial q_\beta \partial q_\alpha} \dot{q}_\beta + \frac{\partial^2 \mathbf{r}_i}{\partial t \partial q_\alpha}

ddt(∂ri∂t)=∑α=1s∂2ri∂qα∂tq˙α+∂2ri∂t2\frac{d}{dt}\left(\frac{\partial \mathbf{r}_i}{\partial t}\right) = \sum_{\alpha=1}^s \frac{\partial^2 \mathbf{r}_i}{\partial q_\alpha \partial t} \dot{q}_\alpha + \frac{\partial^2 \mathbf{r}_i}{\partial t^2}

4. 合并整理

将上述结果代入加速度表达式，整理得：

r¨i=∑α=1s∂ri∂qαq¨α+∑α=1s∑β=1s∂2ri∂qα∂qβq˙αq˙β+2∑α=1s∂2ri∂qα∂tq˙α+∂2ri∂t2\ddot{\mathbf{r}}_i = \sum_{\alpha=1}^s \frac{\partial \mathbf{r}_i}{\partial q_\alpha} \ddot{q}_\alpha + \sum_{\alpha=1}^s \sum_{\beta=1}^s \frac{\partial^2 \mathbf{r}_i}{\partial q_\alpha \partial q_\beta} \dot{q}_\alpha \dot{q}_\beta + 2\sum_{\alpha=1}^s \frac{\partial^2 \mathbf{r}_i}{\partial q_\alpha \partial t} \dot{q}_\alpha + \frac{\partial^2 \mathbf{r}_i}{\partial t^2}

5. 特殊情况

稳定约束（ri\mathbf{r}_i 不显含 tt）时：

r¨i=∑α=1s∂ri∂qαq¨α+∑α=1s∑β=1s∂2ri∂qα∂qβq˙αq˙β\ddot{\mathbf{r}}_i = \sum_{\alpha=1}^s \frac{\partial \mathbf{r}_i}{\partial q_\alpha} \ddot{q}_\alpha + \sum_{\alpha=1}^s \sum_{\beta=1}^s \frac{\partial^2 \mathbf{r}_i}{\partial q_\alpha \partial q_\beta} \dot{q}_\alpha \dot{q}_\beta

此时可得重要关系：

∂r¨i∂q˙α=2∂r˙i∂qα\frac{\partial \ddot{\mathbf{r}}_i}{\partial \dot{q}_\alpha} = 2\frac{\partial \dot{\mathbf{r}}_i}{\partial q_\alpha}

拉格朗日方程

对 ∑i=1nmir¨i⋅∂ri∂qα\sum_{i=1}^n m_i \ddot{\mathbf{r}}_i \cdot \frac{\partial \mathbf{r}_i}{\partial q_\alpha} 化简：

Qα=ddt(∂T∂q˙α)−∂T∂qαQ_\alpha = \frac{d}{dt} \left( \frac{\partial T}{\partial \dot{q}_\alpha} \right) - \frac{\partial T}{\partial q_\alpha}

其中动能 T=∑i=1n12mir˙i2T = \sum_{i=1}^n \frac{1}{2} m_i \dot{\mathbf{r}}_i^2。
若系统为保守系统（主动力有势，即 Fi=−∇iV\mathbf{F}_i = -\nabla_i V），则广义主动力 Qα=−∂V∂qαQ_\alpha = -\frac{\partial V}{\partial q_\alpha}，代入得：

ddt(∂T∂q˙α)−∂T∂qα=−∂V∂qα\frac{d}{dt} \left( \frac{\partial T}{\partial \dot{q}_\alpha} \right) - \frac{\partial T}{\partial q_\alpha} = -\frac{\partial V}{\partial q_\alpha}

定义拉格朗日函数 L=T−VL = T - V，则拉格朗日方程为：

ddt(∂L∂q˙α)−∂L∂qα=0(α=1,2,⋯,s)\frac{d}{dt} \left( \frac{\partial L}{\partial \dot{q}_\alpha} \right) - \frac{\partial L}{\partial q_\alpha} = 0 \quad (\alpha = 1, 2, \cdots, s)

动量守恒与能量守恒

  1. 广义动量守恒：
    若拉格朗日函数 LL 不显含某广义坐标 qβq_\beta（循环坐标），则对应的广义动量 pβ=∂L∂q˙βp_\beta = \frac{\partial L}{\partial \dot{q}_\beta} 守恒：

    $$ \frac{d}{dt} p_\beta = 0 \implies p_\beta = \text{常数} $$

    物理意义：若系统在某方向无外力作用（如平动方向无主动力），则该方向的广义动量守恒。

  2. 能量守恒：
    若拉格朗日函数 LL 不显含时间 tt（即 ∂L∂t=0\frac{\partial L}{\partial t} = 0），则系统的机械能 E=T+VE = T + V 守恒：

    $$ \frac{dE}{dt} = 0 \implies E = T + V = \text{常数} $$

    推导：由 L=L(q,q˙)L = L(q, \dot{q})，计算 dLdt=∑α=1s(∂L∂qαq˙α+∂L∂q˙αq¨α)\frac{dL}{dt} = \sum_{\alpha=1}^s \left( \frac{\partial L}{\partial q_\alpha} \dot{q}_\alpha + \frac{\partial L}{\partial \dot{q}_\alpha} \ddot{q}_\alpha \right)，利用拉格朗日方程 ∂L∂qα=ddt∂L∂q˙α\frac{\partial L}{\partial q_\alpha} = \frac{d}{dt} \frac{\partial L}{\partial \dot{q}_\alpha}，得：

    dLdt=∑α=1sddt(∂L∂q˙α)q˙α+∑α=1s∂L∂q˙αq¨α=ddt(∑α=1s∂L∂q˙αq˙α)\frac{dL}{dt} = \sum_{\alpha=1}^s \frac{d}{dt} \left( \frac{\partial L}{\partial \dot{q}_\alpha} \right) \dot{q}_\alpha + \sum_{\alpha=1}^s \frac{\partial L}{\partial \dot{q}_\alpha} \ddot{q}_\alpha = \frac{d}{dt} \left( \sum_{\alpha=1}^s \frac{\partial L}{\partial \dot{q}_\alpha} \dot{q}_\alpha \right)

    因 ∂L∂q˙α=∂T∂q˙α\frac{\partial L}{\partial \dot{q}_\alpha} = \frac{\partial T}{\partial \dot{q}_\alpha}（势能 VV 与速度无关），且 TT 是广义速度的二次齐次函数，由欧拉定理 ∑α=1s∂T∂q˙αq˙α=2T\sum_{\alpha=1}^s \frac{\partial T}{\partial \dot{q}_\alpha} \dot{q}_\alpha = 2T，故：

    $$ \frac{dL}{dt} = \frac{d}{dt} (2T) \implies \frac{d}{dt} (L - 2T) = 0 \implies \frac{d}{dt} (T + V) = 0 $$

Part2 拉格朗日乘子法

对于带约束 ϕ(x,y)=0\phi(x, y) = 0 的优化问题（主动力 F(x,y)F(x, y)），构造拉格朗日函数：

L(x,y,λ)=F(x,y)+λϕ(x,y)\mathcal{L}(x, y, \lambda) = F(x, y) + \lambda \phi(x, y)

极值点满足：

{∂L∂x=∂F∂x+λ∂ϕ∂x=0∂L∂y=∂F∂y+λ∂ϕ∂y=0ϕ(x,y)=0\begin{cases} \frac{\partial \mathcal{L}}{\partial x} = \frac{\partial F}{\partial x} + \lambda \frac{\partial \phi}{\partial x} = 0 \\ \frac{\partial \mathcal{L}}{\partial y} = \frac{\partial F}{\partial y} + \lambda \frac{\partial \phi}{\partial y} = 0 \\ \phi(x, y) = 0 \end{cases}

其中 λ\lambda 为拉格朗日乘子，物理意义为约束力的大小。
